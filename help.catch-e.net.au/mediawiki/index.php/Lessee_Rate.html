Overview

"Lessee Rate" is the rate of interest required to fund the deal without disclosing the brokerage, i.e. the interest being effectively charged based on the depreciable amount rather than the financed amount.

Open the Lessee Rate spreadsheet to see a calculation model. Note this model does not accommodate the "Agreed Lessee Rate" calculations.

[edit]

Calculation

The Lessee Rate can be calculated numerous ways depending on Setup / Reference Data → gb_controls "lessee_rate_calculation_basis" and Client settings noted below.

[edit]
Standard Calculation

The standard calculation for "Lessee Rate" is based on "Depreciable Amount".
Set Setup / Reference Data → gb_controls "lessee_rate_calculation_basis" to:

  • Total Brokerage Included = no
  • Broker Fee Included = no
  • Commission Fee Included = no
  • Establishment / Admin Fee Included = no
  • Originator Fee Included = no

You can customise the calculation by updating the Setup / Reference Data → gb_controls "lessee_rate_calculation_basis" which adds Brokerage items to the "Depreciable Amount" according to the flag settings in the Global Control as shown below.

If "Total Brokerage Included" is set to ‘no’ the Lessee Rate calculation includes:
"Depreciable Amount"
PLUS
If "Broker Fee Included" = ‘yes’ then Broker Fee amount else "0.00"
PLUS
If "Commission Fee Included" = ‘yes’ then Commission Fee amount else "0.00"
PLUS
If "Establishment / Admin Fee Included" = ‘yes’ then Establishment / Admin Fee amount else "0.00"
PLUS
If "Originator Fee Included" = ‘yes’ then Originator Fee amount else "0.00"
MINUS
One month’s Finance Payment if Periods Deferred > 0.


If "Total Brokerage Included" is set to ‘yes’ the Lessee Rate calculation includes:
"Depreciable Amount"
PLUS
Total Brokerage amount
MINUS
If "Broker Fee Included" = ‘no’ then Broker Fee amount else "0.00"
MINUS
If "Commission Fee Included" = ‘no’ then Commission Fee amount else "0.00"
MINUS
If "Establishment / Admin Fee Included" = ‘no’ then Establishment / Admin Fee amount else "0.00"
MINUS
If "Originator Fee Included" = ‘no’ then Originator Fee amount else "0.00"
MINUS
One month’s Finance Payment if Periods Deferred > 0.

[edit]
Agreed Lessee Rate

You can define the "Lessee Rate" you want for quotes by Client.
All quotes performed will then check the Clients / Quote Defaults "Agreed Lessee Rate" field and if populated, the "Commission Fee" of the quote will be modified to deliver the required "Lessee Rate" for the quote. This will, in turn, affect the finance budget and repayments required for the quoted lease.
The quote "Commission Rate / Fee" value should become read-only if an this feature is being used.

Existing quotes will not be affected by a value being entered into Clients / Quote Defaults "Agreed Lessee Rate" unless the quote is edited and a re-calculation is triggered.

The "Agreed Lessee Rate" affects the calculation of the "Commission Rate / Fee" value.
Visit the Commission page for the calculation details or open the commission model to see the calculation that is applied.

[edit]
Set-up Notes

This feature does not require any set-up, but you can restrict access to this field in [gb_role_restrictions] to prevent users from entering values into "Agreed Lessee Rate", if you do not want to use the feature.

[edit]
Lessee Rate Include VMP (Vehicle Maintenance Program) calculation

If Clients / Quote Defaults "Lessee Rate Include VMP" is checked on, the “Total Budget” amount (incl GST) for “Maintenance Program (VMP)” is to be added to the Lessee Rate basis.
