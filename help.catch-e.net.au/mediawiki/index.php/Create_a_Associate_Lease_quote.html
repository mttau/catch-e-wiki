Create a Associate Lease quote

Main Page → Workflow → Salary Packaging → Items → Create a Associate Lease quote

Qualifiers

  1. An employee would like an Associate Lease

Process

  1. Navigate to Fleet / Administration / Quotes
  2. Click on
  3. You will be navigated to the Quotes / Contract tab
    1. Type Code - enter 'AL'
    2. Term (Months) - enter the term in months
    3. Contract KMs - Enter the expected kms the vehicle will do over the term. This affects the calculated costs for budgets like Fuel, Maintenance and Tyres.
    4. Delivery State - Enter the vehicles State here.
    5. Client Code - enter the Employer's Code
    6. Comprehensive Insurance - leave this checked if the lease includes Comprehensive Insurance
    7. Policy Owner - choose 'Driver Policy'
    8. Insurer - choose the Insurer if known, but it's not used in Salary Packaging processes
    9. Driver ID - enter or lookup the Employee's ID or
    10. Surname - enter the Employee's Surname
    11. Firstname - enter the Employee's First name - the system will find and suggest this Employee's record if it exists.
    12. FBT Type - choose between 'Statutory' or 'Operating'
    13. Bus. Use % - if the FBT Type is 'Operating', enter the Business % here
    14. Use ECM - leave this checked
    15. Help Debt - ignore this field, your Package Statement refers to the Employees / Contact Help Debt field for this detail
  4. Navigate to the Quotes / Vehicle tab
    1. Select vehicle details
    2. Model Year - if the model year is missing, select the closest match
    3. New / Used - change this to 'Used' if the vehicle is used. This affects the calculated costs for budgets like Fuel, Maintenance and Tyres.
    4. Odometer - all used vehicles will need an odometer reading entered here. This affects the calculated costs for budgets like Fuel, Maintenance and Tyres.
    5. Master Quote Lock - check this, all other locks will check. Your changes will not be re-calculated by the system.
    6. List Price - enter the list price
    7. FBT Value - if the FBT Type is 'Statutory', enter the FBT Value here.
    8. FBT Value Lock If the FBT Type is 'Operating', un-check the lock and leave this field to calculate automatically.
  5. Navigate to the Quotes / Finance tab
    1. Months Deferred - update to '0'
    2. Luxury Car Charge- review this amount and remove it if it is not applicable. The value can be displayed on the quote to inform the Associate, but is is not used in the packaging process.
    3. Residual Inc GST - update to '0' unless there is a residual to use in the calculation.
  6. Navigate to the Quotes / Lease tab
  7. Review the budgets and modify the values if required.
  8. Click on
