Overview

Configure the Carbon Emissions Offset (CEO) budget profile by adding 'CEO' to the Contract Types / Accrual tab.
This budget profile is a calculated read-only field.
Use the Clients / Quote Defaults "Carbon Emission Offset" check box to set a client preference for this profile.

[edit]

Calculation Method

Choose which calculation method you would like to use. If both controls are populated, Amount per tonne is used as the default.

Fixed amount per month
Enter a default monthly net amount in [gb_controls]carbon_emissions_offset.

Amount per tonne
Enter a per tonne cost in [gb_controls] carbon_emissions_offset_per_tonne.
The amount stored in the variant's "CO2 Emissions" field is used to calculate the total budget which is then divided into months.

"Monthly Budget" = [qt_variants]co2_emissions * [qt_quotes]contract_kms / 1000 / 1000 * [gb_controls]carbon_emissions_offset_per_tonne / ROUND((Term - Months Deferred),2) * Term

Quotes / Lease - Carbon Emission Offset (CEO) does not calculate

Your Carbon Emissions Offset (CEO) budget will not calculate if:

  1. the amount stored in the variant's "CO2 Emissions" field is blank.
  2. you are using the [gb_controls] carbon_emissions_offset_per_tonne calculation method.

If this is the case:

  1. Ask your data provider (RedBook or Glasses) to advise you of the amount.
