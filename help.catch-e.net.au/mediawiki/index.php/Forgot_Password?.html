Overview

If the Forgot Username? Forgot Password? feature is enabled, users can reset a forgotten password here.
If your users have single signon access, this page is bypassed.

[edit]

Forgot Password?

Field Type Description [table]field
UsernameTextHelper text: Enter username
Email address Text Helper text: Enter email address
Reset Button Click to reset.
If the details cannot be validated, the user will see an alert.
If the validations pass, the user is sent an email containing a url link to the Reset password page.
The email is sent to the first email address stored in the user's "Email" record.


[edit]

Password reset

Field Type Description [table]field
Message Text The screen message comes from a control and can be customised.
The default text is shown below:

Thank you. An e-mail will be sent to your registered e-mail address shortly with further instructions for resetting your password. If you do not receive an e-mail within 10 minutes please call your system administrator.

Return to Login Button Click to return to the Login screen once the message has been read.
