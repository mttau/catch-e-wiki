[edit]

Overview

Exception report to review for contracts that cannot be added to the daily vehicle file.
Go to the AutoGuru page for details about all of the jobs, the process and for set-up steps.

[edit]

Selection Criteria

  1. Contracts / Contract "Reg Number" is not 'ORDER'
  2. Contracts / Contract "Contract Start" is today or earlier
  3. Contracts / Contract "Suspend Date" is blank or after today
    1. Contract Types / Details "Autoguru Export" - is checked or
    2. Contracts / Vehicle "Autoguru Export" - is checked
  4. An exception rule has been triggered in the report (see list below).


[edit]

Exceptions

Exceptions Comments
1 Contracts are excluded from the Autoguru daily vehicle file because of a data issue. Immediate attention required.
1 Build Date field data not in export format Contracts / Vehicle "Build Date" - edit and store as MM/YYYY
1 Vin is duplicated Contracts / Vehicle "VIN"
1 Vin is not 17 characters Contracts / Vehicle "VIN" - remove VIN if it is incorrect.
1 Vin is not 17 characters and Vin is duplicated Contracts / Vehicle "VIN" - update VIN or remove VIN if it is incorrect.
1 Comma in Manufacturer Name field Contracts / Vehicle "Manufacturer Name" - edit to remove comma.
1 Comma in Model Name field Contracts / Vehicle "Model Name" - edit to remove comma.
2 Contract are included in the Autoguru daily vehicle file, but data issue exists that should be fixed. Immediate attention required.
2 Vin is missing for novated contract Contracts / Vehicle "VIN" - add correct vin.
2 Vin is missing for lease contract Contracts / Vehicle "VIN" - add correct vin.
3 Contract are included in the Autoguru daily vehicle file. Note missing VIN.
3 Vin is missing for fleet managed vehicle Contract will be sent with a "FleetCompanyIdentifier" instead of a VIN. A VIN should not be added to these contracts.
4 Contract excluded from Autoguru daily vehicle file where Contract Type and Contract settings are not both checked.
4 Contract is un-checked for lease contract
4 Contract is un-checked for fleet managed vehicle
4 Contract is un-checked for novated contract Contracts / Vehicle "AutoGuru Export" - should be checked for all active novated contracts.
4 Contract Type un-checked Contracts / Vehicle "AutoGuru Export" - uncheck these contract.
Contracts will never be selected for the Autoguru daily vehicle file if they belong to a Contract Type that is not checked.
