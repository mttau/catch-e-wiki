[edit]

Reset Password

If you can see a Reset Password hyperlink on your Login screen, use this to reset your password if you have forgotten it or you've become locked out.
If your login screen does not show this, contact your System Administrator for assistance if you can't login.


[edit]

Getting locked out

If you make three failed attempts to login, you will be locked out.

[edit]

Log in to different environments

[edit]

Live

Live is available to you 24 hours a day.
Your Live environment has scheduler jobs and email functions working.
Use the Password Reset feature in this environment if you need to reset your password.

[edit]

Support

Your Support environment is an environment where a copy of your Live data has been copied into each night.
This means it has the same login credentials as Live, only the URL to access the environment is different.

The Scheduler and the Emailing functions do not run in Support
If you are testing in Support, you will need to

  1. run any jobs you need to see working manually
  2. go to the Mail Queue to see the emails have generated.
[edit]

Staging

Staging is available 6am - 10pm EST.
Staging has it's own login credentials, but you can maintain them to be the same as Live if desired.
If you have trouble when logging into Staging, it's possible that Staging has been refreshed from Live or Support data

  1. So try your Live login details and see if they work before escalating.
[edit]

Test

Test is available 6am - 10pm EST.
Test is only available if you have a VPN Connection to Catch-e running.

[edit]

Password Resets

  1. Use the Reset Password process to reset your password in the LIVE environment.
  2. The Reset Password process will not work in other environments, as they will not send emails
    1. For other environments, ask another 'Admin' user in your organisation to check and update your password in STAGING or TEST
    2. If another 'Admin' user is not available - email Catch-e Support and they will assist


[edit]

Log in with Microsoft Account

If Log in with Microsoft Account is shown on your Login screen, click on it to go directly to the Home screen.
This feature uses your Office 365 account credentials to perform a single sign-on.
If you have 2 Factor Authentication enabled, you will be taken to the authentication screen once this sign-on has completed.

You may be able to continue to use the standard login method I.e.entering a "User ID" and "Password".
If your organisation wants to use the Log in with Microsoft Account method exclusively, then your password will be removed and the standard login method will no longer work.

Troubleshooting


Visit the main Troubleshooting page for a list of all the available problem-solving tips.


[edit]
403 Forbidden

If you get a '403 Forbidden' message when you are logging in, you are trying to log in from a Geo-blocked location.
To login, you will need to either

  1. Run a VPN connection from an unblocked location (like Australia)
  2. Arrange for the IP addressg to be whitelisted. This can only be done if you are running a static IP address. Go to the process page Whitelist an IP address for the steps to do this.
[edit]
Invalid token

If a user clicks on the URL link sent by the Forgot Password? feature and gets this error, it means either

  1. They have clicked on the link previously (it will only work once) or
  2. Their email system has triggered the URL while running security tests for phishing (thus using up the one-time link)

In both cases, the easiest way to manage this is to go to Users / Password and Issue a temporary password to the user.

[edit]
My browser tells me to change this password

Your browser has a monitoring tool enabled that checks the passwords you enter against databases that store known stolen credentials.

If your browser alerts you about this, it means your login password is no longer secure.
You can confirm this by using a site like haveibeenpwned.com/Passwords to check this.

Below are our recommendations if you get this message:

  1. Change your password immediately
  2. Use a good password manager to help you generate and maintain strong, unique passwords for each different system or website you use.
  3. Maintain a unique password for your Catch-e login.
  4. Do not use your Catch-e login password on any other system or website.
