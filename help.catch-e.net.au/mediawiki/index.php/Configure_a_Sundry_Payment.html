Main Page → Workflow → Fleet Management → Set Up → Configure a Sundry Payment

Qualifiers

You plan to may periodic payments to a supplier. E.g. Lease Inclusion Plans Setup


Process

  1. Navigate to Posting Classes / Details
  2. Look up the class you want to use or create one if needed
  3. Click on or
    1. Class code - enter a suitable code. It cannot be a pre-existing coded event.
    2. Name - enter a suitable name
    3. Payment - check on the check box. Only checked classes can be added to the Contract Types / Payments tab.
    4. Supplier - add a supplier code if you want a default supplier for 'Sundry Payments'. E.g. Lease Inclusion Plans Setup
  4. Click on or
  5. Navigate to Accounts / Contract Types
  6. Type code - lookup the Contract Type you are using
  7. Navigate to Contract Types / Payments
  8. Click on
    1. Class - select the required class from the available list
    2. Type - 'Sundry Payment'
    3. Debit - Balance Sheet Provision GL Account.
    4. If you are using supplier-level GL Codes, go to Suppliers / Account GL Code to enter the code.
    5. Credit - Creditors Control GL Account
    6. G - 'Y' or 'N' to calculate and add gst to payment records.
  9. Click on
