Create and add a bank account to an item

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Items → Update → Create and add a bank account to an item

[edit]

Qualifiers

  1. You are modifying an Employee / Item record
  2. Only the Employee's Bank Account needs to be updated


[edit]

Process

  1. Go to Employee / Item
  2. Click on
  3. Click on
    1. Account Name - enter. This field is optional and can remain blank.
    2. BSB - enter
    3. Account Number - enter
    4. Claim - check if this account can be selected in the Contracts / Claim tab
    5. Default - check if this is the default account for claims raised in the Contracts / Claim tab or in the Driver Portal
  4. Click on to create the bank account record and add it to the item
