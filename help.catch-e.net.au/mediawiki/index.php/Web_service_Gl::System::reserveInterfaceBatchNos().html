Web service Gl::System::reserveInterfaceBatchNos()

From Help wiki

Jump to: navigation, search

Main Page → Web Services → Global (gb) → reserveInterfaceBatchNos()

Synopsis

Reserve a range of mail batch numbers.

Input Fields


  Query string key
Format
Notes
Mandatory
1 mail_batch_nos
integer
The number of mail_batch_nos to reserve
yes


Input URL Example 

https://yourname.catch-e.net.au/services/gb/system/reserveInterfaceBatchNos?interface_batch_nos=3


Output Fields


  XML element
Format
 Notes
1 return_status
bool
 1 on success
1 interface_batch_no_start
integer
 Reserved interface_batch_no start
1 interface_batch_no_end
integer
 Reserved interface_batch_no end


Output Example

<?xml version="1.0" encoding="UTF-8"?> <response>

<return_status>1</return_status>
<interface_batch_no_start>128252</interface_batch_no_start>
<interface_batch_no_end>128254</interface_batch_no_end>

</response>



Error Codes

  Error Codes Notes
1 CATCH_E_ERROR_INTERFACE_BATCH_NOS_REQUIRED The mandatory Input Field interface_batch_nos is missing.
2 CATCH_E_ERROR_INTERFACE_BATCH_NOS_INVALID The Input Field interface_batch_nos does not contain a positive integer.


Error Output


<?xml version="1.0" encoding="UTF-8"?>

<response>

<error>CATCH_E_ERROR_INVALID_INTERFACE_BATCH_NOS</error>

</response>
