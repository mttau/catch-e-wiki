Buttons

Button Description
Click to create a new quote. You will be navigated to the Quotes / Contract tab in edit mode.
Click to copy an existing quote. Use this when you want to create variations of a quote.
The "Effective Date" of the quote remains the same as the copied quote but the "Quote Date" is updated to today's date. The "Quote Date" is normally shown on quote PDFs.
Visit the Quotes → Copy page for all the behaviour and set-up details.
Accept a quote to create a contract with the status of "Order" in the Contracts area.
Once a quote has been accepted the "Status" field in the header becomes a pop-up hyperlink. This allows you to quickly navigate to any contract that was accepted from the quote.
Your system can be configured to allow multiple contracts to be created from a quote, or restricted to allow only one contract.
Visit the Quotes → Accept and Settle page for all the behaviour and set-up details.
Settle a quote to update the linked contract with updates made in the quote. This is useful if the quote has been changed (e.g. budget changes) and the contract needs to be updated.
If an Contract Types / Accruals Posting Class has been added after the contract was originally created, "Settle" will add the posting class into the contract's budget records.

The Settle button is only displayed if

  1. The quote has been accepted
  2. There is only one linked contract

The Settle button will not run if the Contract Type on the quote and contract do not match.
If there is a mismatch, you will be given an Alert: "Contract Type does not match in Contract, cannot Settle!"
Visit the Quotes → Accept and Settle page for more behaviour and set-up details for Settle.
