Main Page → Workflow → Salary Packaging → Adjustments → Package adjustments | Adjustment claims | Adjustment approvals | Adjustment approvals for periodic roundings

Qualifiers

  1. You need to create an adjustment to correct the transaction balance of a benefit item
    1. E.g. you have received a refund or money was returned
  2. The employee has an Employees / Item record where
    1. Status is 'Active'
    2. Disbursement Type is 'Claim' or Recurring'
    3. Contract ID is populated


Process

  1. Navigate to Fleet / Administration / Approvals → Approval
  2. Click on
  3. Supplier- enter or select the Supplier
  4. Reference - enter a reference
  5. Reference Date - select the date that the transaction belongs to
  6. Row # 1
    1. Class - select the appropriate option from the list
    2. Description - enter a description of the adjustment
  7. Price - enter the total excluding GST (if used). This can be a positive or negative number.
  8. GST - GST will calculate if required and can be edited to match if the rounding is not correct. It will set to 0 if the claim excludes GST.
  9. Click on
  10. Click on
  11. Attach the supporting file you have for the adjustment being made
  12. Click on
  13. Click on or


Next Steps

  1. The transaction will be added or, subtracted from the benefit item's balance
