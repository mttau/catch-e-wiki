[edit]

Overview

This table contains rates per financier and per lease term.

Go to the Financier page for a complete list of steps to follow to set-up a new financier.

[edit]

Filters

Filter NameTypeDescription
Financier CodeLookupEnter a code or look up the financier record and press Go to return records for a particular financier. Leave this field empty and press Go to see all stored records.


[edit]

Field Descriptions

Note: The Lower and Upper Limits do not work. Only one Interest Rate per Financier/Term can be applied.


Field NameTypeDescription[table]field
IDNumUnique identifier for this record. This is a display-only field.[fm_financier_rates]financier_rate_id
FinancierTextEnter the "Supplier Code" of the financier you are adding a record for.[fm_suppliers]supplier_code
NameTextDisplays the "Name" of the financier[fm_suppliers]name
TermNumEnter the term that the interest rate applies for. "Term" cannot be 0. This will be used on quotes that have a term that is on or below this term. E.g. a quote with a "Term" of '13' would use the interest rate stored on the record where "Term" is '24'.[fm_financier_rates]term
Lower LimitNumDefaults to 0.The Lower and Upper Limits do not work. Only one Interest Rate per Financier / Term can be applied. [fm_financier_rates]lower_limit
Upper LimitNumMandatory, e.g. 9,999,999,999.99[fm_financier_rates]upper_limit
Interest Rate% (4dp)Enter the Interest Rate to be used. This will populate into quotes that have a matching financer and term as the Quotes / Finance "Base Rate".[fm_financier_rates]interest_rate
EffectiveDateEnter the effective date for new records. The date defaults to today's date, but this can be edited. When records are being matched to a quote, the record with a date that is on or before the quote's "Effective Date" will be selected.[fm_financier_rates]date

Action Buttons

Field Description
Press 'Go' to execute your chosen filtering. The page will refresh with the selected transactions and these can be viewed page by page. If 'Go' is greyed out, then the current filtering has been executed.
Press 'Edit' to adjust a record. We recommend only using this to correct errors in a record you have created.
Press 'New' to create a new record. Do this when making a change to the values that will default into a quote. Creating a new record for each change will allow you to filter and review changes you have made to each Financier / Term combination over time.
