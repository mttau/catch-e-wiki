Overview

A Maintenance Program profile and budget will be available in the Quotes / Lease tab if there is a linked Contract Type / Accruals "Class" of 'VMP'.
A default amount can be set in the Clients / Quote Defaults "Maintenance Program Fee" field.

If populated, the amount will populate into new quotes and the profile will be selected.
If '0', new quotes will have the profile un-checked and the amount field disabled.

You can edit the quote inclusion check box and amount as required.


[edit]

Set Up Steps

[edit]
Posting Class

Check that the Posting Class that is required for this profile is set-up correctly.

  1. Go to Posting Classes / Details
  2. Enter 'VMP' into the "Class Code" field and tab out
  3. If 'VMP' exists, great. Also check the details described to create this record are correct for the existing code.

To create a new posting class

  1. Click on New
  2. Enter class code 'VMP' ('VMP' is a reserved code that helps keep consistency throughout quotes and reports).
  3. Enter "Name" as 'Maintenance Program' (you can nominate another name if you prefer, go to Posting Classes to find out more about how the "Name" works).
  4. For "Quote Profile Table", select 'qt_quote_vehicle_maintenance_program' from the list.
[edit]
Contract Types

Add the posting class 'VMP' to any Contract Type records you want to add this budget to. Remember that you'll need to update quote documents to accommodate this budget.
Go to menu Fleet / Controls / Accounts / Contract Types

  1. Enter or search for the "Type code" you want
  2. Go to the Contract Types / Accruals tab
  3. Click on Edit
  4. On a new row, in the "Class" column, select 'VMP' from the list box
  5. Enter the remaining set-up details in the right of screen and Save



Note: New quotes you create now will have the product available. Existing quotes will need to be edited to add 'VMP' to the quote.
