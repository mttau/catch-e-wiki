[edit]

Overview

Access Type: Internal
Restriction: The 'senior manager' has full system access like 'admin', except for the following:

  1. Setup / Reference Data - can access all tables except gb_controls
  2. Setup / Import or Export Files - no access
  3. Setup / Import Table Data - no access
  4. Setup / Export Table Data - no access
  5. System / Users - no access
  6. System / Roles - no access
  7. System / Workgroups - no access

Set Up: This user role type is set up by an existing system Administrator.
