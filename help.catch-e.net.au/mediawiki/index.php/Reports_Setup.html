Report Selection List

Individual reports can be removed from the "Report Title" list by detailing them in [gb_controls]reports_not_available.
A default exclusion is 'asset_list'.

[edit]

Logos

Each logo file is stored in a client folder reserved for logos. Each Standard Report is stored in gb_templates. The templates for Driver Reports also store email content. The logo configuration is stored in gb_template_placeholders. The configuration records define the position, size, and resolution of the logo. See an illustration of these records below.

[edit]

Channel Logos

Channel logos can be configured if required. This is done by creating a new channel folder and storing their logo files in that folder.
The channel needs to be identified by populating [gb_templates]channel_id, which is set to '0' for standard reports.
See also Driver Report Setup to see the setup required for channel emails.

Some clients have client-specific reports, so their logos are not stored here and require development to be changed.

Contact your Account Manager to add, change or adjust report logos.

[edit]

Logo Adjustment

On setup, Catch-e systems are built with 'standard' logo records. These 'standard' records work with all logos that share these aspect ratios:

logo_s.png 515:100
logo_m.png 1257:244

Ideally the logo that is used match the aspect ratios above so the reports work 'out of the box' and no further adjustments are required.

If the logo files require different aspect ratios the records will need to be adjusted either by script or by Whole Table Import. Scripts are stored by Catch-e internally at Client Support → Data Updates → Report Templates.


When adding or adjusting logos, all reports need to be tested because the logo has a fixed starting position on the report.
Therefore depending on the aspect ratio of the logo (rectangular, wide and sort, narrow and long etc), this fixed positioning may have to be changed so that all channel logos of different aspect ratios present nicely on all reports.
