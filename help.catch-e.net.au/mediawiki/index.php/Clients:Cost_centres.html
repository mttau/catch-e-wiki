Overview

The Clients / Cost Centres tab lists the cost centres used by the client.
A cost centre can be stored against quotes and contracts.
Once a cost centre has been used elsewhere in the system, the record cannot be deleted from this screen.

Invoices are sorted, grouped and sub totalled by Cost Centre.
Cost centres can be used as a filter in the Contracts / Search screen.
Cost centres can be used as a filter in many Administration / Reports and Reporting / Standard Reports.

Clients / Cost Centres is similar to Clients / Divisions but the two functions operate independently.

[edit]

Quotes

You can require the Quotes / Contract "Cost Centre" field to be populated for a client.
If a cost centre has not be entered, the user will be alerted and asked to select a correct entry before the quote can be saved.
For this to work, the check box "Cost Centres Mandatory" needs to be checked in the Clients / Quote Defaults tab.

[edit]

Contracts

Each time a vehicle changes to a new cost centre, add a Contracts / Events Cost Centre Change record.

[edit]

Using the cost centre

"using the cost centre." means the Contracts / Events, Clients / Events, Bookings or Quotes / Contract has a record of the cost centre.
The record may not be the current allocated cost centre.
These records are counted to prevent editing or deleting cost centres which have been used and are part of records available for reporting purposes (e.g. Cost Detail and Cost Summary reports).


[edit]

Output Options

You can print a XLS or CSV report of the cost centres. The output fields are different depending on whether the Clients/Cost Centre screen is enabled or not.
If disabled: Client Code, Client Name and Cost Centre.
If enabled: Client Code, Client Name, Cost Centre, Client Manager, No of FTEs, Site Visits, Name, Job Title, Phone, Email, Address (Line 1), Address (Line2), Area, State, Postcode, Comments.

[edit]

Set-up Notes

A list of a client's cost centre records can be called using the getClientCostCentres API.

The Clients / Cost Centre tab may be hidden in your system, but it can be enabled in the gb_role_restrictions table.
Contact Catch-e Support if you want to access this tab.
