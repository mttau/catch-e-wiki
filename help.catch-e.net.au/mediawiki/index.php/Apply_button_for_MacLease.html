Apply button for MacLease

From Help wiki

Jump to: navigation, search

Main Page → Workflow → System Administration → User Management → Apply button for MacLease

Qualifiers

  1. You want to give a user, or users access to use the Quotes / Finance button.
  2. The Maclease Partner Interface feature is set-up in your system
  3. Your business is listed with Macquarie Leasing as using API Partner services.
  4. You (a System Administrator) has received an email from Macquarie with the following details for each user or,
    1. Firm
    2. Broker Name
    3. Mac ID
    4. API Token
  5. One of your users has forwarded you an email from Macquarie with the following details
    1. Firm
    2. Mac ID
    3. API Token


Enable a user's access to the Apply button

  1. Navigate to System / Users
  2. User Login - lookup or enter the user's login
  3. Navigate to Users / Login
  4. Service - select 'Maclease Partner'
  5. Login - enter the user's Mac ID
  6. Password - enter the user's API Token
  7. Click on the button.


Disable a user's access to the Apply button

  1. Navigate to System / Users'
  2. User Login - lookup or enter the user's login
  3. Navigate to Users / Login
  4. # - find and select the 'Maclease Partner' service record
  5. Click on the button.
  6. Status - select 'Inactive'
  7. Click on the button.
