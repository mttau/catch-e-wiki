Overview

Use this report to review and manage Roadside Assistance information for contracts. Contracts are identified for this report if Contracts / Vehicle "Roadside Managed" is checked.
The report can be printed in two different formats: office(xls) format and text(csv) format
It is sorted by Supplier, Client group, Client, Division, Cost Centre and Registration.

[edit]

Report Filters

  • Roadside End Date - the default 'End Date' is the last day of the previous month. These dates can be edited.
  • Supplier - the report can be limited to displaying information for one supplier.
  • Client Group - the report can be limited to select only Clients from a particular Group.
  • Client - the report can be limited to select only a particular Client.
  • Division - the report can be limited to select only a particular Division.
  • Cost Centre - the report can be limited to select only a particular Cost Centre.
  • Contract Type - the report can be limited to select only a particular Contract Type.
  • Roadside Managed - By Default this checkbox is ticked. You can uncheck to filter your report. If "Roadside Managed" is checked ON then only contracts that have "Roadside Managed" checked on in Contracts/Vehicle will be shown in the report.
