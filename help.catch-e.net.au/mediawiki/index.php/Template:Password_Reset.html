[edit]

Is the temporary password feature in use?

A password reset works in different ways depending on whether the temporary password feature is in use. You can tell by doing the following

  1. Click on Reset
  2. If you can't see
    1. the
    2. the "Temporary Password" check box
    3. the Send Password button

Then temporary password is not in use. Go and set it up â€¦it's handy and it's our recommended approach.

[edit]

Reset a password

  1. Go to Users / Password
  2. Click Reset
  3. Enter and re-enter a new password which is allow 64 characters or less.
  4. Click Save
  5. Let the user know what the new password is
[edit]

What does the user do next?

  1. On Login, the user is navigated to the Home screen. they are not required to change their password.
  2. They can go to Users / Password to change their password at any time.


[edit]

Reset a temporary password

  1. Go to Users / Password
  2. Click Reset
  3. Generate a random password, click on the .
  4. You can enter and re-enter a new password directly (this might be better than using a random password if you're on the phone to the user and want to relay a password that's easy to remember).
  5. Passord field must allow 64 characters or less
  6. Click Save
  7. Click Send Password if you want to email the user to advise them of the password.
[edit]

While a temporary password exists

  1. Until the user has updated their password, the temporary password is visible to you on the Users / Details tab
  2. You can re-send the advice email from either the Users / Details tab or the Users / Password tab at any time.
[edit]

What does the user do next with a temporary password?

  1. On Login, the user is navigated to the Users / Password tab and must change their password before they can continue to use the system.
  2. Once the user has updated their password, Users / Details is updated so that:
    1. the "Password" field is hidden
    2. the "Temporary Password" field is un-checked
    3. the Send Password button is hidden
