Main Page → Workflow → System Administration → Global Controls → Create a global control

Qualifiers

  1. You need to create a new record for an existing control where a new Effective From date is required
  2. In most cases, updating a global control is recommended
  3. You have read the relevant set-up documentation
  4. We recommend first testing your changes here in your Staging or Support environment


Process

  1. Go to # Go to Setup / Reference Data
  2. Table - select 'gb_controls'
  3. Name - enter the name, or part of the name of the control you want to change
  4. Click on
  5. Name - click on the hyperlink of the target record
  6. You will be navigated to the Reference Data / Detail tab
  7. Click on
  8. Effective From - enter the date the new control should take effect
  9. Value - enter the required value
  10. Click on
