[edit]

Overview

If the list of possible variants to choose from in Quotes / Vehicle is still a long one after you have chosen the "Make", "Model", "Year" and "Body Type", use this pop-up to filter on five key vehicle features to narrow down the list of variants to choose from.

If there are no filter choices available for the field, it will be display only.
Change the filter away from the opening choice of '--All--' to the choice you want.
Some filters may change to display only if filtering in other fields has removed the subsequent filter choice.
Re-set filters by returning the drop list choice back to '–All—' or close the pop-up and re-open it.

When you have not yet selected a variant, the lookup icon takes you to the Variant Code Filter pop-up where you can enter a variant code directly if you know it.
