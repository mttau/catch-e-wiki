[edit]

Overview

Allows you to automate payments from the Contracts/Payments tab .

[edit]

Permission

The permission for this api is stored in [gb_api_permissions]permission as 'EmployeePayments: Create '.
Only roles with a stored link to this permission in [gb_role_api_permissions] can run this api. By default 'admin' and 'web-services' have already been stored.

[edit]

Authentication

The APIs on this page use an access_token as the authorisation key. Use the Authenticate with the API function below to generate the access_token needed to use the other APIs.

MethodComments
Authenticate with the APIRequest an API token for use with secured methods.
Generate a signon token Request a signon token to enable single signon for a user.


[edit]

HTTP Method

Use the HTTP Method 'POST' for consuming this web service.
