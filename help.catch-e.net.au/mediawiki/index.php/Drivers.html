Overview

The Driver module is used to store driver details, generate login access and links to their contracts.
Your system can be configured to provide access to a Driver Portal for drivers to lodge reimbursement claims and view their vehicle balance.

You can also use the Packaging module if you want to hold and report to packaging employees on salary packaging data from other systems.
The data stored for drivers and employees are held in the same underlying tables, but the fields shown in the screens are different.

If the driver has an active vehicle contract, additional vehicle screens are displayed which are identical to the following contract tabs.

  • Vehicle Events is the same as Contracts / Events
  • Vehicle FBT is the same as Contracts / FBT
  • Vehicle Maintenance is the same as Contracts / Maintenance
  • Vehicle Claim is the same as Contracts / Claim
