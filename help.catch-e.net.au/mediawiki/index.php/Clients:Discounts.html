This screen displays the discount level per manufacturer and any confidential rebates per variant. The discount level and confidential rebates are applied automatically to quotes where the relevant make and variant are selected and the discount data tables are populated.

Edit allows changes to existing records.
New allows you to create new records.
A Model and Model Year need to be selected in the top filter section to allow new Confidential Rebates to be added. By selecting the Model and Model year any available (i.e. not 'inactive') variants are presented for creation.

Tables updated on this screen are:

[qt_client_make_discounts] (Discount level per make).
[qt_variant_rebates] (Confidential Rebate per variant per client). Note this table is also able to be updated at SETUP > Reference Data



Note: Quotes / Vehicle "Dealer Rebates" are not updated by this function.
