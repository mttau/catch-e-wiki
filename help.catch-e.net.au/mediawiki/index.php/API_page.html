Permissions

This permission and API are not available by default.
Contact your Account Manager to get access to this API.

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on Add permission here.

Response Details

Validation Messages Comments
200 OK 
{
    "quote_id": 132657
}
The request was a successful request.
401 - Unauthorized 
{
    "type": "http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html",
    "title": "Unauthorized",
    "status": 401,
    "detail": "Unauthorized"
}
  1. You have not authenticated before running this API or
  2. The token_timeout of the current session has passed. You need to authenticate again.
403 - Forbidden 
{
    "type": "http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html",
    "title": "Forbidden",
    "status": 403,
    "detail": "Forbidden"
}
You do not have permissions for this request.
  1. Go to System Roles and enter 'web_services'
  2. Navigate to the Roles / APIs tab to make sure the permission you need to run this API is checked.

        "annual_kilometreS": {

Input field annual_kilometres is missing.

    "type": "http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html",
    "title": "Failed Dependency",

    "detail": "Upstream service temporarily unavailable. Please retry."
}

  1. service resource limits have been exceeded
  2. the system is in maintenance mode
