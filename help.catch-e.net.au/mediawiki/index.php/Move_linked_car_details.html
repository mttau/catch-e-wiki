Move linked car details

Main Page → Workflow → Salary Packaging → Items → Car Details → Move linked car details

Qualifiers

  1. Your system has been set-up with to hold a benefit contract and a separate vehicle contract
  2. You have linked an item to a vehicle contract
  3. The vehicle contract has
    1. been created against the wrong client OR
    2. the employee has moved to a different client within a group


Remove Car Details from the existing item

  1. Navigate to Employees / Item
  2. Click on
  3. Car Details
    1. Quote ID - blank out the Quote ID of the vehicle quote
    2. Contract ID - blank out the Contract ID of the vehicle contract
  4. Click on


Update Vehicle Contract

  1. Go to Contracts / Contract
    1. Client - update to show the new client
  2. Go to Contracts / Events
    1. Create 'Driver Change' event and pick the new driver record


Update Vehicle Quote

  1. Go to Quotes / Contract
    1. Client Code - update to show the new client
    2. Driver ID - update to show the new driver


Create New Item

  1. Create an item for the new employee
  2. Car Details - select and add the updated vehicle contract


Troubleshooting

Visit the main Troubleshooting page for a list of all the available problem-solving tips.


Car Details - unable to add quote or contract

A quote and/or contract record cannot be added to an item if it is already linked to another item.
Follow the process steps here: Move linked car details to firstly un-link the contract then link to the new contract.

As a last resort, contact Catch-e Support and note the Contract ID that needs to be unlinked and they will do this.
