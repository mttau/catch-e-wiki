Overview

Use the Current Packages tab to record employee benefits.
Visit Drivers / Current Packages for the driver version of this page.

[edit]

Filters

Filter Type Description [table]field
FBT Year Ending List The default selection is '--All--'. Choose an FBT Year if you want to review items that are dated within the chosen FBT year.
Status List The default selection is '--All--'. Choose a status if you want to review only items in the chosen status.

Action Buttons

Field Description
Print is disabled if your report selection is for 'Package Activity Statement' and the field "FBT Year Ending" is set to '--All--' or when there are no records on screen to print.
Select the record you want to edit by checking the "#" of it's row. Click on Delete to delete the item.
Items that are linked to a contract cannot be deleted. The Delete button will be disabled for these items.
Deleted items are no longer visible on the Current Packages tab or the Item tab.
Select the record you want to edit by checking the "#" of it's row. Edit will become enabled.
Press New to create a new item.

Output Fields

Screen NameTypeDescription
Report NameListPackage Activity Statement is the default report. If you have Screen Reports set-up, they can also be chosen here.
Report MonthListDisplays as 'YYYY Month'
This list displays the months that fall within the chosen "FBT Year Ending" year and is ordered from April to March.
This field only applies to the 'Package Activity Statement '. It is not selectable for other reports.
Print Format Drop List Chose the print format you want to use.
PrintButtonThe print button creates the selected report. If the selected "Report Type" is 'Email', the Send Email pop-up will be launched.



[edit]

Output Options

Output NameDescription
Package Activity Statement Displays a summary of the package items you can see on the screen.
The output format will default to 'PDF', but you can choose from 'Email', 'PDF' or 'Text(CSV'.
If you select 'Email', the PDF report will be attached to a system email addressed to the driver.
Contract Transactions This report is a companion to the scheduler report SP - Contract Balance Checks.

It contains many of the same fields as SP - Contract Balance Checks but the information here is by transaction.
The report selects contract transactions for all the employee's benefit items.
The direct link vehicle contracts are not currently shown here.

Impact Statement This report is designed to illustrate the packaging benefits to the employee.
The report can be renamed to suit your business.
Ongoing maintenance is required to keep this report updated with the latest ATO PAYG Tax and HECS calculation factors.
Visit the Impact Statement page for more details.
Screen ReportsIf set up a customised report can be generated from here.
