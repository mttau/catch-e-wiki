Overview

The portal is set-up separately of the main Catch-e system depending on your organisation's needs.
It's known as the Driver Portal because the first features were tailored for use by the Driver role, but it now contains features for the Package Employee role also.
The menu items described here are for the Driver Portal. Visit the Employee Portal page for that list of menu items.

If the user is a driver, they are navigated to the Driver Portal when they login.
Entries made through the driver portal will be saved into system tables, just as they are when drivers enter records on the Claims or Events screens.
Audit history for these changes are also created when they are made in the driver portal.
Open the Driver Portal Guide for full details of portal features and screens.

Driver Portal Summary

Screen NameDescription
Login
Portal Login Opening page where driver logs in.
If the driver has been issued with a temporary password, they will be navigated to the Change password screen to update their password.
Forgot Username? If the Forgot Username? Forgot Password? feature is enabled, the Driver can click on Forgot Username? to see a message about who to contact to find out what their username is.
Forgot Password? If the Forgot Username? Forgot Password? feature is enabled, users can click on Forgot Password? to reset their password if forgotten.
Home
Raise a vehicle claimDrivers can submit claims here for costs incurred that can be reimbursed. If using a phone, the driver can use the phone camera to take a photo of the receipt.

If the driver is using an iPhone with an IOS 13+, they may need to update their settings for this to work.

Employee ContributionDrivers can can be redirected to make an employee contributions here instead of making claims if there is a shortfall.
View submitted vehicle claimsDisplays claims submitted by the driver and shows their status. Attached receipts can be downloaded.
View my vehicle balanceDrivers can check the current balance of their vehicle contracts.
Send an enquiryDrivers can email enquiries to you here. These emails can be stored as Contracts / Events for your staff to track and manage.
Change passwordDrivers can change their password here. If you setup access via single signon, this button can be hidden
2 Factor Authentication You can make 2 factor authentication available to drivers and it can be set to be either optional or mandatory. By default it is turned off.
Logout
Sidebar / Vehicles
Vehicle Details If set to ‘show’, drivers will see the menu option ‘Vehicle Details’ in the sidebar menu. When selected, drivers can access the Vehicle Details screen in the Driver Portal.

The sidebar menu is only available when the global control ‘driver_portal_sidebar_enabled_flag’ = ‘yes’.

Finance Details If set to ‘show’, drivers will see the menu option ‘Finance Details’ in the sidebar menu. When selected, drivers can access the Finance Details screen in the Driver Portal.

The sidebar menu is only available when the global control ‘driver_portal_sidebar_enabled_flag’ = ‘yes’.

Sidebar / User
Contact Details If enabled, drivers can view and update their Contact Details.
When logging in for the first time, you will be asked to update and/or confirm your current details.
Employment Details If enabled, drivers can view and update their Employment Details.
When logging in for the first time, you will be asked to update and/or confirm your current details.
