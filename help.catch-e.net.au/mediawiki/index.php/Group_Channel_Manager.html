Overview

Access Type: Internal

Restriction: The Group Channel Manager can view quotes, drivers, events, and clients that are linked to clients that belong to the nominated "Channel Group Code"

Set Up:

  1. Restrict Key - enter the Channel Groups / Contact "Channel Group Code" that you want to restrict records to
  2. All clients that belong to channels that have the same channel group code stored in their Channels / Contact "Channel Group Code" field are visible to this role's users.
  3. The Restrict Range feature is not available for this role.
