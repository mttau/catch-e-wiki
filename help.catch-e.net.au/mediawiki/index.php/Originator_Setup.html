[edit]

Overview

If configured, the field "Originator" can be shown in Quotes / Finance or a Contracts / Finance.
If Quotes / Finance "Originator" is populated, Contracts / Finance "Originator" will be updated when a quote is accepted or settled.

Use the API getOriginators to get a list of originators.

[edit]

Setup

Use Import Table Data to upload records into [gb_originators].
Each Originator record must be linked to a Supplier record.

Field NameNotesMandatory
originator_idSystem Generated. Leave blank (or 0) when uploading via Import Table Datayes (system generated)
supplier_id A valid Supplier record must be populated here. This does not have to be the same supplier as the Quotes / Finance or a Contracts / Finance "Financier".
However, the "Originator" drop list will only show Originator records if they are linked to a Quote Financier or Contract Financier. 		yes
originator_code Report only field. Not used in Catch-e application. Character Limit = 10.yes
user_code Report only field. Not used in Catch-e application. Character Limit = 10.no
description This is the value shown in the Quotes / Finance and Contracts / Finance "Originator" field and list box. The character limit is 50.no
status_flag Set as 'active' or 'inactive'. Only active [gb_originators] records are shown in the list box.
If a Quotes / Finance or Contracts / Finance "Originator" record has become 'inactive', the record will continue to be displayed, but if removed, the inactive originator cannot be re-selected. 		yes
