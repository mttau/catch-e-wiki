[edit]

Overview

If a GL Codes is entered here, the supplier's purchase ledger transaction will be directed to this code instead of the standard code specified in the Contract Type.

This feature only works for Finance and Insurance payments (see Payments Setup).
All other payments (such as Maintenance, Registration or CTP) still refer to the posting maps in Contract Types.

Before you can enter a code into this field, it needs to be set-up in GL Codes / Details.
