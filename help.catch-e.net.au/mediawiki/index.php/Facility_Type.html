Overview

This list box is available in the following screens;

  1. Clients / MLA
  2. Clients / Financiers
  3. Contracts / Finance


[edit]

Set-Up

  1. The list is stored in the [fm_facility_types] table.
  2. Only records where [fm_facility_types]status_flag = 'active' can be selected in the list boxes.
  3. The value of [fm_facility_types]description is shown in the list boxes.


[edit]
Updating your list
  1. Follow the process in Export data from a table to download the current [fm_facility_types] table records.
  2. Use this as a template to create a Data File that contains your new records.
    1. Create a new row for each new record you want to show in the list.
    2. If you want to make a current record inactive, update the [fm_facility_types]status_flag to 'inactive' for thos records
    3. Remove any records that you don't want to add or change
  3. Follow the process in Import data to a table upload your additions and changes into [fm_facility_types]
  4. Make sure you upload this file into your Support environment first and check the list works as expected before you update your Live environment.
