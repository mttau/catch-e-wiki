Overview

The booking screen is used to display bookings. From this screen a new booking can be created or existing bookings can be selected for editing and further processing. A number of filters in the header section of the screen allow the user to search for particular booking or types of bookings. The default criteria for this screen is to display all bookings.

[edit]

Action buttons

Field Description
Press 'Go' to execute your chosen filtering. If 'Go' is greyed out, then the current filtering has been executed.
Press 'New' to create a new booking. You will be taken to the 'Booking' tab and this screen will be in 'Edit' mode.
Print a CSV output of the filtered search results

Note: When printing the CSV output the column "FBT Value" uses the Contract's FBT value (fm_contracts->fbt_value) amount when an internal vehicle is allocated and the Variant's list price (qt_variant->list_price_gross) amount when an external vehicle is allocated and a Variant Code is selected on the Booking screen.
