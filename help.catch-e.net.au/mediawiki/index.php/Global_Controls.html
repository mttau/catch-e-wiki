Overview

Global controls are system-wide settings contained in the data table gb_controls. They contain controls such as tax rates, default amounts, standard text and system email addresses.

Each global control record is date driven, allowing controls to be set in the future (e.g. known tax change) and to keep a history of global controls. Specific controls are detailed below.

Module - each Global Control is classified within a "Module", which indicates the area it is used in.

Default Value - this column describes the default field parameters of each system on implementation or deployment, unless you have asked for something else.
These values are shown in serialised form - 1 Data Type (S=String): 2 Length of Field: 3 Value of Field.
Values that are very long and complex are not shown in this listing. Contact Catch-e support if you would like a listing of your organisations settings.
Visit www.functions-online.com for more information about serialised values.
