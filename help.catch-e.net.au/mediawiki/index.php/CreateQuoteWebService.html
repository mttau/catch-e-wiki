[edit]

Overview

This API allows you to create a new quote using a simple set of variables.
This performs the same function as navigating to Fleet / Administration / Quotes and creating a new quote.

Alternative API:
The createQuotes API is fully featured and all the available quote variables can be used.
This API is still being developed. A few features are not yet complete. Contact your Account Manager to discuss it's suitability for you.

Supplementary SWAGGER documentation is available here: api.catch-e.com/docs/#/Web%20Services/createQuoteWebService

[edit]

Enable

Go to Setup / Reference Data → gb_controls and find the "api_web_services_wrapper_login" control.
Populate this control with a 'web_services' user login and password.
This user will be used to run the API from within code.

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on WebServices.

[edit]

Authentication

Authenticate with the API before running this API.

[edit]

HTTP Method

Use the HTTP Method 'POST' for consuming this web service.

[edit]

URL Examples

https://api.test.catch-e.com/web-services/qt/salesforce/createQuote

Troubleshooting

The Financier used for this service will default from the Clients / Quote Defaults "Financier" record.

This Financier must have all of the following table records present (if not the Financier will be blank):

  • [fm_suppliers] - Populated when creating a new Suppliers record.
  • [fm_financiers] - Populated when creating a new Suppliers record and the Supplier Type is 'Financier'.
  • [fm_financier_rates] - Go to Reference Data screen to create/edit these records.
