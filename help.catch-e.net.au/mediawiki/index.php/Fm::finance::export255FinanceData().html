[edit]

Overview

This job type creates and puts an XML file as detailed in the 255 Finance page into the receivers nominated SFTP folder.

Event Code, Deals Partner Code and Destination are all mandatory fields.


[edit]

Job Steps

  1. File is written and stored in [gb_interface_exports] at 'pending' status.
  2. File is transferred via STFP to the nominated folder and [gb_interface_exports]status_flag is set to 'processsed'
  3. The [fm_contract_events] records used to identify contracts for this file have their [fm_contract_events]completed_flag set to 'yes'


[edit]

Parameters

Once the "Method" 'export255FinanceData' has been selected, the following input parameters will be displayed

Parameter Notes
Event Code Select the relevant event code from the available list. See 255 Finance for the full set of set-up steps for this process.
Deals Partner Code Enter the Deals Partner Code advised by 255 Finance.
Destination Drop down list which shows the stored active sftp connections added in gb_dmz_upload_file_destinations table.
Remote Prefix 255Finance will advise the text for this field. Format is likely to be MFT_{clientname}


[edit]

Destination

For this job to make a successful SFTP transfer, the SFTP access credentials need to be stored in the table gb_dmz_upload_file_destinations.

Field Datatype
dmz_upload_file_destination_id int(10) unsigned
name varchar(50) NOT NULL
type ENUM (sftp,s3) NOT NULL
configuration BLOB [encrypted]
status_flag enum('active','inactive')
