Overview

Opening the Booking Driver modal window enables you to create a driver record and add to their Contact details.
Once created, you can view and edit the driver's Contact and Events details.

You can access this window from the Boookings / Booking tab

  1. Click on the icon to create a driver from a booking in edit mode.
  2. Click on the icon to edit a driver from a booking in edit mode.
  3. Click on the icon to view a driver from a booking in view mode.


[edit]

Updating the same driver fields in the booking and driver records

There are some fields that can be stored in both the booking and the driver records that are the same: Salutation, Name, Mobile and Licence.

  1. If you navigate here and update the driver record, the fields that are the also stored in the booking will also update.
  2. If you update driver details directly in the booking, the same details in the driver record will not update.
  3. If you hyperlink to the Drivers / Contract tab and update the driver record, the driver-related details in the booking will not update.

Once you have a linked driver record, we recommend that you always navigate here to the Booking Driver modal window to make changes to the driver details so they are always updated for future bookings.

Action Buttons

Button Description
Create
Click to create the entered record. You will be returned to the Bookings / Booking tab.
If you choose Cancel, you are returned to the Bookings / Booking tab.
View
Click to edit the record. The modal window will be put into edit mode.
Click to close the modal window. You will be returned to the Bookings / Booking tab.
Edit
Click to save an edited record. You will be returned to the Bookings / Booking tab. This screen will be refreshed and the changes will be visible.
If you choose Cancel, you are returned to the Bookings / Booking tab and no changes are saved.
