Overview

Many fields in the system have a icon next to them when you have clicked on , or .
Click on this if you do not know the required code and want to search for a particular record to use.
The filter fields and the display columns in the lookup pop-up vary depending on the search that is required for the particular field.

Also visit the Lookup with Supplier Type page. This is a variation of the lookup that contains a separate filter for "Supplier Type".

[edit]

Filters

Filter Type Description
By: Drop List Select the field you want to search on. Each lookup will contains a different set of choices that are relevant to the lookup that is being done.
If there are no fields to search on other than the lookup field, this will be blank. E.g. Payments / Periodic "Reference".
Text: Text Enter the value you want to search on. This is not case-sensitive. E.g. entering 'a' will return all records that contain 'a' or 'A'
Close on select: Check Box Checked by default. When a record is selected, the field will populate with the chosen record and the lookup pop-up will close automatically.
If this is un-checked and a record is selected, the field will populate with the chosen record and the lookup pop-up will remain open.

This field is only present on some lookup pop-ups. If it is not shown, the lookup pop-up will close automatically when a record is chosen.

Contains text: Check Box Un-checked by default. Check this if the entered text should be searched for as a value within a string.
E.g. if un-checked, entering 'a' for "Supplier Code" will return 'A 1FIN' and no other records. If checked, any record that contains 'a' or 'A' will be returned.
Button Click to close the lookup pop-up.
Button Click to execute your chosen filtering.
