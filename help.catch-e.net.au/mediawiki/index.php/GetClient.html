[edit]

Overview

Enables you to get all fields for a given Client/s including associated Client Bespoke fields, Quote Defaults and Contract Defaults.

Supplementary SWAGGER documentation is available here: https://api.catch-e.com/docs/#/Clients

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on Clients:Get

This API is not configured for external use but can be made available on request.
Contact Catch-e Support to request this update.


[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

URL Examples

https://api.catch-e.com/fm/clients/{client_id}
