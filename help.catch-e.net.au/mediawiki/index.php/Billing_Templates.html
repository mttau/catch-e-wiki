Overview

A generic set of invoice templates exists in the system.
The headers and footers can be customised with your own logo, text and system placeholders.

Invoice Set-up

Each invoice template is stored in gb_templates which also contains emailing information.
The illustration below shows the links to email recipients and invoice placeholders.

[edit]

Emailing

The subject, text and sender details can be updated as standard. Visit the Email Setup page for standard configuration details.
Billings can be emailed Client/Contact "Contact 1" and cc'd to Client/Contact "CC". These are hard-coded and no entries are required in gb_template_recipients.
Billings can be emailed to other email addresses if they are set-up in gb_template_recipients.

Older clients may have custom templates and/or email rules. Check with your Account Manager if you're not sure.


[edit]

Invoice layout

The content of each header and footer are stored in the table gb_template_placeholders which is a linked to a [gb_templates]template_id.
The placeholder_name contains the name of the placeholder location and the placeholder_value contains the text or system placeholder that is applied.
System placeholders can also be used in conjunction with text.

[edit]

Channel Invoices

To enable Channel specific logos, header and footer text, a channel_id can be added to gb_templates and a unique key created combining the following fields (channel_id, name, sub_name).

gb_templates table with addition of channel_id


fm_channels table

Where an invoice for a channel is being set up and the channel_id in the gb_template record is omitted the logo, logo placement and footer will default to the main template data maintained in the table gb_template_placeholders.

The [gb_templates]sub_name field for a channel record should also be left blank.



A separate invoice template can be set up for each Channel. See Channel Billing section. To request a quote to set this up please contact your Catch-e Account Manager.

Useful Queries

# Logo
select ch.channel_code, t.name, tp.*
from gb_templates as t
inner join gb_template_placeholders as tp on tp.template_id = t.template_id 
left join fm_channels as ch on ch.channel_id = t.channel_id
where t.name in ('pdf_periodic_perlc','pdf_combined','pdf_recharges','pdf_recharges_fuel')
and placeholder_value LIKE '%logo%'
ORDER BY t.name, ch.channel_code, tp.placeholder_name
;

# Header Left
select ch.channel_code, t.name, tp.*
from gb_templates as t
inner join gb_template_placeholders as tp on tp.template_id = t.template_id and tp.placeholder_name IN ('header_1_2_text','header_1_3_text','header_1_4_text')
left join fm_channels as ch on ch.channel_id = t.channel_id
where t.name in ('pdf_periodic_perlc','pdf_combined','pdf_recharges','pdf_recharges_fuel')
ORDER BY t.name, ch.channel_code, tp.placeholder_name
;
# Header Middle
select ch.channel_code, t.name, tp.*
from gb_templates as t
inner join gb_template_placeholders as tp on tp.template_id = t.template_id and tp.placeholder_name IN ('header_5_3_text')
left join fm_channels as ch on ch.channel_id = t.channel_id
where t.name in ('pdf_periodic_perlc','pdf_combined','pdf_recharges','pdf_recharges_fuel')
ORDER BY t.name, ch.channel_code, tp.placeholder_name
;
# Header Headings
select ch.channel_code, t.name, tp.*
from gb_templates as t
inner join gb_template_placeholders as tp on tp.template_id = t.template_id 
	and tp.placeholder_name IN ('header_7_1_text','header_7_2_text','header_7_4_text','header_7_6_text','header_7_7_text','header_7_8_text','header_7_9_text','header_8_1_text','header_8_2_text','header_8_8_text','header_8_9_text')
left join fm_channels as ch on ch.channel_id = t.channel_id
where t.name in ('pdf_periodic_perlc','pdf_combined','pdf_recharges','pdf_recharges_fuel')
ORDER BY t.name, ch.channel_code, tp.placeholder_name
;
# Body
select ch.channel_code, t.name, tp.*
from gb_templates as t
inner join gb_template_placeholders as tp on tp.template_id = t.template_id 
	and tp.placeholder_name IN ('body_1_2_text')
left join fm_channels as ch on ch.channel_id = t.channel_id
where t.name in ('pdf_periodic_perlc','pdf_combined','pdf_recharges','pdf_recharges_fuel')
ORDER BY t.name, ch.channel_code, tp.placeholder_name
;
# Footer Left
select ch.channel_code, t.name, tp.*
from gb_templates as t
inner join gb_template_placeholders as tp on tp.template_id = t.template_id and tp.placeholder_name IN ('footer_2_1_text','footer_3_1_text')
left join fm_channels as ch on ch.channel_id = t.channel_id
where t.name in ('pdf_periodic_perlc','pdf_combined','pdf_recharges','pdf_recharges_fuel')
ORDER BY t.name, ch.channel_code, tp.placeholder_name
;
# Footer Middle
select ch.channel_code, t.name, tp.*
from gb_templates as t
inner join gb_template_placeholders as tp on tp.template_id = t.template_id and tp.placeholder_name IN ('footer_1_2_text','footer_2_2_text','footer_3_text','footer_3_2_text','footer_4_text')
left join fm_channels as ch on ch.channel_id = t.channel_id
where t.name in ('pdf_periodic_perlc','pdf_combined','pdf_recharges','pdf_recharges_fuel')
ORDER BY t.name, ch.channel_code, tp.placeholder_name
;
