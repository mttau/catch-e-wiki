Main Page → Owl → Pay Run Processing → Skip Process

Qualifiers

  1. Payroll Summary Report has run
  2. The client has advised which Employees and Items should not be paid in the next payroll
  3. The required scheduler jobs for this process have been created and are active


Skip payroll deductions

Skipping a payroll deduction means you are shifting the deduction to the following pay run.
If the client advises that some deductions need to be skipped, you can do this manually or by automation.
You can only do an automated skip on a contract if:

  1. The Contract is linked to an Item record
  2. Contracts / Lease - has two or more pending periods
  3. Contracts / Lease - the next pending period does not already hold an adjustment record

Automated skip - Not currently functional

  1. Assemble a list of the contracts that need to be skipped
  2. Save them in a spreadsheet with the column heading 'contract_id' in *.csv format
  3. Navigate to Setup / Import Table Data
  4. Module - select '_sch|Scheduler'
  5. Table - select '_sch_payroll_deductions_skip_import'
  6. Import Table Data

Manual skip

  1. Note the dates of the Employee's next two pay runs
  2. Note the contracts that need to be skipped
  3. Navigate to the Contracts / Lease tab
  4. Contract ID - enter the contract that needs to be skipped
  5. Period - select the period that you are going to skip
    1. Click on
    2. Update the amounts to be 0
    3. Click on
  6. Period - if the skipped deduction is to be deducted later, select that period
    1. Click on
    2. Enter the amounts to match the skipped deduction
    3. Click on
  7. Repeat for all of the contracts that need to be skipped for this Employee

Update the event

  1. When the automated or manual skips have been done, edit the client event and update Action to 'Re-issue'
  2. The scheduler job will run and
    1. The existing email event is updated to set Completed as checked
    2. A new email event is created with an Action of 'Pending', Completed is un-checked

Client confirms the payroll deductions

  1. Navigate to the Clients / Events tab
  2. Select and edit the pending event, update Action from 'Pending' to 'Authorised'

Next Steps

You can now Bill the Pay Run
