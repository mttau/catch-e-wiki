Overview

The Dealer Quote Request is generated from the Quotes / Vehicle tab.
PDF, Office (xlsx) outputs are available.
Print or email to the "Preferred Dealer" recorded on the quote.

Most of the form fields are self-explanatory and often have the same names as field name.
To get the full list of available placeholders, an 'admin' role user can select and print the 'Field List' csv from the output list box.
Not all placeholders are listed below as there are 490 placeholders available.<vr> Below are details of some key Dealer Quote Request placeholders that require some calculating or the description does not match the screen name.

[edit]

Process pages

To update your Dealer Quote Request

  1. Go to Setup / Import or Export Files and Export the current form
  2. Each form has a different set of placeholders, so print the field list to see what's available for this one.
  3. Use these details to create the layout of the document you want, save this file
  4. Go to Setup / Import or Export Files and upload your new form.

Below are some of the placeholders you can use, to see all of the placeholders you can use on this form, print the field list.

Email Set-up

The email is stored in [gb_templates] where name = 'pdf_dealer_quote_request'. The email may be hard-coded for some older clients.
The Quotes / Contract "Assigned" user can be added as a CC or BCC recipient in gb_template_recipients.
The sender is usually the user that launches the email. If their email address is malformed, the sender details will switch over to use the stored [gb_controls]mail_sender_address email instead.
Visit Email Setup for general information on configuring email.
