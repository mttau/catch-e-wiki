Delete a Disbursement BPAY file

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Disbursements → Delete a Disbursement BPAY file

Qualifiers

  1. You need to Unwind a pay run because the BPAY file total is incorrect
  2. You have only recently created the file
  3. No transactions that belong to a future pay run have been posted (I.e. un-wanted transactions will not be selected for disbursement)


Select client

  1. Navigate to Accounts / Export / BPAY → Disbursements
  2. BPAY ID - enter or lookup the required record
  3. Confirm this is the correct record to delete
  4. Status - check the status is 'Pending', if it is not:
    2. Status - update to 'Pending'
  6. You will be warned of this change
    1. Click OK
