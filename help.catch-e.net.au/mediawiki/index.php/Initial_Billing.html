Overview

The main features of initial rentals are:

  1. Delivery Fee, Fuel on Delivery and Establishment Fees appear on the Lease Screen in periods marked as 'Initial Fee' (i.e. period 1).
  2. The amounts for these fields default from the Client/Defaults Screen but can be over-ridden on the Lease Screen.
  3. The Fees allowed per contract are controlled by the Contract Type/Initial Fees Screen (Novated contract types have been setup to have Establishment Fee instead of Delivery Fee).
  4. Initial billing in Catch-e will create entries to be imported into MYOB for both the Sales Ledger and General Ledger. These should be imported at least daily and checked for errors just as with the Purchase Ledger import.



There is a select box on the bottom control line next to the select box for out mode. This select box will let you choose the template type. The default is 'Plain' for plain paper to give the existing format (including logos and full details), and 'Letterhead' to use a new format created for use on Letterhead paper.

The field 'Cost Centre Fuel' under is applies to Initial Billing where applicable. The Initial Rental invoice has appended the text ' (Cost Centre: xxx)' after 'Fuel On Delivery' when the field 'Cost Centre Fuel' is non-blank (xxx represents Cost Centre Fuel).

Delayed Initial Billing:

Note that the system expects initial billing to be dated the same as the contract start date. In the case of arrears or delayed billing, this can simply be over-ridden by changing the date on the Fleet/Billing/Initial screen to the desired date for the invoice. The initial invoice and associated accounting entries will then be dated using the desired date. Regular periodic invoices can be controlled using the start periodic field on the billing screen. Thus a delayed contract may have 1 initial billing, with periodic billing commencing in month three.

Client Recharges may not appear on the Recharge Billing screens, depending on which Billing Type has been set in the Client's defaults.
