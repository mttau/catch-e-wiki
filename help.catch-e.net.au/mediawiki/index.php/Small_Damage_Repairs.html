Overview

Small Damage Repairs (SDR) is a quote profile. If configured and selected, the 'best fit' premium will default into a quote automatically when the profile is checked.
If the Quotes / Contract "Term (Months)" is changed, the profile will also update to the 'best fit' premium.
If the quote's "Type Code" (Contract Type) is changed, and the new "Type Code" (Contract Type) does not have an SDR class accrual, Small Damage Repairs will be un-checked and the values removed from the quote.
The value of the Small Damage Repair "Premium" field will update the Contracts / Finance "Small Damage Repairs Collected" field when a quote is accepted or settled.

Best fit: Premium records are stored in the in [qt_small_damage_repairs] table with months as one of the variables. This profile is designed to work with records stored in annual increments. E.g. the months are stored as 12,24,36,48,60.
If the quote "Term" matches the months of one of the stored record (12,24,36,48,60), that premium record will be selected.
If the quote "Term" falls between the months of the stored premiums, the premium record with a higher months value will be used. E.g., if the term is '25', the premium record where months = '36' will be selected.

The Small Damage Repairs profile can be added to or excluded from a quote when using the createQuoteWebService API.

Set Up

Enable the Small Damage Repair profile as follows.

[edit]
Global Controls
Module Name Description Value
qtsmall_damage_repairs_type Check and update the value of this control to either create a budget for the Small Damage Repairs premium,
or to add it to the quote's brokerage as a financed amount. 		budgeted or financed


[edit]
Small Damage Repair Premiums

Get an agreed table of premiums from your provider.
Premiums should be in yearly increments. E.g., 12, 24, 36, 48, 60.
Upload these into the [qt_small_damage_repairs] table. Contact Catch-e Support for assistance if you are setting up this profile for the first time.

[edit]
Posting Classes

For the Set Up data to appear in the quoting process, 'Small Damage Repair' must be created as a Posting Class. Go to menu Fleet / Controls / Accounts / Posting Classes / Details Tab.

To create a new posting class,

1. Select 'New'
2. Enter class code 'SDR' - this must be 'SDR' to ensure that there is consistency through out quotes and reporting.
3. Enter Name 'Small Damage Repair'*
4. Check ON the box next to 'Event View Flag?' if you wish to create contract events
5. Select Quote Profile Table 'qt_quote_small_damage_repairs' from the list to enable the Profile to be created when quoting.

If a posting class already exists, make sure that the Quote Profile Table is set to 'qt_quote_small_damage_repairs'.

If you want to create an event for Small Damage Repairs, but not have it link up to Quotes, simply leave the posting class screen reference 'Quote Profile Table' blank.

  • You can nominate another name if you prefer. Refer to Posting Classes for more information and what impact this has on screen field names.
[edit]
Contract Types

Now that the product has been set up, it needs to be included into the Contract Type you will offer it under. Go to menu Fleet / Controls / Accounts / Contract Types.

1. Enter or search for Contract Type Code
2. Go to Accruals Tab
3. Select Edit
4. From the drop list select 'SDR'
5. Enter required set up details in the right of screen and save
  • Any new quotes will have the product available. Existing quotes, will need to be re-calculated to introduce the new posting class eg: re-setting the contract term will have this affect.
[edit]

Rename

  1. Quotes / Lease - rename the budget and profile name of "Small Damage Repair"
    1. Navigate to Accounts / Posting Classes
    2. Class Code - enter or lookup 'SDR'
    3. Click on Edit
    4. Name - enter the name you want
  2. Contracts / Finance "Small Damage Repairs Collected" - can be renamed.
[edit]

Restrict

Contracts / Finance "Small Damage Repairs Collected" - can be role restricted to be 'read only' if required.
