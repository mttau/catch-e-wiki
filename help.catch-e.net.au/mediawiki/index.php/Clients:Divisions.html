Overview

Record your client's divisions here.

Quotes / Contract "Division" - you can add the required division to quotes if you know it.
A Contracts / Events 'Division Change' contract event will be created when a quote is accepted. The "Date" field of this first record is blank.
Add or change a contract's division by creating a Division Change contract event. Populate the "Date" field with the date of the change.

Only active divisions are available for selection in quotes and contract events.

Divisions can be used as a filter in the Contracts / Search screen.
Divisions can be used as a filter in selected Reporting / Standard Reports.

Clients / Divisions is similar to Clients / Cost Centres but the two functions operate independently.

[edit]

External Users

Division is one of the Restrict Range parameters that can be used to filter which contracts and drivers external users can see when they login to Catch-e.
If configured, the Clients / Division tab can be made available to the following external user roles: fleet_manager, fleet_manager_ro, group_manager and package_employer. Contact your Account Manager if you want to arrange this.
