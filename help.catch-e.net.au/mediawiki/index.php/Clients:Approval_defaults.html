Authorised Suppliers

Enter records here if you want to allow only authorised Suppliers to provide services for the client. The restriction is at Posting Class level. You can allow more than one supplier to provide services and you can nominate more than one Posting Class.
E.g. Your client has an agreement with a tyre business to source all tyres from their outlets. Record the preferred supplier and the Posting Class 'T'. When an Approval is raised, the user will be warned if a tyre entry is selected for another supplier on a contract linked to this client.

The user is not prevented from raising an approval, but will be warned if the Supplier/Posting Class combination is not authorised. If configured, the user can be alerted to send an email to the client when an unauthorised supplier has been used on an approval.

Authorised Suppliers Example

In this case Supplier Code 'SMASH' is permitted to fit Posting Classes 'Accident' & 'Glass' but not 'Tyres'. They are also permitted to fit any other Posting Class not listed in the Class column (eg. 'Body & Trim').
'Tyres' can be fitted only by Supplier Codes TSUP1 & TSUP2.
'Glass' can be fitted only by Supplier Codes WSRU & WS4U.
