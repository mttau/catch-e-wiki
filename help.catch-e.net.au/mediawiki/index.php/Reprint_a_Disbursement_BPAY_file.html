Reprint a Disbursement BPAY file

From Help wiki

Jump to: navigation, search


Main Page → Workflow → Salary Packaging → Disbursements → Reprint a Disbursement BPAY file

Qualifiers

  1. BPAY Batches are being generated in your system


Locate the BPAY Batch you want to reprint

  1. Navigate to Accounts / Export / BPAY → Disbursements (BPAY)
  2. BPAY ID - use the lookup to find the batch you want
  3. Output - select 'Transactions'
  4. Click on
