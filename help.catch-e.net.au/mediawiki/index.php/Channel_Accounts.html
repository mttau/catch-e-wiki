Overview

Access Type: Internal
Restriction: The Channel Accounts role can....view quotes, drivers, events, and clients that are linked to clients that belong to the nominated channel.
Set Up: The Restrict Key uses the Channel to restrict records.
The Restrict Range feature is not available for this role.
Visit the Channel Setup page for complete details about how to set-up and use the Channel feature.

Use the Channel Accounts role if you are using a channel to manage your Salary Packaging.
