[edit]

Overview

A Sub Status can be used to extend information about the "Quote Status".
It can be a sub-set of a "Quote Status" field, or can be selected any time, depending on your configuration.



[edit]

Set-up Notes

The list box is stored in the table qt_quote_sub_statuses.

Populate [qt_quote_sub_status]parent_status_description with the value stored in [qt_quote_statuses]description to link this sub status to display only for the nominated "Quote Status".
If you leave [qt_quote_sub_status]parent_status_description blank, the record will only be selectable if the quote's "Quote Status" is not linked to any other records.

Change [qt_quote_sub_status]status_flag to 'inactive' if you want to halt use of the records. The record will no longer be available in the "Sub Status" list box.
Quotes that exist where the sub status has been changed to 'inactive' will not be altered unless edited. If edited, the sub status record will be removed and cannot be re-selected.
