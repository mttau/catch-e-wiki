Create default quote fees for a Financier by Contract Type

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Fleet Management → Set Up → Create default quote fees for a Financier by Contract Type

Qualifiers

  1. The financier's Suppliers / Financier Finance Method field must be set
  2. The financier should have interest rates stored in the Setup / Reference Data 'fm_financier_rates' table.

Process

  1. Navigate to Setup / Reference Data
  2. Table - select 'qt_financier_posting_map_defaults'
  3. Click on
    1. Financier - enter the Financier's supplier code
    2. Contract Type - enter the Contract Type that the fees are for
    3. Term - enter a term that the fees are for, or if the fees are for any term, leave this as '0'.
    4. Advance / Arrears - pick one if you want to apply different fees for each situation, otherwise leave this field blank
    5. Finance Budget Term Deferred - pick one if you want to apply different fees for each situation, otherwise leave this field blank
    6. Est / Admin Fee - enter the fee, or leave blank
    7. Originator Fee - enter the fee, or leave blank
    8. REVS / PPSR - enter the Fee, or leave blank
    9. Mortgage Duty - enter the Duty, or leave blank
    10. Fee - enter the Fee, or leave blank
      1. Record the GST Exclusive amount
      2. A fee may be applicable to your Angle Auto Finance applications.
    11. Novated Commission % - enter the %, or leave blank
    12. Novated Commission Max - enter the Max, or leave blank
    13. Effective - leave blank
  4. Click on
