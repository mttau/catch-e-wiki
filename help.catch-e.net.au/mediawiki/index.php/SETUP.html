Setup is a module designed to allow users to add and edit reference tables and data, or administer custom business templates.

Import Table DataUser can import the selected file using this screen , import includes overwriting the record, updating and creating record, also user an import a partial upload as well
Export Table DataUser can export the data from the selected tables.
