Filters and Buttons

Filter Type Description [table]field
Supplier Code Text Enter the supplier code you want to select.

Use the lookup to search for a supplier by "Supplier Code", "Name", "ABN" or "Type".
If the Suppliers / Detail "ABN" is shown with an alias, then the alias is also shown here as one of the lookup selections.

[fm_suppliers]supplier_code
View
Button Click to create a new record. Suppliers / Account "Status" defaults to 'Active'.

Field Descriptions

Field Type Description [table]field
Column One
Supplier ID Number Unique id used to identify the supplier allocated by the system. [fm_suppliers]supplier_id
Supplier CodeText (10)Unique code used to identify the supplier. User determined and can be edited after creation as a Supplier id is stored within the database. Must not be blank or constructed entirely from spaces.[fm_suppliers]supplier_code
NameTextSupplier name[fm_suppliers]name
Address 1TextAddress 1[fm_suppliers]address1
TextAddress 2[fm_suppliers]address2
AreaText (50)Enter the suppliers physical address Suburb or Town. [fm_suppliers]area
List BoxList of valid states[fm_suppliers]state
Num (4)Postcode[fm_suppliers]postcode
Country Drop List Select your suppliers physical address country from the list. This will select your country by default.
e.g. if your [gb_controls]country_code is 'AUS', Australia will be used to populate this field. 		[fm_suppliers]country_id
ABN Text Enter the Supplier's ABN or NZBN here.
This field can be renamed.		[fm_suppliers]abn
ContactTextContact person's name[fm_suppliers]contact
MobileTextSupplier mobile number[fm_suppliers]mobile
PhoneTextSupplier phone number[fm_suppliers]phone
FaxTextSupplier facsimile number
EmailText Enter the supplier's email address here.
Multiple email addresses can be entered by adding a semicolon (;) to separate each email address. Leave no spaces in-between.
The email addresses are validated. Go to the Email Address Validations page for more details on how this is checked.
If the supplier is a Financier or Insurer that is being paid using the "Payments" module, their email address will be needed to email their Remittance Advice. See Payments Setup and Fleet / Payments / Periodic for more details.		[fm_suppliers]email
Web SiteTextSupplier web site address. Clickable hyperlink
External CodeText (18)Holds a "Salesforce ID" if you are interfacing with salesforce CRM software. May be used as a reference to other systems, e.g. 1link.
Column Two
Channel CodeTextEnter a channel code the or use the lookup to find it. If you are a Channel Accounts role, this field will be read only and your channel code will populate in automatically.[fm_suppliers]channel_id
Supplier GroupText (10)Enter or search for the Group Code that this supplier belongs to, if any. You need to set-up the Supplier Group record first in the Supplier Groups → Contact screen. Once selected, the Group Code and Group Supplier Name will be displayed.
Notes:TextFree text field to record notes about suppliers
