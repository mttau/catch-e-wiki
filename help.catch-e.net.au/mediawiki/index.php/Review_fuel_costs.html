Review fuel costs

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Fleet Management → Set Up → Review fuel costs

Qualifiers

  1. Your variants are set-up with a linked fuel type
  2. You want to review the settings and costs used in the Quotes / Lease - Fuel (F) profile to calculate the Fuel budget.


Check the control settings

  1. Go to Setup / Reference Data
  2. Table - select 'gb_controls'
  3. Name - enter 'fuel_usage' - check the required default is stored. set as 'combined', 'metro' or 'country'.
    1. This is the default entry for the Quote / Lease Fuel Profile "Fuel Usage" field.
    2. This field is used in the vehicle's fuel consumption calculation and thus, the fuel budget.
  4. Name - enter 'fuel_cost_adjustment'
    1. check and update each state amount
      1. if an amount is entered here for a state, the Quote / Lease Fuel Profile "Cost per Litre" amount will be scaled up or down by the amount of this entry
      2. Quote / Contract "Delivery State" is the state that is used to apply this scaling.

Update fuel costs

Export data from a table
  1. Go to Setup / Export Table Data
  2. Module - select 'qt|Quotes'
  3. Table - select 'qt_fuel_types'
  4. Click on
  5. Save your file


Review and update costs
  1. Save a copy of the file, this will be your archive file
  2. Save a new copy of the file, this will be your upload file
  3. You can create new records with a new effective_date each time you update the costs, but we recommend maintaining just one set of fuel records so that fuel costs are always updated to use the current costs if you edit a quote's fuel profile.
  4. If you only have one one set of fuel records (I.e. [qt_fuel_types]effective_date is 0000-00-00), (I.e. only one record set exists)
    1. Update the [qt_fuel_types]fuel_cpl field as required for each record
  5. If you have more than one set of fuel records and want to overwrite them, then
    1. Find the record set with the most recent effective_date
    2. Remove all the records that are earlier than this from the upload file (I.e. only one set remain)
    3. Update the [qt_fuel_types]fuel_cpl field as required for each record
  6. If you want to create new records for each cost change, then
    1. Locate and make a copy of the last set of tyre records in use
    2. Remove all of the pre-existing records from the spreadsheet (I.e. only one set remain)
    3. Set the[qt_fuel_types]fuel_type_id to be blank
    4. Update the [qt_fuel_types]effective_date field to show the date that the costs will be used from
    5. Update the [qt_fuel_types]fuel_cpl field as required for each record
  7. Save your new upload file


Import data to a table
  1. We recommend testing this update in your support environment first
  2. Go to Setup / Import Table Data
  3. Module - select 'qt|Quotes'
  4. Table - select 'qt_fuel_types'
  5. Drop or browse to import the data file
  6. Click on Upload
  7. Click on
  8. Navigate to the system and check the data you have imported.
  9. Test the changes by creating and copying quotes to see the updated costs applied
