Overview

This method is used to reconcile event subscriptions.

Supplementary SWAGGER documentation is available here: https://api.test.catch-e.com/docs/?/Events/reconcileEventSubscriptions.

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on Events:ReconcileSubscriptions

This API is not configured for external use but can be made available on request.
Contact Catch-e Support to request this update.


[edit]

HTTP Method

Use the HTTP Method 'POST' for consuming this web service.

Response Details

Validation Messages Comments

Subscriptions reconciled.

  2. The token_timeout of the current session has passed. You need to authenticate again.

{
    "type": "http://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html",

    "detail": "Forbidden"

  2. Navigate to the Roles / APIs tab to make sure the permission you need to run this API is checked.
