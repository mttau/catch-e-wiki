[edit]

Overview

A Loan Termination Insurance (LTI) premium can be selected by choosing a "Plan" and then a "Top Up Benefit" in the Quotes / Lease tab in the Profiles area.
If the quote's "Type Code" is changed, and the new "Type Code" does not have an LTI class accrual, Loan Termination Insurance will be un-checked and the values removed from the quote.

[edit]
Plan

You can store and offer one or more LTI product, but if only one "Plan" is configured, it will be selected and displayed by default.

[edit]
Top Up Benefit

Select a "Top Up Benefit" from the list and the "Premium" amount will populate. This amount can be edited by the user and is usually displayed as GST inclusive.
The "Top Up Benefit" list can be filtered further by using upper and lower limits to display eligible choices for the Quotes / Finance "Vehicle On Road" amount.
If there is only one benefit record that meets the selection requirements of the quote, it will be selected and displayed by default.

[edit]
Financed

LTI is shown in the Quotes / Lease tab as a Profile for selection. It is also shown in the Budget section with a zero value.
It is a financed product and is also displayed in the Quotes / Finance tab as "Loan Termination Insurance" in the Brokerage section and forms part of the "Total Brokerage" amount.
This amount is included in the Insurance parameter of Maclease and Sovereign finance applications made via Catch-e.

[edit]
Timing

Once the LTI profile is created, any new quotes will have the profile available on creation. To add the LTI profile to an existing quote, just edit the quote and select the profile.

Plans and benefits are recorded with a status of 'active' or 'inactive'. Only 'active' records are visible and selectable.
Once a plan or benefit is made 'inactive', it is not available on a quote and an edited quote cannot be saved with an 'inactive' record. If all plans are inactive, all benefits should also be made inactive.

There is no reference to the "Effective Date" of the quote.

[edit]

Renaming

The Quotes / Finance "Loan Termination Insurance" field name can be re-labelled.
The name of the field shown in the Quotes / Lease tab is the one stored in the Posting Classes record. Change the name of the posting class to update this.

[edit]

Read Only

The "Premium" field can be made read only by role using Role Restrictions.

[edit]

Set Up steps

1. Populate qt_lti_plans and qt_lti_top_up_benefits.
2. Create a Posting Class called LTI - Loan Termination Insurance. The field "Quote Profile Table" should be set to to 'qt_quote_lti'.
3. Add the Posting Class to the Contract Types / Accruals tab of each Contract Type where LTI will be offered.
4. Update and test your quote documents to show this insurance.
5. Update and test internal forms and supporting reports to show this insurance.

Contact your Account Manager for assistance to set-up this feature.

Placeholders

Placeholders have been added to the QT getQuote() web service and Settlement template to support this profile.
Placeholders for both are mapped as follows:

PlaceholderSource TableSource FieldDescription
{lti_gross}qt_quote_ltiretail_price_gross Retail price incl GST of LTI product
{lti_cost}qt_quote_lticost_price_net Cost price of LTI product
{lti_margin}qt_quote_ltimargin_net Margin
{lti_net}qt_quote_ltiretail_price_net Retail price excl GST of LTI product
{lti_gst}qt_quote_ltiretail_price_gst GST amount of LTI product
