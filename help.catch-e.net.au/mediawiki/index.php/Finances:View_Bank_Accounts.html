Field Descriptions

Field Type Description [table]field
View Bank Accounts
#Check BoxSelection checkbox to enable you to edit or delete the bank account.
Account Name Text “Account Name” can be edited. [fm_driver_bank_accounts]account_name
BSB Number Can only be edited when a new bank account is being created, otherwise this is a read only field. [fm_driver_bank_accounts]bsb
Account Number Number Can only be edited when a new bank account is being created, otherwise this is a read only field. [fm_driver_bank_accounts]account_number.
Claim Check Box This is checked by default.
Checked accounts can be selected in the Contracts / Claim tab when creating reimbursement claims.
This check box cannot be edited if the selected record is the default bank account. 		[fm_driver_bank_accounts]claims_access_flag
Default Check Box The first bank account record that is created has the default bank account field checked automatically. When new vehicle claims are created in Contracts / Claim tab, or in the Driver Portal, the default bank account record is selected.

Choose a new default record by checking this field on your preferred record. The existing record will be updated on save, only one record can be the default. If you edit a record and un-check the “Default” check box, there will be no default bank account until another record is made so.

If a Driver enters a new bank account number in the Driver Portal when making a claim, that bank account will become the new 'default bank account. The existing default bank account will be updated to un-check it's "Default" check box. Remember - there can be only one.

[fm_driver_bank_accounts]claims_default_flag

Action Buttons

Button Description
View
Select a record to enable this button. Click to apply any updates made to existing bank account records.
Select a record to edit to enable this button. Then click to edit a record.
“Account Name”, "Claim" and "Default" fields can be edited.
You cannot change the “BSB” or “Account Number” of any existing record. If you need to use a different “BSB” or “Account Number”, create a new record for this and select it.

If you made a mistake when entering a record and the “BSB” or “Account Number” are incorrect, delete the record.

Select a record to edit to enable this button. Then click to edit a record. If the record selected also has the "Default Flag" checked ON, this button remains disabled.
If you click on Delete, the selected record is updated to have a status of ‘inactive’. Inactive accounts are not visible or available for selection anymore.

If claims have been created previously using a bank account record that is now ‘inactive’, it remains the stored record on the claim. If you are deleting a bank account record, check that there are no pending or unpaid claims outstanding that use the target record.
