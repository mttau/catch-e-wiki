Update a pay run to remove transactions

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Pay Runs → Update a pay run to remove transactions

Qualifiers

  1. The client has requested some transactions be removed from a pay run


Process

Remove transactions
  1. Navigate to Packaging / Pay Runs
    1. Client Code - enter or lookup your client's code
    2. Pay Cycle - select the pay cycle you need (it will default in if there is only one).
    3. Status - select 'Posted'
    4. Period Date - enter or pick the date of the pay run to update (this uses the date from each item's Contracts / Lease "Period" date). Tab out of this field.
    6. If required, add further filters for the "Employee ID" or "Contract ID" to narrow down your search
    7. Contract ID - click on the hyperlink
  2. You are now in the Contracts / Lease tab
    1. Period - select the period that has been posted in the pay run
    3. Alert: This action will modify or delete invoice no xxxx, are you sure? Image:OKKbutton.gif
    4. Click on the back button to navigate back to the Packaging / Pay Runs screen
  3. Repeat the steps above for any other transactions that need to be removed from the pay run


Re-print Pay Run reports
  1. When the Pay Run updates are complete, re-print the Pay Run reports and send them to your client
  2. Status - select 'Posted'
  3. Output File- select the report you want to send to your client


Next Steps

You can now Receipt the pay run when the Payroll Funds have been received.
