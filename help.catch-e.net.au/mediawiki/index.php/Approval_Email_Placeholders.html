Overview

The markers ## or {} can be used to make placeholders. E.g. #contract_id# or {contract_id} are both fine.

[edit]

The placeholders listed below are available for use in Contract Event emails when [gb_templates]name = 'pdf_email_review'

This will print for the first approval row of each job created in the approval.
You need to use the HTML pre Tag to invoke a carriage return for each job description.
Font instructions are ignored when "pre" is used, so you also need to insert a HTML font face Tag to re-define the font you are using.

[fm_maintenance]description (first line of each job#)
approval_gross Approvals/Approval "Total"[fm_maintenance]total_cost
franchise_notes Approvals/Approval "Franchise Notes"

You need to use the HTML pre Tag to invoke carriage returns, so that the text displays as it does on the Approval screen.
Font instructions are ignored when "pre" is used, so you also need to insert a HTML font face Tag to re-define the font you are using.

[fm_approvals]franchise_notes
