Overview

Download this request form Quote Event Email Request and send it to Catch-e Support if you want to set-up a Quote Event Email.

[edit]

Placeholders

The placeholders listed below are available for use in Quote Event Emails when [gb_templates]name = 'pdf_quote_event_email_message'.

The markers ## or {} can be used to make placeholders. E.g. #quote_id# or {quote_id} are both fine.
