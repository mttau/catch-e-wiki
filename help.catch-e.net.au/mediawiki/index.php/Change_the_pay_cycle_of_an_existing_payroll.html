Change the pay cycle of an existing payroll

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Employers → Change the pay cycle of an existing payroll

Qualifiers

  1. An employer is changing the pay cycle for an existing Payroll. E.g. a monthly payroll will now be done fortnightly


Process

Work out the calculations
  1. Each item is build using calculations and records based on a particular pay cycle.
  2. If this is being changed, the existing items will need to be closed down and new items created that use the timing and amounts of the new pay cycle
  3. Contact Catch-e to get a report of the existing items that includes the remaining cycles and amounts they have
  4. Use this report to build a report to show what cycles and amounts the new items should have
Close out the existing items
  1. After the last pay run has been completed for the payroll
  2. Suspend each existing item
Change the Employee's Pay Cycles
  1. Update each employee to have the new pay cycle stored in their record (you may be able to do a bulk upload to achieve this)
Create new items
  1. Using the calculation report, create a new item for the remaining cycles and amounts required for each employee.
  2. If any of the items are for Novated Leases, take extra steps to update the vehicle details they require.
  3. Contact Catch-e to run a revised report for you to confirm the changes are all correct.
