Internal Users

Users can click on the icon and this wiki system will appear in a pop-up screen. The pages are context-sensitive, which means that every system page has a corresponding page in this Help wiki.

There are many other pages that are shown as hyperlinks in the help text to provide you with more information.


[edit]

External Users

External users such as Drivers and Fleet Managers also have access to the icon. This will launch a pop-up screen that contains either a PDF or HTML content.
The information is not context-sensitive. It shows a Contents listing and then provides details relevant to external users.
The default help is an HTML pop-up based on Catch-e's generic features.

The UserGuide.pdf is used in the external help wiki. You can create or edit your guide and upload it in the Setup / Import/Export Files screen. Here is a sample to start with.
