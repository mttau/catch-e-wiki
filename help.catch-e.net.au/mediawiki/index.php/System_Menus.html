Overview

Most roles have a unique set of menus they can access. The menu items each role can access is controlled by their entry ib gb_role_restrictions using type = page_map & restriction_type = bit_array.

Contact your Account Manager if you want changes to the menus available for a particular role. Please do not attempt changes to this area without assistance from your Account Manager.

Catch-e Staff - refer to Internal wiki page 'Top Menu Configuration'.


[edit]

Excluded Roles

The roles listed below are not updated using this feature. They either do not have access to the system or have a single programmed function outside of these menus. Adding a gb_role_restrictions for these roles will not affect their function.

  • Franchise
  • Password Reset
  • FBT Declaration
  • Web Services
