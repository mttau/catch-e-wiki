Main Page → Workflow → Accounting → Create a new receipts bank account

Qualifiers

  1. You are creating a new bank account to receipt funds into


Process

  1. Go to General Ledger / GL Codes
  2. Click on
  3. GL code - enter your GL Code
  4. Description - enter the bank account name
  5. Account Type - select 'Bank' from the list
  6. Click on
