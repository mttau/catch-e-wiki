Overview

Supplier Events are intended to record transactions relevant at the supplier level.
Events can be viewed across all Suppliers at Events / Suppliers.

Event screens exist for Clients, Quotes, Contracts, Drivers, Employees and Suppliers.

[edit]

Workflow

New events are started by clicking on the New button. Click on Create to save the event.
Use the # check box to select an event to Edit or Delete.
Use the # check box to select an event to Attach a document to. Attached documents can be downloaded for viewing by clicking on the paper clip icon .
Every event can be managed by flagging the required action, setting a due date to be completed by and also giving you the flexibility to mark the event as completed or incomplete.
Incomplete tasks are summarised in the KPI Home screen.
