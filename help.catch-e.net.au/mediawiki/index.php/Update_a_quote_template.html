Main Page → Workflow → System Administration → Update a quote template

Qualifiers

  1. You have an in-house or third-party web designer managing changes to your quote HTML files


Get the latest zip from Live
  1. Go to Setup / Import or Export Files
  2. Select the Export radio button
  3. File Type - select 'Quote Templates'
  4. File Name - select the required quote zip file
  5. Click on Export
  6. Save the zip file down
  7. Keep an archive copy of each zip file you have used.
  8. Older versions of the zip file are not kept in the system.
Update your quote zip
  1. Prepare a new zip file for your quote template
  2. Visit the Quote Template page for more details about how the zip must be named and assembled
  3. Upload your new zip file to Staging to UAT the changes before you commit it to Live.
Import to Staging for UAT, then update Live
  1. Go to Setup / Import or Export Files
  2. Select the Import radio button
  3. File Type - select 'Quote Templates'
  4. File Name - select the required quote *.zip file
  5. Drop or browse to attach the zip file saved from Staging
  6. Click on Import
