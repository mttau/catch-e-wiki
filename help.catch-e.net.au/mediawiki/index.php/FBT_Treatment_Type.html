Overview

The "FBT Treatment Type" setting determines if a quote is exempt for FBT Payable or not.

[edit]

ATO Rules




[edit]

FBT Treatment Type field logic

The "FBT Treatment Type" field defaults to 'Standard' when a quote is first being raised.
When a variant has been selected, this field will be updated to show 'EV Exempt Reportable' if;

  1. the linked "Vehicle Type" is FBT exempt
  2. the vehicle's Base Price is below the LCT threshold
  3. the quote's “Luxury Car Tax” is '0.00'
  4. if the vehicle is 'Used' or 'Demonstrator' then also
    1. “Date First Held” must be on or after '01/07/2022'
    2. “Luxury Car Tax Paid” must be 'No'
    3. if “Date First Held” and “Luxury Car Tax Paid” have not been populated, "FBT Treatment Type" will revert to 'Standard'

In some cases, you can update this field directly to show 'EV Exempt - Reportable' if required.
If the stored vehicle type is not exempt, you will see an alert: Vehicle Type is not FBT Payable Exempt - are you sure?

Open the Luxury Car Tax Calculations spreadsheet to see the logic used to get the quote's default "FBT Treatment Type" and to see how the Base Price is calculated.

[edit]

Vehicle Type - fbt_payable_exempt_flag

The linked "Vehicle Type" determines if it is classified as 'Standard' or 'EV Exempt Reportable'.
If [qt_vehicle_types]fbt_payable_exempt_flag = 'yes', the variant is eligible to be classified as 'EV Exempt Reportable' in a quote if the other conditions listed above are met.

If your quotes are not returning the "FBT Treatment Type" you expect, check and update the qt_quote_profiles "Vehicle Type" to suit.
Visit the process page Update a Profile Vehicle Type for instructions.

[edit]

Quotes Placeholder Logic

If the "FBT Treatment Type" is 'Standard'

Quotes will show either an ECM Amount or an FBT Payable Amount.

If the "FBT Treatment Type" is 'EV Exempt Reportable'

Quotes / Contract "Use ECM?" is un-checked
Quotes will show both the ECM Amount and the FBT Payable Amount as 0.

If a variant is already selected in your quote and you update to use a 'Standard' vehicle, the "Use ECM?" field will update to match the Clients / Quote Defaults "ECM Flag" field.
