Overview


Our password based legacy web services are no longer supported by Catch-e.
You must use the token based APIs instead.
New use of legacy web services is not authorised by Catch-e.

If you have any APIs you want to use that are not available in the token based suite of APIs contact your Account Manager to discuss.


Summary

Executes an SQL script comprising of one or more SQL statements of the following type, UPDATE / INSERT / DELETE / SET. You can use this service to update data in the Catch-e database. Each SQL script is logged in the gb_imports table for audit and possible troubleshooting purposes.

Transaction Safe

The SQL Script must be transaction safe.
