[edit]

Overview

Access Type: External

Restriction Type: Can view contracts and drivers that are linked to the nominated client record(s).

Set Up: Can be set-up to have access at a client level using either

  1. the Users / Detail "Restrict Key" field to nominate one client code, or
  2. checking on the Users / Detail "Restrict Range" check box and then nominating the Clients, Client Divisions, Client Cost Centres and States that the user can access.
