November and December 2010 distributed on 07/02/2011

  • Start Period - the Client’s Start Period default (which defaults in that client’s contracts) can now be set as a standard in Global Controls.
  • Division - the field “Division” has been added into the quote record. Like “Cost Centre”, if a quote is accepted, the division must be linked to the Client’s list of divisions, or be blanked out.
  • Reimbursement Notification - a generic email name and address can now be used for the Reimbursement Notification feature instead of the Users if preferred.
  • Receivables - the Receivables module is now operational and allows you to perform your cash receipting in Catch-e. This gives you the extra flexibility to use Catch-e for your Debtor management.
  • Import - if you have a large volume of receipt data, you can use our Receivables “Import” feature to upload a bulk file of receipts. These are allocated on an oldest first basis and if an overpayment occurs, an “On Account” entry is created for the contract.
  • Insurance - a new role has been created called “Insurance”. This is like "user" but has Read-Only access to the Finance screen.
