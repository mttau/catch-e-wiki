Overview

Bespoke fields can be configured for you to store information that is particular to your business.

These can used in and displayed in three system locations:

  • Clients / Other
  • Quotes / Other
  • Contracts / Other

Visit the Bespoke Fields page for full details on how these screens and fields can be configured and used.

[edit]

Your own tabs

If this tab is displayed in your screens (perhaps with another name), then your organisation is using Bespoke Fields.
Refer to your organisations documentation and processes for information about the use of these screens.

This flag is used to trigger a Share of Savings calculation in a Novated Lease quote. Here is a zip file (Share of Savings Calculations) which contains a spreadsheet that uses a macro to model results.
