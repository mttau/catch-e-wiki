[edit]

Overview

The selected contract type determines which budgets can be used on invoices and expenses.

[edit]

Contracts

This field is displayed on the Contracts / Contract tab and is editable on the Contracts / Billing tab.
It is populated when a quote has been accepted.
If you create a contract, you can enter or lookup a contract type directly.

[edit]

Changing a "Contract Type"

You can change a contract's Contract Type in the Contracts / Billing tab.

When a contract type is changed, the posted billing records are unaffected, but pending periods are updated and only keep the Contract Type / Accruals budgets of the new contract type

Any other existing budget records are removed, even if they contain values.

Any inserted records will also only add budget records that exist for the current Contract Type / Accruals budgets.
