Quotes

Screen NameTypeDescription[table]field
Quotes / Vehicle
Initial CTP GST Number The "Initial CTP GST" calculation is varied for NZ systems.
When the Quotes / Vehicle "Initial CTP Exc GST amount is entered, the GST amount will be calculated using the value stored in the Global Control for gst_rate.
The calculation is as follows
"Initial CTP Ex GST" * gst_rate = "Initial CTP GST" 		[qt_quotes]ctp_initial_gst

Settings to check

ItemComment
Setup / Reference Data → gb_controls → gb "country_code"The value should be 'NZ'
Setup / Reference Data → gb_controls → gb "gst_rate"Check the GST rate is correct for NZ.
[gl_gst_codes]gst_rate Check the GST rate is correct for NZ.
[qt_vehicle_types]lct_exempt_flag Update so that all records have this field set to 'yes'. This will prevent Luxury Car Tax (LCT) and Luxury Car Charge (LCC) values from calculating.
