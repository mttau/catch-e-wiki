Overview

Review each section below and set-up financier details as required for your system. Contact your Account Manager for guidance if required.

[edit]

Set-up a Supplier Account for each financier

Enter basic contact information for the financier as a supplier

  1. Suppliers / Detail - enter basic supplier details.
  2. Suppliers / Account "Supplier Type" - set as 'Financier'
  3. Suppliers / Account "Interface Type" - set as 'angle' if you are setting up records for Angle Auto Finance
  4. Suppliers / Financier "Finance Method" - choose the require method
[edit]

Set-up a Broker for each financier (optional step)

Do this is if you want to link a broker to the financier record. Visit the Broker page for more details about how brokers work in the system.

  1. Suppliers / Account "Agent" - in the Financier record, lookup and add the required broker
[edit]

Set-up default financiers for quotes

The Quotes / Finance "Financier" is populated in the following preference order. Review and populate the required records.

  1. If Clients / Quote Defaults "Financier" is populated, use this financier.
  2. If Contract Types / Details "Quote Financier" is populated, use this financier.
  3. Lookup or directly enter a financier.
[edit]

Set-up interest rates for each financier

Process page: Create Financier interest rate records

The Quotes / Finance "Base Rate" is populated in the following preference order. Review and populate the required records.

  1. Values stored in the Setup / Reference Data 'fm_financier_rates' table.
  2. Setup / Reference Data 'gb_controls' "interest_base_rate"
  3. Directly enter a base rate.
[edit]

Set-up client margins for each financier

The Quotes / Finance "Client Margin" is populated in the following preference order. Review and populate the required records.

  1. Values stored in the Clients / Financiers "Margin Rate" field for the selected "Financier"
  2. Setup / Reference Data 'gb_controls' "interest_margin_rate"
  3. Directly enter a margin.
[edit]

Set-up quote defaults for each financier

Process page: Create default quote fees for a Financier by Contract Type

  1. Enter each financier's default fees and preferences in Setup / Reference Data 'qt_financier_posting_map_defaults' table
[edit]
Notes

If the 'qt_financier_posting_map_defaults' table is not populated, then no default values will be applied for the quote fields listed below.

  • Advance / Arrears
  • Est / Admin Fee (except for Novated Leases which users [gb_controls]novated_establishment_fee as the default instead).
  • Originator Fee
  • REVS / PPSR
  • Mortgage Duty
  • Fee

Useful Queries

select s.supplier_code, s.name, s.supplier_type, f.finance_method 
#, s.interface_type
, fr.term, lower_limit, upper_limit, interest_rate, date
from fm_suppliers as s
left join fm_financiers as f on f.supplier_id = s.supplier_id
left join fm_financier_rates as fr on fr.supplier_id = s.supplier_id
where s.supplier_type = 'financier'
;
select s.supplier_code, s.name#, s.supplier_type
, pm.posting_map_code as contract_type, pm.posting_map_group as ct_group
, fpmd.term, revs_fee, mortgage_duty_flag, include_commission_flag, novated_commission_rate
, novated_commission_maximum_amount, novated_commission_maximum_rate, novated_commission_maximum_basis
, establishment_fee, establishment_fee_cost, originator_fee, administration_fee, advance_arrears
, finance_budget_term_deferred, effective_date
from fm_suppliers as s
left join qt_financier_posting_map_defaults as fpmd on fpmd.supplier_id = s.supplier_id
left join gl_posting_maps as pm on pm.posting_map_id = fpmd.posting_map_id
where s.supplier_type = 'financier'
;
