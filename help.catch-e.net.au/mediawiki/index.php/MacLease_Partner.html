Main Page → Workflow → Users → Quotes → MacLease Partner

Qualifiers

  1. You need to to use the Quotes / Finance button.
  2. The Maclease Partner Interface feature is set-up in your system
  3. Your business is listed with Macquarie Leasing as using API Partner services.
  4. You have received an email from Macquarie with the following details;
    1. Firm
    2. Mac ID
    3. API Token


Process

  1. Forward the login details to one of your System Administrators
  2. Your Administrator can then add you login details to your user records to enable the Apply button.
