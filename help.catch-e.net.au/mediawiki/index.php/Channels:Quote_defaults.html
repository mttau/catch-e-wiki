Overview

The Channels / Defaults tab stores values that will populate into the matching Clients / Quote Defaults, Clients / Contract Defaults and Contracts / Finance fields when new client and contract records that are created for the selected channel.

Note: If you want to use Import Table Data to create your [fm_channels] records, a Channel Quote Defaults [qt_channel_defaults] record is mandatory
