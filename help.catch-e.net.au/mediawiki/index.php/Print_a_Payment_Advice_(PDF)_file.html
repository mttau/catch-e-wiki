Print a Payment Advice (PDF) file

Jump to: navigation, search

Main Page → Workflow → Fleet Management → Payments → Print a Payment Advice (PDF) file

Qualifiers

  1. The payment record or batch is posted.
  2. Below are three ways to print the Payment Advice


Payments / Periodic (current file)

  1. Go to Payments / Periodic
  2. Status - 'Posted'
  3. Use various filters to narrow down the screen selection to the record or batch you want to print the Payment Advice for.
  4. Click on
  5. Check the selection is correct
  6. - for any valid row, click on the PDF icon


Contracts / Payments (current file)

  1. Go to Contracts / Payments
  2. Payment Type - select the one you are looking for
  3. Schedule - select the one you are looking for (if there is more than one)
  4. - click on the PDF icon on the row you want the Payment Advice for


Mail Queue (original file if emailed)

  1. Go to System / Mail Queue
  2. Type - select 'Payment'
  3. Status - select 'Sent'
  4. Use various filters to narrow down the screen selection to the Mail batch you want to print the Payment Advice for.
  5. Click on
  6. - click on the PDF icon on the row you want the Payment Advice for
