[edit]

Overiew

Import Redbook photos from Redbook's FTP server.

[edit]

Manual Processing

We recommend using scheduler jobs to process RedBook data, but you can also manually import the data in the following system menu.
Fleet / Import Data → RedBook Data
Fleet / Import Data → RedBook Photos

[edit]

Job Process

  1. Download the zip file containing the latest Redbook photos from Redbook's FTP site
  2. Extracts files into the standard photo directory.
  3. New photos are created and existing photos updated as required.
  4. A report can be send to you listing all the changes made.


  This Job Type requires additional set-up. You will need to forward your Redbook FTP connection details to mailto:support@catch-e.com.au first.


[edit]

Parameters

These are the different parameter settings that you can set up on the Job Screen screen:

Parameter Notes
Filename This is the name of the file to download from Redbook's FTP server e.g. RBPhoto.zip.
