Overview

This popup is launched from the "Description" field lookup when the screen is in edit mode.
When opened, the details of the stored variant will be shown.

If the stored variant is not correct, you can use this popup to search for the correct variant and update the contract.
Use the filters and buttons described below to find and select the right variant.
