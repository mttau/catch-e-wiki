Workflow

New events are started by clicking on the New button. Click on Create to save the event.
Use the # check box to select an event to Edit or Delete.
Use the # check box to select an event to Attach a document to. Attached documents can be downloaded for viewing by clicking on the paper clip icon .
Every event can be managed by flagging the required action, setting a due date to be completed by and also giving you the flexibility to mark the event as completed or incomplete.
Incomplete tasks are summarised in the KPI Home screen.

Set-up Notes

Review the default suite of event records and update to suit your business. These records are stored in [fm_driver_events_codes].

[edit]
Change the "Code" sort order

The "Code" list shown when you create new events is ordered using the field [fm_driver_event_codes]driver_event_code.
Update these codes to change your sort order.
Before you do this, check and update any links you may have to scheduler mailQuery jobs.

[edit]
Add a custom list to an event "Value" or "Description" field

If you want a predefined selection list for the "Value" field, add records into the table [fm_driver_event_values]. You can also set a default record for the list. If you want a predefined selection list for the "Description" field, add records into the table [fm_driver_event_descriptions]. You can also set a default record for the list. Pre-defined values and descriptions can be used separately or together. Contact your Account Manager to help you set up any value or description lists you want to use.
