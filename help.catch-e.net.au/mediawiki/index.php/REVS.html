[edit]

Overview

"REVS / PPSR" can be entered into a quote in the Quotes / Finance tab and will populate automatically if a linked financier default is set-up.

A "REVS / PPSR Fee" field also exists in the Clients / Quote Defaults tab. This is a reference field only and it is not used in any system functions.

[edit]

Client set-up

Go to Reference Data / qt_financier_posting_map_defaults and enter a "REVS / PPSR" amount for each Financier / Contract Type combination that you want to use.

Vehicle encumbrances are handled by the nationwide PPSR system as of 30/01/2012. State based values have been deprecated.
