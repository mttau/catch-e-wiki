Overview

Channel data is stored in the tables [fm_channels] and [qt_channel_defaults].

[edit]

Importing Channel records

If you want to use Setup / Whole Table Import to create new channel records in [fm_channels], then you must also import matching [qt_channel_defaults] table records.

[edit]

[edit]

Quote PDF Template and Email

Channel branding can be used if you create separate Quote Templates. If a channel quote template is configured the channel branding will be used if the quote's client is linked to the channel. The quote email text can also be channel-specific.

[edit]

Payroll Advice

A channel logo can be embedded into the Payroll Advice template, so that different logos can be invoked on the form for different channels.

[edit]

Bank Fields

The table fm_channels contains optional bank fields which can be used on invoice templates (templates need to be setup by Catch-e to use these fields):
bank_account_name
bank_bsb
bank_account_number

[edit]

Separate Channel branding on screens

You can provide your channels with a separate logo and colour scheme on the system screens. Visit Appendix F: Client Login Setup for full details of how the URL for a channel is set-up.

[edit]

Invoices

Invoice logos and footer details can be customised to cater for Channel specific Bank Details, Addresses etc. A separate invoice template can be set up for each Channel. See Channel Billing section. To request a quote to set this up please contact your Catch-e Account Manager. Refer to Billing Templates for specific setup notes.

[edit]

Emails

System generated emails for Channel related functions can use a customised message content. Further detail on what can be done and how to set-up can be found in Email Setup
Refer to Driver Report Setup for specifics of configuring an email for a 'With Funds Driver Report'.
Also visit the Driver Contract Access page if you want to set-up a channel-specific driver access email and user guide.

Certificate of Currency (COC) - you can set-up channel specific emails that contain a customised coc.pdf attachment.

[edit]

Emails - Scheduler

Emails created using the Scheduler's Mail Query function can be configured to send only for a channel.

Channels can be catered for in these emails with the following adaptions:

  1. Channel specific logo on Office(xls) report attachments
  2. Channel specific content for the subject and body of the email
[edit]

Reports

Reports can be configured to use a channel-specific logo. Note: because this is at the report generator level it will automatically apply to ALL PHP template based reports if they already include channel_id in their output.

Refer to Reports Setup for logo setup notes.

[edit]

Logo

The field channel_code uses the same name as the stored logos.
E.g. [fm_channels]channel_code = retail.
Thus, the stored logos should be retail_logo.bmp, retail_logo_m.png and retail_logo_s.png

Example: for channel_code retail the channel level logo would be:

/clients/[client_name]/channels/retail/images/Page/logo_m.png
Note: the folder name (retail) must be lower case. The channel_code itself may be a mixture of upper & lower case.

, and the standard logo is:

/clients/[client_name]/images/Page/logo_m.png
