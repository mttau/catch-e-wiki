Overview

  1. A new budget item called Insurance Uplift (INSUP) is introduced.
  2. INSUP automatically calculates a monthly allowance to cover annual Comprehensive Insurance increases.
  3. The calculation is based on the Monthly Premium in the Comprehensive Insurance Profile section of the Lease tab.
  4. INSUP will be applied to all Quotes, regardless of whether the insurance is through the Internal Insurer or a Driver Policy.
[edit]

Calculation

A budget amount will calculate and populate if set-up to do so.
Open the Insurance Uplift calculation model.xlsx to see the working calculation.
The INSUP Budget will not calculate if the Insurance Uplift Rate is set to '0.00'.

To use this budget:

Go to Posting Classes / Details and check that "INSUP" exists and create it if it is missing.
When creating 'INSUP', make sure the event check-boxes are un-checked. The class "Name" displays as the Quotes / Lease Budgets also.
Go to Contract Types / Accruals and add the "Class" 'INSUP'.
