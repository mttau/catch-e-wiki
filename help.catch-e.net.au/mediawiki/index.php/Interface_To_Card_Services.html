Overview

Card Services allow users to create card records, import and process files for card-based transactions.
Visit Card Interfaces for a directory of the available interfaces with a link to detail page for each one.

[edit]

Menu Summary

Fleet / Card Services
Transactions / Batches Import, Validate, Correct and Post card transactions.
Transactions / Transaction Displays the details of one card file transaction.
Cards Create, edit and view card details with their associated business rules.
Fleet / Controls / Interfaces / Card Services
Interface Find and update card interfaces.
Mappings Review and update card product mappings to system classes and vmrs codes.

Card Interfaces

Interface Supplier Description
7ELEVEN7-Eleven Australia The 7ELEVEN card can be used for purchasing fuel as well as a range for vehicle services such as maintenance, tyres, scheduled services etc.
The file format must be .txt, i.e. Text (Tab Delimited).
Column headings are expected in the file a well as three account opening and closing balances.
BPBP Australia Fuel and services format 16 file.
BPNZBP New Zealand Fuel and services restricted to BP NZ outlets only.
CALSFAmpol One of the Ampol interfaces. This includes vehicle servicing from authorised centres (e.g. Dealerships).
CALNPAmpol One of the Ampol interfaces. The transactions are restricted to those generated at Ampol outlets. This is an alternative interface to 'CALNS'.
CALNSAmpol One of the Ampol interfaces. The transactions are restricted to those generated at Ampol outlets. This is an alternative interface to 'CALNP'.
CHARGEFOXChargefox This interface imports fuel transactions via fuel import file. This file must be in .csv format.
CLINKLinkt Toll charges invoiced by Linkt. Unzip the .zip file supplied and import the .CSV file.
CUSTACustom Fleet - All transactions This interface imports all vehicle contract related transaction types from the standard Custom Fleet text file.
EVIEEVIE This interface imports electricity charges via an import file. This file must be in a .csv format.
FLEETAFleet Card - All transactions This interface imports all transaction types from the standard Fleet Card text file. Interface was formerly named CUSTA.
FLEETOFleet Card - Fuel transactions An electronic version of their invoice which includes fuel and servicing transactions. Lease rental charges etc are excluded. Interface was formerly named CUSTO.
FLEETCORFleetcor This interface imports fuel transactions via fuel import file. This file must be in .csv format.
FLEETCORDFleetcord This interface imports fuel transactions via fuel import file. This file must be in .dat format.
FPARTFleet Partners This interface accepts the standard FleetPartners .csv electronic invoice file.
FSMARTFleet SmartInterface used in New Zealand for fuel and servicing.
MOBILMobil The Mobil interface for Fuel and services restricted to Mobil outlets only.
MOBILNZ Mobil The Mobil interface for Fuel and services restricted to Mobil outlets only and tailored for use in New Zealand.
MPASSWEX Motorpass The WEX Motorpass (WEX) card can be used for purchasing fuel as well as a range for vehicle services such as maintenance, tyres, scheduled services etc.
This interface is used to upload "E05" format files.
The file format must be .txt, i.e. Text (Tab Delimited). Column headings are expected in the file but not account opening and closing balances.
MPASS07WEX Motorpass This interface is used to upload WEX Motorpass (WEX) format files., previously referred to as "E07".
WEX has recently (from before 01/07/2021) changed the file format to "E17". The "E17" file format must be .txt, i.e. Text (Tab Delimited). Note the E07 file format will fail as it has 31 columns, ensure you use the E17 file format which has 30 columns.
NABNational Australia Bank The NAB interface can be used to import charges as fuel expenses from a spreadsheet derived from NAB internet banking site. The .xls file needs to be saved as .csv prior to importing.
ORIXOrix Raw data sheet from the Orix excel Tax Invoice. Save As .txt Tab Delimited.
PUMAPuma The Puma card can be used for purchasing fuel as well as a range for vehicle services such as maintenance, tyres, scheduled services etc.
The file format must be .txt, i.e. Text (Tab Delimited).
Column headings are expected in the file a well as three account opening and closing balances.
REIMBURSEReimbursement Set up requirement to facilitate the completion of Claim Reimbursements via the Contracts / Claim tab.
Set-up a Reimbursement Interface
SGFleetSG Fleet Xls File format, convert to tab delimted .txt before importing.
SHELLBDDShell Australia Bulk Data Download (BDD) File format for fuel and services restricted to Shell outlets only.
This interface is no longer supported by Shell Australia.
The interface was abandoned by Shell and ultimately was never properly released in Australia. This interface should not be used.
SHELLShell Australia EC File format for fuel and services restricted to Shell outlets only.
SHELLCShell Australia EC File format for fuel and services having been run through the conversion program supplied by Shell (decomissioned by Shell 24/6/2013).
SHELLOShell Australia Legacy format for fuel only.
SMARTFLEETSmartFleet File received from SmartFleet and should be saved in .csv format.
SMRInnovation GroupService, Maintenance and Repairs Data to be used in conjunction with either Redbook or Glass's Vehicle Data.
SUMMITSummit Fleet Leasing The Summit interface imports their tax invoice. This is received as a .xls file and needs to be saved as a Text (Tab Delimited) .txt file.
TFSToyota Financial Services The TFS interface can be used to import lease and other charges from the TFS supplied spreadsheet. The .xls file needs to be saved as .csv prior to importing.
TROPICTropic Fuel Use this interface to import fuel charges using the *.csv file supplied by Tropic Petroleum. https://www.tropicd.com.au
UNITEDUnited CSV (or CSV zipped) file formats only for United.
