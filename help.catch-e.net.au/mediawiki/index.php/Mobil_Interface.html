Main Page → Fleet / Controls / Interfaces / Card Services → Interface | Mappings → Card Interface Set-up → Mobil

  • A fuel record is created for every Transaction Item record (Type '15) in the interface file.
  • The interface file is a TXT format file. An example of the file structure can be seen here.
  • The Mobil Card Fee in the Transaction Header record (Type '10') is allocated to the first Transaction Item record (Type '15) that comes after it. All other subsequent Transaction Item records that come after that Transaction Header record (if any) will have the Mobil Card Fee set to zero. That way there will be one Mobile Card Fee per Transaction.
  • GST is calculated on the Mobil Card Fee using the GST rate looked up from the the sytem controls using the current date.
