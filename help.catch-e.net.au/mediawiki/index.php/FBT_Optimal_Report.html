[edit]

Overview

This report compares the statutory and operating methods and recommends the lowest tax cost method. Vehicles which are recommended to change from one method to the other are highlighted in yellow on the report. Based on the results you can then change a vehicles default method on the fbt screen and run the final statutory and operating method reports for your client.

Refer to the FBT Operating Report and the FBT (Statutory) Report for specific business rules for each method.

  • Field called 'Non Use days' can be populated in the FBT Optimal Report by adding Non Use days event in the Contract/Event section

Refer 'Days Not In Use' field from the Contracts / FBT section to get more details

If the Contracts / FBT "Benefit Type" = 'statutory', the value recorded in the "FBT Reduced Value / Date" field is applied in this report in the 5th FBT year.


This report can be emailed to the clients in a scheduler job. More details are available on the System Reports page.
