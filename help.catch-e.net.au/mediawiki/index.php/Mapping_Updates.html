Main Page → Workflow → System Administration → User Management → Mapping Updates

Qualifiers

  1. Angle Finance has an API connection with your system
  2. Angle Finance has new field name(s) that need to be added or removed from their mapping tables.


ADDING a new 'field name' to the mapping set up

  1. Go to Setup / Export Table Data
  2. Select Module = 'qt|Quotes'
  3. Select Table = 'qt_interface_field_translations'
  4. Select 'Download'
    1. When you download the file excel will want to convert the large number in the first column into a scientific notation.
    2. Be sure to select 'Don't Convert' to retain all the numbers else excel will abbreviate the number.
    3. This will corrupt the data for import purposes and you will need to export the file again.
  5. Find the existing field names in the exported file and copy the value in the 'interface_field_id' into the same column at the bottom of the data set in a new line.
  6. Leave 'interface_field_translation_id' blank
  7. In the 'field_value' enter the related Catch-e value
  8. In the 'interface_value' enter the new Angle Finance 'Title'.
  9. 'status_flag = 'active'
  10. Save as a .csv file



IMPORTING the saved .csv file

  1. Go to Setup / Import Table Data
  2. Select Module = 'qt|Quotes'
  3. Select Table = 'qt_interface_field_translations'
  4. Attach saved file to import
  5. Select 'Upload'
  6. If 'Upload' fails, open the Error Report to determine why it failed - correct errors and then Re-import
  7. A successful 'Upload' will prompt you to select 'Import' in the bottom right of the screen to complete the process.


REMOVING an existing 'field name' from the mapping

  1. Go to Setup / Export Table Data
  2. Select Module = 'qt|Quotes'
  3. Select Table = 'qt_interface_field_translations'
  4. Select 'Download'
    1. When you download the file excel will want to convert the large number in the first column into a scientific notation.
    2. Be sure to select 'Don't Convert' to retain all the numbers else excel will abbreviate the number.
    3. This will corrupt the data for import purposes and you will need to export the fiel again.
  5. Find the required 'Salutation' in the list to remove
  6. Update 'status_flag' to 'inactive'
  7. Save as a .csv file
  8. Go to Setup / Import Table Data
  9. Select Module = 'qt|Quotes'
  10. Select Table = 'qt_interface_field_translations'
  11. Attach saved file to import
  12. Select 'Upload'
  13. If 'Upload' fails, open the Error Report to determine why it failed - correct errors and then Re-import
  14. A successful 'Upload' will prompt you to select 'Import' in the bottom right of the screen to complete the process.
