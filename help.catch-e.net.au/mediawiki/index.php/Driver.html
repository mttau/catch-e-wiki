[edit]

Overview

Access Type: External or Portal
Drivers can be set-up to access a restricted set of Catch-e system screens, or they can be directed to a portal login.

Restriction: A driver can view selected Contract screens and the Driver Contact screen. In addition they can reset their own Password.
Set Up: A Driver must have a Contract to be granted a Driver login. Further set up instructions are found in the Driver Contract Access page.
Fleet / Administration / Contracts / Events - view and create events based on external event flag set up.
Note: Use the Package Employee role instead if you want to provide salary packaging information from this system.
