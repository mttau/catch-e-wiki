Overview

Create an Impact Statement that can be printed from the Employees / Packages tab.
It is a customised screen report. It can be renamed to suit your business.
It uses stored factors for PAYG Tax and HECS to generate statements with calculated Income Tax.

[edit]

Set-up

During a system implementation, your Account Manager will help you build this report.

You will need to make ongoing updates to keep the Tax Rates up to date, as the report does not refer to the stored controls.

[edit]

Ongoing Maintenance

[edit]
Tax Details

Refer to the following Help wiki pages and your own resources to update the stored factors as required.
Main Page → Setup / Reference Data → gb_controls → Tax Rates | ATO Loan Rates

[edit]
Get the current template
  1. Go to Setup / Import or Export Files to download your file
    1. Export - check this radio button
    2. Platform - 'Catch-e'
    3. File Type - 'Packaging Templates'
    4. File Name - 'impact_statement.xlsx'
    5. Click on Export
[edit]
Make changes
  1. Tax rates are stored in tab input. Look and find the cell range used in your spreadsheet.
  2. Save a copy of this template (E.g. impact_statement_YYYY-MM-DD.xlsx)
  3. Update rates
  4. Save a new copy (E.g. impact_statement.xlsx)
[edit]
Upload the new template
  1. Go to Setup / Import or Export Files to download your file
    1. Import - check this radio button
    2. Platform - 'Catch-e'
    3. File Type - 'Packaging Templates'
    4. File Name - 'impact_statement.xlsx'
    5. Browse and upload or drag and drop
    6. Click on Import
