Overview

The Order Management screen enables you to see and manage all the vehicles you have on order.
Vehicles on order can be identified by their "Reg Number" - ORDER. Contracts with a "Suspend Date" are excluded (i.e. cancelled orders).

You can use a range of filters and dynamic column sorting to show the desired vehicles on screen.

Filtering examples:

* Filter by a particular supplier when you are talking to someone from that Dealership
* Filter by "Confirmed To" date to check that cars that are being delivered tomorrow are all in order.

You can update relevant order dates and status fields for all the displayed vehicles directly on this screen and hyperlink to the vehicle's Contract/Order screen if desired.

Action buttons

Field Description
Press 'Go' to execute your chosen filtering. If 'Go' is greyed out, then the current filtering has been executed.
Print a PDF or Text(CSV) output of the filtered search results. All the fields displayed on the Search screen are included in the output. If filtered results belong to multiple Channels, the PDF report will add a page break for each new Channel.
Use the Edit feature to update order dates and other information for multiple contracts on this screen.
