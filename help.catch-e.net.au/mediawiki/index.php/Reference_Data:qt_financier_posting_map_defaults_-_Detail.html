Overview

Most of the [qt_financier_posting_map_defaults] record fields are shown in the List tab, but not all.
The Detail tab shows all of the available fields.

[edit]

Process Pages

Create default quote fees for a Financier by Contract Type

Header

Filter Type Description [table]field
Financier Posting Map Default ID Number Shows the selected record. Use the lookup to find another record go back to the List tab.

Field Descriptions

The fields below are listed in the order they appear on the Quotes / Finance tab.

Field Type Description [table]field
IDNumThe unique number of the qt_financier_posting_map_defaults record. [qt_financier_posting_map_defaults]financier_posting_map_default_id
FinancierTextSupplier Code of the financier. Hyperlink navigates you to the Suppliers tab.[fm_suppliers]supplier_code
Contract TypeTextContract Type code. Hyperlink navigates you to the Contract Types tab.[gl_posting_maps]posting_map_code
TermNumThe lease term, e.g. 12 months, 24 months. If a quote is created for a term not exactly matching the previous term is used, e.g. a 38 month quote would use the 36 month rate[qt_financier_posting_map_defaults]term
Advance / ArrearsListUsed to calculate finance costs as applicable. Select either 'Advance' or 'Arrears'[qt_financier_posting_map_defaults]advance_arrears
Finance Budget Term Deferred Drop List Choose from ‘Any’, ‘No’ and ‘Yes’. ‘Any’ is the default value.
Setting this flag to 'No' or 'Yes' will override the Clients / Quote Defaults "Finance Budget Term Deferred" selection. 		[qt_financier_posting_map_defaults]finance_budget_term_deferred
Est / Admin FeeNumA fee charged by the financier may be included here. Refer to the Establishment Fee page for full calculation details. [qt_financier_posting_map_defaults]establishment_fee
Est / Admin Fee CostNumThe cost of the "Est / Admin Fee" your business is charged by the financier may be included here. Refer to the Establishment Fee page for full calculation details. [qt_financier_posting_map_defaults]establishment_fee_cost
Originator FeeNumAn Establishment Fee amount can be included and interfaced to Financiers as required (e.g. this field is included in the St George Interface)[qt_financier_posting_map_defaults]originator_fee
REVS / PPSRNumYou can include an encumbrance fee into the financing. If an amount is entered here, this value will be populated into the quote.[qt_financier_posting_map_defaults]revs_fee
Mortgage DutyCheck BoxMortgage duty applies in NSW for certain finance products, e.g. a Consumer Loan. Open the Mortgage Duty spreadsheet to see the full calculation and logic used. This is not selected by default but can be selected.[qt_financier_posting_map_defaults]mortgage_duty_flag
Fee Number If a monthly administration fee is charged by your financier, enter the GST exclusive amount here.

This will be populated into the Quotes / Finance Monthly Payments "Fee" field and forms part of the finance amount calculation.
It is also included in the 'FIN' budget.

Visit the Angle Interface Setup for their fee details.

[qt_financier_posting_map_defaults]administration_fee
Include Commission Check Box When checked, Quotes / Finance "Commission Rate / Fee" will be calculated using the applicable Commission business logic and calculations.

It is checked by default when creating new records, but can be unchecked.
When unchecked, "Commission %", "Commission Max" and "Commission Max %" are all re-set to 0.00 and disabled.

If the field is unchecked, Quotes / Finance "Commission Rate / Fee"" will be re-set to 0.00 on existing quotes when they are next edited.

[qt_financier_posting_map_defaults]include_commission_flag
Commission % % (5dp) Enter a percentage here to apply a standard rate to novated quotes.

For context for this field, go to the Commission page to see the business logic used to apply a commission to a quote and open the commission model to see how the commission calculates using different settings.

[qt_financier_posting_map_defaults]novated_commission_rate
Commission Max Num (2dp) Populate this field if the Financier has set a maximum allowable commission based on a dollar amount.

For context for this field, go to the Commission page to see the business logic used to apply a commission to a quote and open the commission model to see how the commission calculates using different settings.

[qt_financier_posting_map_defaults]novated_commission_maximum_amount
Commission Max % % (5dp) Populate this field if the Financier has set a maximum allowable commission based on a percentage.

For context for this field, go to the Commission page to see the business logic used to apply a commission to a quote and open the commission model to see how the commission calculates using different settings.

[qt_financier_posting_map_defaults]novated_commission_maximum_rate
Commission Max BasisListboxSelect from ‘Amount Financed’, ‘Depreciable Amount’ and ‘On Road Price'.
‘Amount Financed’ is to be the default value used when creating new records. This field is populated for all records, even if "Include Commission" is unchecked. The value in this field is only used if "Include Commission" is checked.

For context for this field, go to the Commission page to see the business logic used to apply a commission to a quote and open the commission model to see how the commission calculates using different settings.

[qt_financier_posting_map_defaults]novated_commission_maximum_basis
EffectiveDateThe effective date of the record. When quotes lookup values, the record with a date that is on or before the quote's "Effective Date" will be used.[qt_financier_posting_map_defaults]effective_date
