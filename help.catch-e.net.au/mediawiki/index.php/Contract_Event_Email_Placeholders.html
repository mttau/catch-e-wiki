Overview

Download this request form Contract Event Email Request and send it to Catch-e Support if you want to set-up a Contract Event Email.

[edit]

Placeholders

The placeholders listed below are available for use in Contract Event Emails when [gb_templates]name = 'pdf_contract_event_email_message'.

The markers ## or {} can be used to make placeholders. E.g. #contract_id# or {contract_id} are both fine.
