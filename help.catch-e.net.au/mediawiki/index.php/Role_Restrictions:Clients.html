Clients

Name[gb_role_restrictions]nameComments
Client tabstype = sub_page_map, restriction_type = bit_arrayThe client tabs that are accessed by roles can be configured by updating the bit array.
By default, the Cost Centre and Other tabs are hidden.

Note: The Clients / Other tab will not be available to external users, regardless of the bit_array setting entered here. 

SELECT * FROM gb_role_restrictions WHERE page = 'clients' AND type = 'sub_page_map' ORDER BY role_id;
