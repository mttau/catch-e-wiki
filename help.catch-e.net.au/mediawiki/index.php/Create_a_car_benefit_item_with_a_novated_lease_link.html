Create a car benefit item with a novated lease link

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Items → Create a car benefit item with a novated lease link

Qualifiers

  1. Your system has been set-up to allow direct link Novated Lease contracts
  2. An employee would like to package a Novated Lease
  3. You have created a vehicle quote for this


Gather information

  1. Navigate to Quotes / Contract
  2. Select the required quote
  3. Print and save this vehicle quote
  4. Navigate to Employees / Current Packages
  5. Select the screen report 'Vehicle Quote Details'
  6. Click on
  7. Open and save this report

Create the item record

  1. Navigate to Employees / Item
  2. Click on
  3. Use information from the quote and the report generated above to enter the required information into the item
  4. The item should be set with the following defaults:
  5. Compress - un-checked
  6. Inertia - un-checked
  7. Status - 'Quoting' or 'Pending'
  8. Disbursement Type - 'Claim'
  9. Supplier - 'Employee Disbursement'
  10. Reference - enter the Quote ID of the vehicle quote (no spaces, no other characters. E.g. '123456').
  11. Payee - 'Employee'
  12. Payment Method' - 'Direct Credit'
    1. Add the employee's bank account details
    2. Claim - must be checked
  13. Click on


Next Steps

  1. When the employee has signed off on the vehicle quote paperwork
  2. Activate the car benefit item
