Overview

The legacy version of the Balance screen uses the same details and values that are in the Driver Report.
If you navigate to the Contracts / FBT tab and Print the 'Standard' report, you'll get a similar layout and the same balances.

Report Content

The content of the report depends on the vehicle's nominated Contract Type which links to two templates. One is designed for a Statutory benefit and the other for an Operating benefit.
The system looks to the "Benefit Type" recorded on the Contracts / FBT tab to determine which template to apply here.
If the "Benefit Type" is 'exempt', this screen uses the Statutory template.
For this report, the information used is the same regardless of which benefit type is assigned to the vehicle, as no FBT information is used.
