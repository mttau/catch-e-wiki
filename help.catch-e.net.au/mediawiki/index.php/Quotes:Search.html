Overview

The search screen is used to display quotations.
From this screen a new quote can be created or existing quotes can be selected for editing and further processing.
A number of filters in the header section of the screen allow the user to search for particular quotes or types of quotes.
The default criteria for this screen is to display all active quotes, but results are limited to 1000 records by default. See the Pagebar Limit field description for more details.

Pagebar

Field Type Description [table]field
Page Number Use the arrows to navigate to the First Page, Previous Page, Next Page or Last Page .
You can skip to any page within the range by typing the page number into this field. Hit 'Enter' to refresh the screen.
Page x of x Number Displays the page you are on and the total number of available pages.
Limit Number The quotes shown on navigation to this screen are limited to 1000 records by default to optimise the loading speed of the page.
You can update the limit on your screen by entering any number in this field.
To show all active quotes, remove the limit amount or update the field to '0'.
The default limit is applied again next time you navigate into the Quotes module.

Your System Administrator can update or remove this limit by updating [gb_controls]quotes_search_results_limit.

Quotes / Search "Rate" doesn't match the Lease Total

If a quote has a budget that is no longer also stored in the linked Contract Types / Accruals tab, it will form part of the Quotes / Lease Totals, but it will not be part of the calculated Quotes / Search screen "Rate" amount.
To fix this, go to the quote, Edit and then Save it. This will cause the links on the quote to be checked and restored.
