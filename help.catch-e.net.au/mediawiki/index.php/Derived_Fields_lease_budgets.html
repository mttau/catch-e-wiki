Derived fields from the /lib/PDF/lease_budgets

Use this library is used to derive the GST Overridden flag on the Contract / Lease screen.

The following Input Fields are required as input from your query:

Field Source Rules
contract_id fm_contracts->contract_id Mandatory
billing_period fm_contract_budgets->billing_period Mandatory

The following Derived Fields can be added to your query using the Derived Fields syntax:

Field Description
is_gst_overriden_flag Calculates GST from the net values of all budget, including an adjusting value in the largest budget, and then compares with the actual GST values. If there any discrepancies 'yes' is returned otherwise 'no'.
