Overview

Create quote templates using our standard set of quote template placeholders and HTML mark-up. The output renders into PDF for viewing or emailing.

You can create quote templates that have different designs for different channels.

An alternative approach is to deliver Web Quotes. This generates a URL link in the quote email. The quote is then rendered in a browser using the getQuote() web service in conjunction with your own web-page design.

[edit]

How to proceed

Sample Quote Template - use this sample quote template as a starting point for designing your own templates. Quote templates can be built with a portrait or landscape orientation.
We can manage the design process for you, or you can create them yourselves with your favourite web-designer.
The attached design guidelines provide detail of how to assemble and manage quote templates in Catch-e. Please read this carefully.

My html.zip quote will not print and there is no error message

Check that you have a User called 'web_quote'. The password for this user must be the same as the one set in the Global Controls record html2pdf_service_login.
