Overview

Separate Driver and Employee Portals can be setup for different Channels.
Each Portal is created with separate URL Access.
Developing a channel portal is chargable and some differences in the setup steps apply.

[edit]

Access Email

Drivers / Contact → Access - create a Driver Access Email for each channel.
Employees / Contact → Access - create an Employee Access Email for each channel.

[edit]

Forgot Username? Forgot Password?

If you want to use channel specific messages with this feature, you can add the specific text to each portal in their language files.
This text will only be invoked if the control record is blank.

Go to the Driver Portal Text for more details about these file and how to make changes. You will need assistance from Catch-e Support.

The following two controls can be customised.

  1. password_forgotten_message
  2. password_forgotten_user_id_message
[edit]
Password Reset Email Message

A separate email template can be created for each channel.
[gb_templates]channel_id must be populated.
