Overview


This feature is currently under construction.


This API allows you to create a new quote.
This performs the same function as navigating to Fleet / Administration / Quotes and creating a new quote.
This API is still being developed. A few features are not yet complete. Contact your Account Manager to discuss it's suitability for you.

Alternative API:
The createQuoteWebService API is simpler and uses less variables.

Supplementary SWAGGER documentation is available here: api.catch-e.com/docs/#/Quotes/createQuotes
All text fields must only contain characters found in the CHARSET=latin1 (ISO 8859-1) character set. Refer to wikipedia for additional details.

[edit]

Permissions

This permission and API are not available by default.
Contact your Account Manager to get access to this API.

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on Quotes:Create.

[edit]

Authentication

Authenticate with the API before running this API.

[edit]

HTTP Method

Use the HTTP Method 'POST' for consuming this web service.
