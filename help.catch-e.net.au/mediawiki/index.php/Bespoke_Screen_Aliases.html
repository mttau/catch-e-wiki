Overview

Create a Screen Aliases record to rename a bespoke tab or field.
Add an entry into the table [gb_screen_aliases] for your preferred field name.
The sample data below shows the three bespoke tabs and one field being renamed.

[edit]

Set-up Notes

1. The alias field will allow up to 40 characters, but only a maximum of 14 characters will display. Longer names would disrupt screens at smaller resolutions.
2. A field_name formatted correctly to allow it to appear on screen can be renamed using this table to appear on screen with an alternative label.
    E.g., The bespoke field_name ‘m_l_a_date_04_2012’ will appear on screen as ‘M L A Date 04 2012’ but could be renamed using [gb_screen_aliases]alias as ‘m.l.a. date 04-2012’ as shown above.

The Edit History screen displays [gb_bespoke_fields]field_name as "Field".
We recommend that you use aliases for field names only when necessary and keep them similar to the field name so users can understand the Edit History better.
