What is Catch-e?

Catch-e is a fleet management and leasing system. It is designed to service the needs of small and medium sized fleet management and leasing organisations. The system runs on a web-based platform and is hosted external to user’s locations.

The system records all cost and revenue transactions for each lease or "contract" and exports these to an external accounting system which in turn performs accounts payable, accounts receivable and general ledger functions.

Hosting is performed by a professional hosting organisation in a secure location. The hosting provider is responsible for computer hardware, operating systems, database monitoring, data backups and other services required to give the system 24x7, 99.5% minimum availability.

Users require an Internet connection and a web browser program such as Windows Explorer to access the system. Access is given over a Socket Security Layer (128bit encrypted) using a login id and encrypted password.

[edit]

System Overview

Major functions of the system are:

  • Quoting - create quotes for vehicles using stored vehicle, pricing and financing details
  • Order Management - order and manage vehicle procurement
  • Contracts - set-up and manage vehicles with over 100 fields of information
  • Authorisations – issue and manage order numbers
  • Bookings - manage short-term hire of vehicles using your internal fleet and third-party hire companies

  • Clients – record customer details, quotation, management and billing preferences
  • Drivers – record driver details
  • Suppliers – record supplier information for use in approvals, maintenance and cost transactions

  • Billing – generate lease rentals, fees, booking charges and expense recharges
  • Payments - generate bulk payment transactions for scheduled finance and insurance expenses
  • Expenses – process Registration and Compulsory Third Party Insurance charges
  • Card Services - add new cards and upload bulk card transactions (E.g. fuel, citylink)

  • Reports – a report suite provide fleet information in PDF or electronic (e.g. CSV) formats
  • Controls – create your own account codes, product types and expense descriptions to use in authorisations (e.g. standard services)
  • Export – controls export of Sales Ledger, Purchase Ledger and *General Ledger to the external accounting system (e.g. MYOB)
  • Scheduler - schedule targeted queries to help manage your business and processes

[edit]

System Architecture

The system runs in a hosted environment via Internet based connections. As a true Web-based system the user requires simply a PC or Laptop and an Internet connection to access the system. All data is stored securely at the remote hosted site. A dial up or broadband connection (preferable) can be used as the connection type.

User Access Mode

User access is via a graphical user interface using a web-browser program. Data is entered using keyboard and mouse responses. Data uploads can be performed with the assistance of the Catch-e help desk. Some data such as fuel interface files are uploaded using an interface procedure within Catch-e.
