Suspend Date DateEnter the date that the vehicle is suspended. Suspended contracts are shaded in pink on display screens. Select "Include Suspended?" if you want to see suspended contracts in the Contracts / Search tab. If the suspend date is future-dated the pink shading is not done.
  1. Billing - periodic billing stops once the suspend date is reached. You can enter a suspend date in advance, but the periodic billing will continue until the suspend date is reached. The invoice date used on the Contracts / Billing tab needs to be less than the suspend date to continue billing.
  2. Payments - entering a suspend date will stop any active payment schedules as of the suspend date. All pending records dated on or after the suspend date are updated to 'Stopped', any pending records dated before the suspend date remain as 'Pending'. If any records are 'On Hold', the 'On Hold' is removed and returned to 'Pending' or 'Stopped' as required.
  3. Approvals - users below manager level cannot create approvals for contracts once the suspend date is reached and are warned that it is suspended.
  4. Reporting - suspended contracts do not appear in reports unless the filters allow you to identify and include suspended contracts.

If the contract belongs to the Salary Packaged group, this field is read only. Make changes in the item record.

If the "Suspend Date" has been entered and "Terminated" is blank, display the suspend date in the Contracts / Header message row. E.g. Suspended 05/05/2018

TerminatedDateEnter a terminated date when all operational and accounting procedures have been completed. If the field "Suspend Date" is blank when this date is entered, it will be populated with the same date.

If the contract belongs to the Salary Packaged group, this field is read only.

If the "Terminated" date has been entered, display the date in the Contracts / Header message row. E.g. Terminated 05/05/2018.

3rd Party StartDateEnter a date here if the date the client first leased or acquired the vehicle is different to the "Contract Start" date.

Set-up Notes

Change [gb_controls]contract_internal_user_default_landing_page from 'search' to 'contract' if you want users to land on the Contracts / Contract tab when navigating into the Contracts area. The default landing page for the Contracts menu is the Contracts / Search tab.
