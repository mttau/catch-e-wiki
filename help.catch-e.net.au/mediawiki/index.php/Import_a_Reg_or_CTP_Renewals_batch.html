Import a Reg or CTP Renewals batch

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Fleet Management → Contract Actuals → Import a Reg or CTP Renewals batch

Prepare your file

  1. Use the sample spreadsheet REG_CTP_Import.csv as a basis to create your import files
  2. Create a separate import file for each supplier. I.e. one file per supplier.
  3. Populate your file with the required records.
  4. All field headers must exactly match the headers as they appear in REG_CTP_Import.csv
  5. The cost_net and cost_gst columns in the file must be formatted as number, not currency.
  6. Save the file as a CSV file. If saving in excel, make sure you... Save as type: CSV (Comma delimited) (*.csv)

Upload your file

  1. Go to Maintenance / Reg/CTP Renewals
  2. Click on
  3. Browse and attach the required file
  4. Click on Upload
  5. Once uploaded,
  6. Check that "Variance" is zero and has a green background.
    1. If it is not, check the batch total and data entry for errors.

Check your Batch

  1. Click on to verify that the uploaded registration numbers are valid and other data is valid.
The check will also run the GST Checks to make sure the "GST" amounts are within the tolerance range set in gst_tolerance.

Post your Batch

  1. Click on to post the batch.
Posting the batch will update the contract "Reg Expiry" date and "Reg BPAY Ref" fields (if "BPAY" is checked in the Batch).
If any contracts are currently being edited by other users than a warning message will be displayed.
The system must have exclusive access to the contract records in order to update the "Reg Expiry" date.
