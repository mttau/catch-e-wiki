[edit]

Live

Live is available to you 24 hours a day.
Your Live environment has scheduler jobs and email functions working.
Use the Password Reset feature in this environment if you need to reset your password.

[edit]

Support

Your Support environment is an environment where a copy of your Live data has been copied into each night.
This means it has the same login credentials as Live, only the URL to access the environment is different.

The Scheduler and the Emailing functions do not run in Support
If you are testing in Support, you will need to

  1. run any jobs you need to see working manually
  2. go to the Mail Queue to see the emails have generated.
[edit]

Staging

Staging is available 6am - 10pm EST.
Staging has it's own login credentials, but you can maintain them to be the same as Live if desired.
If you have trouble when logging into Staging, it's possible that Staging has been refreshed from Live or Support data

  1. So try your Live login details and see if they work before escalating.
[edit]

Test

Test is available 6am - 10pm EST.
Test is only available if you have a VPN Connection to Catch-e running.
