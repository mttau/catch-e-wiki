Overview

Setting up Car Profiles allows you to create links between Packaged Items and vehicle quotes and/or contracts.

A car profile enables the following

  1. A Package Item can be configured to be a car benefit
  2. Each available Contract Type is can be linked to it's appropriate profile
  3. Car benefit items can be linked to their related vehicle quote and contract

If you try to link an item to a quote or contract that is already linked to an item, you will be alerted.

[edit]

Data Setup

To view the table relationships required for set up, click here.

[edit]
Package Items

Update each Package Item that is a car benefit to record the car profile. I.e. [sp_package_items]sp_profile_table_name = ‘sp_package_item_profile_cars’

[edit]
Car Profiles

Create a record in the [sp_package_item_profile_cars] table for each kind of car profile you offer.

  1. [sp_package_item_profile_cars]package_item_id -link the profile to the appropriate Package Item.
  2. [sp_package_item_profile_cars]quote_id - 'yes' if the package item will be linked to a quote. I.e. Novated Leases, Associate Leases. 'no' if there is no internal quote. I.e. Vehicle quotes from another Novated Lease provider.
  3. [sp_package_item_profile_cars]contract_id - 'yes' if the package item will be linked to a contract. I.e. Novated Leases. 'no' if there is no vehicle contract. I.e. Associate Leases, Third Party providers.
[edit]
Contract Types

Edit each Contract Type that can is available as a car benefit and link it to the appropriate car profile.
Each Contract Type can only be linked to one car profile.

  1. Go to Contract Types / Details
  2. Edit
  3. Packaged Car Profile - select from drop list
  4. Save
[edit]

Process

  1. When a car benefit item is created or edited, the Car Details section is shown.
  2. The user can be offered
    1. A quote selection field only (suits Associate Leases)
    2. Both quote and contract selection fields (suits Novated Leases)
    3. No selections (suits Third Party Novated Leases)
[edit]

Quote Selection logic

  1. Quotes can only be selected if
    1. Employees / Contact "Client" → Quotes / Contact "Client"
    2. Employees / Contact "Pay Cycle" → Quotes / Contact "Pay Cycle"
    3. Contract Type → car profile → Package Item
    4. Quote has not already be stored against another Item record.
  2. If a quote is selected and it is linked to a contract, the contract field is updated
[edit]

Contract Selection logic

  1. Contracts can only be selected if
    1. Employees / Contact "Client" → Quotes / Contact "Client"
    2. Employees / Contact "Pay Cycle" → Quotes / Contact "Pay Cycle"
    3. Contract Type → car profile → Package Item
    4. Contract has not already be stored against another Item record.
  2. If a contract is selected and it is linked to a quote, the quote field is updated
[edit]

Vehicle Details

  1. If a contract is linked, 'Contract Status (Reg Number)' is displayed and is a hyperlink to the Contracts / Billing tab.
  2. If a quote is linked, 'Quote Status' is displayed and is a hyperlink to the Quotes / Contract tab.
[edit]

Valid Setup Links

Quote ID
The search window for Quote IDs will only display Quotes that are,

  1. Linked to the item's driver
  2. Linked to the driver's client
  3. Linked to a profile with a linked Contract Type
  4. The Quote Status is,
    1. In the 'Active' list, or
    2. 'Accepted'
  5. Not already linked to another Item


Contract ID
The search window for Contract IDs will only display Quotes that are,

  1. Linked to the item's driver
  2. Linked to the driver's client
  3. Linked to a profile with a linked Contract Type
  4. Not already linked to another Item
