Overview

The information below details the steps required to build or modify a quote template using our recommended method and settings.

  • Download and install Visual Studio Code.
  • Install Live Preview Extension.
  • Download and open the quote template.
  • View available placeholders with the value.


[edit]

Build or Modify Quote template

1. You can download Visual Studio Code from https://code.visualstudio.com/Download and install it.
2. You'd need to install the 'Live Preview' extension to render the html. Search for it in the extensions section. Make sure to install the one by Microsoft.

3. Now that you've setup the tools, you can download and open the template. Navigate to the 'Setup / Import or Export Files' page in Catch-e and export the quote_novate.zip file, selecting the 'Catch-e' Platform and 'Quote Template' File type from the respective drop-down lists. Extract the files into a folder. Within Visual Studio, select the 'File / Open Folder...' option and select the folder you've extracted the files to.

4. You can click on the html file to view the code. To preview it, right click on the file and select 'Show Preview'
5. The file should render. You will need to add ?quote_id=<quote_id> to the end of the URL to show the quote.

6. You can access the Dev Tools pane by using the menu option highlighted above, in visual studio. In the Dev Tools pane, you can expand the ‘object’ to see all the placeholders available with the value relevant to the current quote. You can use ctrl + F to search for a value or placeholder you are looking for.

7. Note that you may need to further expand the list of placeholder by clicking the three dots '(...)', at the bottom of the list.
