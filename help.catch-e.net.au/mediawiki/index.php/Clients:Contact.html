Overview

The client contact screen is used to record the name, address and contact details of the client.
This screen also allocates the default Account Manager and Sales Manager.
If the client is part of a Client Group it can be linked to the group on this screen.

[edit]

Header Buttons

Button Description
Click to create a new client. Three fields in the Clients / Contact tab are mandatory. These are: "Client Code", "A/C Manager" and "Sales".
Click to copy an existing client record. All existing record details except for the "Client Code" are copied to the new client record.

Set-up Notes

The system contains the following tables which are not yet editable on the client screens but can be access using import/export features and report writers such as the MySQL Query Browser:

  • fm_client_industries
  • fm_credit_categories
  • fm_supplier_client_credit_rating

Contact Catch-e if you wish to use these tables.

Note: the Header section "Average Fees" field is an approximate average monthly management fee net of GST.
