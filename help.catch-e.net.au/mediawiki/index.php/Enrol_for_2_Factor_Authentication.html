Main Page → Workflow → Login → Enrol for 2 Factor Authentication

Qualifiers

  1. Your organisation uses the 2 Factor Authentication function
  2. You want to enrol for 2 Factor Authentication

Process

  1. Go to System / Change Password
  2. Navigate to Users / 2 Factor Authentication
  3. Click on
  4. Method - select 'Auth App'.
  5. Auth App
    1. The screen will display a unique QR barcode.
    2. Download or use an existing 2 factor authentication application on your phone ( such as Authy )
    3. Use this app to scan the QR code or enter the unique key displayed directly below the QR code
    4. You app will use the QR code to identify Catch-e and produce a distinct 6 digit code every 30 seconds.
    5. Enter two successive codes to verify
    6. Click on
    7. You are now successfully enrolled.
