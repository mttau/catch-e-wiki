Overview

Use Channel Groups to create a group of channels. The Group Channel Manager role can access Clients, Drivers, Quotes, Events and selected reports for all of the channels in the group.

The Channels / Contact tab stores information about the selected channel group.

Visit the Channels page for all the details about using channels.

Channel Group CodeText Create a channel code. We suggest limiting this to a maximum 6 characters, so that it presents nicely on reports. The code cannot be blank or constructed entirely from spaces.
Enter this code in the Channels / Contract tab to add the channel to this group. Only active channel groups can be used.
System access is filtered by this code for Group Channel Manager users.
