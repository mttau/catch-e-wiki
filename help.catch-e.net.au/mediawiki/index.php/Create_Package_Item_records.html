Qualifiers

  1. Perform this set-up if you are providing Salary Packaging services.
  2. You have decided which benefits to offer in the table [sp_package_items].
[edit]

Download current details

  1. Go to Setup / Export Table Data
  2. Module - select 'sp | Salary Packaging'
  3. Table - select 'sp_package_items'
  4. Output Options
    1. Include Field Headers - leave checked
    2. 5 Sample Rows Only - leave unchecked
    3. Remove Null Values - leave unchecked
  5. Click
[edit]

Update details

  1. Save the downloaded file
  2. Review and add any missing items
  3. Save the revised file
[edit]

Upload revised details

  1. Go to Setup / Import Table Data
  2. Module - select 'sp | Salary Packaging'
  3. Table - select 'sp_package_items'
  4. Data File
  5. Click Upload
  6. Click
