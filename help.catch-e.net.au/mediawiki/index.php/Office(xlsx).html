[edit]

Overview

Office(xlsx) is the default value for the field "Attachment Format" for the following scheduler job Methods.

  • executeQuery
  • mailQuery
  • interfaceQuery
  • interfaceQueryIncrementatl


[edit]

Benefits

  • Files can be opened on smart devices such as phone or tablet.
  • Files sizes are smaller than the equivalent data stored in an Office(xls) file.
  • Row and column limits are larger than Office(xls)
Office(xlsx) row limit is 1,048,576 (220) and column limit is 16,384 (214)
Office(xls) row limit is 65,536 (216) and column limit is 256 columns (28)
[edit]

Presentation

Office(xlsx) files are created by using Openoffice to run a document conversion process. Because of this, you cannot customise the file presentation.
Use 'Office(xls)' if you want Customised Spreadsheet Formats.

[edit]

Embedded Formulae

The technique of using Embedded Formulae also works in Office(xlsx) files.

You can embed formulae in the spreadsheet using special markup in your SQL.

For example, the following SQL contains a formula that multiplies the value in column A with the value in column B: 

SELECT finance_pmt_regular AS 'Payment', term AS 'Term', '[formula]=A#row_no# * B#row_no#' AS 'Total Payments'
FROM fm_contracts AS t1
WHERE contract_id = '100001'

Notice the use of the marker [formula]= , this tells the processor that the code following the equals sign is an Excel formula.

The placeholder #row_no# is replaced with the current row number transforming, 

[formula]=A#row_no# * B#row_no#

to 

= A6 * B6

It would be possible for you to include values from 'fixed' cells using absoluted cell references. These cells would have to be in the header section of the report.

[edit]

Output Change

If an Office(xlsx) or Office(xls) report is generated from one of the locations below, and contain more than 65,500 individual rows, then it will be automatically converted to Text(CSV) format.

  1. Fleet / Administration / Reports
  2. Reporting / Standard Reports or
  3. Scheduler / Job executeQuery


[edit]

Set-Up Notes

Each role that will run scheduler jobs that use Office(xlsx) must have a [gb_role_api_permissions] record for the permission 'DocumentConversion'.
If you would like to convert existing jobs that use Office(xls) to Office(xlsx) in bulk, contact Catch-e Support who can do this for you.
