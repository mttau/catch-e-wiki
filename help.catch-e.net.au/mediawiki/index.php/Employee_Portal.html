Overview

The portal is set-up separately of the main Catch-e system depending on your organisation's needs.
It's known as the Driver Portal because the first features were tailored for use by the Driver role, but it now contains features for the Package Employee role also.
The menu items described here are for the Employee Portal. Visit the Driver Portal page for that list of menu items.

If the user is an employee, they are nagivated to the Employee Portal when they login.

Employee Portal Summary

Screen NameDescription
Login
Portal Login Opening page where driver logs in.
Forgot Username? Forgot Password?Drivers can reset their password if forgotten, or receive a message if they forgot their username.
Home
View my salary packageDisplays the employees packaged benefits.
View my salary package balance Displays packaging transactions and a running balance.
All recorded pending, posted and history-only transactions are shown by default.

The screen can be configured to show transactions occurring on or after a nominated date.

Raise a claim Employees can submit claims here for costs incurred that can be reimbursed.

Employees are presented Claims can only be raised if the employee has a linked contract. If using a phone, the employee can use the phone camera to take a photo of the receipt.
If the employee is using an iPhone with an IOS 13+, they may need to update their settings for this to work.

Employee contributionEmployees can can be redirected to make an employee contributions here instead of making claims if there is a shortfall. This menu item is only visible if the employee has a linked contract and employee contributions are required.
View submitted claimsDisplays claims submitted by the driver and shows their status. Attached receipts can be downloaded. This menu item is only visible if the employee has a linked contract.
View my balanceEmployees can check the current balance of their vehicle contracts. This menu item is only visible if the employee has a linked contract.
Sidebar / System
Send an enquiryEmployees can email enquiries to you here. These emails can be stored as Contracts / Events for your staff to track and manage.
Sidebar / Vehicles
Vehicle Details This option will appear in the sidebar menu when the global control ‘driver_portal_view_vehicle_details’ = ‘show’.

If enabled, drivers can view their vehicle details. This screen is read-only. The sidebar menu is only available when the global control ‘driver_portal_sidebar_enabled_flag’ = ‘yes’.

Finance Details This option will appear in the sidebar menu when the global control ‘driver_portal_view_finance_details’ = ‘show’.

If enabled, drivers can view their lease’s finance details. This screen is read-only. The sidebar menu is only available when the global control ‘driver_portal_sidebar_enabled_flag’ = ‘yes’.

Sidebar / User
Contact Details Employees can view and update their Contact Details.

When logging in for the first time, users are asked to update and/or confirm their current details.

Employment Details Employees can view and update their Employment Details.

When logging in for the first time, users are asked to update and/or confirm their current details.

Change PasswordEmployees can change their password here.
If an employee is logging in for the first time, or their password has been reset, they will be navigated to this screen to change their temporary password.
If you want to set-up access via single signon, this button can be hidden.
2 Factor Authentication You can make 2 factor authentication available to employees and it can be set to be either optional or mandatory. By default it is turned off.
Logout
