Overview

If a variant hasn't been selected yet, but you know the variant code you want to use in the quote, use this pop-up screen to enter or paste the code into the "Variant Code" field.
The variant's details will be displayed below for confirmation. Click on Save to use the code in the quote.
If the variant is not the one you want, Cancel to return you to the Quotes / Vehicle tab.

When a make and model is recorded in the quote, the lookup icon is still available when you're editing, but it takes you to the Vehicle Filter (Advanced) pop-up.
