[edit]

Overview

Allows you to retrieve a list of configured suppliers. This method requires the Suppliers:Get permission to be associated with your role.
By default only 'admin' role is linked in [gb_role_api_permissions].

[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

Input Fields (Header)

  • Header should contain the 'Content-Type' Key with the Value = 'application/json' or 'application/hjson'.
[edit]

Input Fields (Body)

JSON FieldFormatNotesMandatory
namestring Optional supplier name. This field supports the wildcard % operator. No
external_codestring Optional supplier name. This field supports the wildcard % operator. No
supplier_codestring Optional supplier name. This field supports the wildcard % operator. No
supplier_typestring Optional supplier type.
Available values : dealership, service-centre, card-services, tyres, glass-repairs, financier, transport, detailing, disposal, body-repairs, artwork, insurer, broker, residual-insurer, maintenance-controller, rental, roadside, salary-packaging 		No
exclude_empty_name_flagstring Optional flag. Set 'yes' to exclude suppliers with an empty name field.
Available values : yes, no. Default value : no 		No
statusstring Optional status.
Available values : active, inactive. Default value : active 		No
interface_typestring Optional interface type.
Available values : card-services, maintenance-controller, claim, contract-events 		No
pagestring Optional page number. Defaults to 1No
page_sizestringOptional Page size. Defaults to 25 records per page.No
[edit]

Input URL Examples

For a list of Asset Types: 

https://api.catch-e.com/fm/suppliers?page=1&page_size=5
