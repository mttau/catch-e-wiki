[edit]

Overview

Allows you to retrieve a list of Finance Liability Financiers for use in the createDriverLiabilities() and updateDriverLiabilities() web services.

[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

Input URL Examples

For a list of Liability Types: 

https://api.test.catch-e.com/fm/finance/liability/financiers
