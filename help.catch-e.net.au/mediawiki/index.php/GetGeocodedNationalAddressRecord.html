[edit]

Overview

Allows you to get an address detail record from the Geoscape Geocoded National Address File (G-NAF) for the stored PID.
Supplementary SWAGGER documentation is available here: https://api.catch-e.com/docs/#/G-NAF/getGeocodedNationalAddressRecord

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on GNAF:GetAddressRecord

This API is not configured for external use but can be made available on request.
Contact Catch-e Support to request this update.


[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

URL Examples

https://api.catch-e.com/gnaf/{address_detail_pid}
[edit]

Parameters - Path

Key Format Notes Default Value Mandatory
address_detail_pid string Pass the address_detail_pid for the required address. yes
