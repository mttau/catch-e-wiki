Field Descriptions

Field Type Description [table]field
Column One Fields
Supplier Lookup Enter or lookup the "Supplier Code" of the vehicle supplier
DescriptionText (40)Vehicle description.

When in Edit mode, a lookup icon enables you to select a variant from an Advanced Vehicle Search. Search fields include Make, Model, Model Year, Body Type and Variant (much like the Quotes / Vehicle tab). If saved, the selected variant_id and its associated data populate into the contract record.
The variant_id is useful for functions in the Approvals area and is also required to generate a Certificate of Currency.

Description Lookup Click on this lookup to use the Advanced Vehicle Search pop-up to find and pick a variant to populate or refresh the contract's vehicle details from.
Model CodeText Model code identifier
ClassificationText Asset classification. this field is included on the Text(CSV) version of the inventory report
ColourTextEnter the vehicles exterior colour. If required, you can enter up to three preferred colours using the by colour swatch icon to launch the pop-up when in Edit mode.

In the pop-up, you can enter up the colour preferences along with an additional note. The first preference, "Colour Preference 1" is displayed in the field "Colour" when saved. To view the other fields, hover your mouse over the "Colour" field. If these preferences are entered on a quote, they will copy into the linked contract on acceptance. The contract values are not refreshed if a quote is settled. All four colour fields are available to use in the Dealer Order.

Reg Managed / Expiry Date When you receive a vehicle's registration details, enter the expiry date here.
It is also updated when a Reg / CTP Renewal is processed for the vehicle, or when a Driver Portal claim is entered for registration (the Expiry Date must be entered).
If registration renewals are processed in approvals or claims, this date will need to be updated separately.

The registration date of HO (History Only) vehicles can be edited by fleet managers.

The 'State' field on the Contract screen is used to identify the state location of the vehicle.

Access to this field can be role restricted.

VINText (20)Vehicle Identification Number (17 characters typically but not for certain equipment and vehicles pre 1980). Motor vehicles (and towed vehicles, motorcycles, scooters and mopeds) VIN's (since 1981) do not include the letters I (i), O (o), or Q (q) (to avoid confusion with numerals 1 and 0). The user is warned if the VIN is not 17 characters long but the record can still be saved. Where two contracts share the same VIN Number, their Maintenance History will be shared.[fm_contracts]vin
