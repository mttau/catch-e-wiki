Overview

Use the web address provided to access the system.
You can bookmark the URL or create desktop shortcuts.
The URL will take you to the login screen of the system where you enter valid credentials.

[edit]

Setting up shortcuts

You can set up a working icon on your desktop using the following steps:

  • Right click on your desktop and select New → Shortcut to get the 'Create Shortcut' dialog box
  • Enter the URL similar to this: 
https://demo.catch-e.net.au
  • Type in a suitable name e.g. Catch-e PROD, a new icon should appear on the desktop
  • Click OK and that should be it
[edit]

Environments

Store a shortcut or bookmark for each environment that you use.
Visit the Environments page for a detailed description of each one.
For most users, Live, Support and Staging are useful.
Each enviroment has it's own url. The url pattern is shown below:

Live: https://CLIENTNAME.catch-e.net.au/
Support: https://CLIENTNAME.support.catch-e.net.au/
Staging: https://CLIENTNAME.staging.catch-e.net.au/
[edit]

Setting up a second shortcut

You can set-up a second shortcut that uses your second domain name to run two sessions of Catch-e on your computer.
Catch-e prevents the use of the right click on hyperlinks item "Open in new window" to prevent session cross-over problems.

  1. Copy your existing shortcut
  2. Rename the new shortcut (something different to the first)
  3. Edit the target and insert a "2" at the end of your domain name. Our example below is for the domain "demo". 
https://demo2.catch-e.net.au
[edit]

Add in your login

To customise your login, you can add your user_login. This will appear on the login screen.

"https://demo2.test.catch-e.net.au?user_login=myuserlogin
[edit]

Configure your screen resolution

When Catch-e is launched from a shortcut or from your browser address bar, it is usual to let your browser use your default screen resolution i.e. desktop resolution. It can also be forced to your desirable screen resolution. If the system detects you are using an unsupported screen height then it will automatically force the resolution down to the closest supported value.

In this case, define a supported screen resolution by adding the required dimensions to the shortcut or address by specifying the width and height as an additional parameter. In the following example, we are defining a 1024x768 screen resolution. 

https://demo2.catch-e.net.au?user_login=myuserlogin&screen_width=1024&screen_height=768

Go to Supported Screen Resolutions to see a list of supported resolutions that are suitable to apply.

Logging On

The login page is the same for all users.
It has links to your Terms of Use & Privacy Policy pages and there is access to a summarised version of the online user manual by clicking on the icon.
If configured, you can reset your password here by using the Reset Password link and you can contact your System Administrator by clicking on the icon.
