Overview

This report is intended to alert users to any contracts where there may be processing issues to investigate or correct.
This report is a companion to Employees / Packages 'Contract Transactions'.

  1. Items linked to benefit contracts are selected.
  2. Records are only shown if the calculation logic detailed below does not pass.
[edit]

Actions

Review the report daily and investigate records that appear on the report.
Go to Employees / Packages and print the 'Contract Transactions' report.
Use this report to identify and fix any issues you have found.
Contact your Account Manager if you need advice on how to fix an issue.
