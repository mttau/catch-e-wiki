Overview

This API returns details for a single driver_event_id.
Supplementary SWAGGER documentation is available here: https://api.catch-e.com/docs/#/Drivers%20%2F%20Events/getDriverEvent

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on DriverEvents:Get.

This API is not configured for external use but can be made available on request.
Contact Catch-e Support to request this update.


[edit]

Authentication

Authenticate with the API before running this API.

[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

URL Examples

https://api.test.catch-e.com/fm/driver-events/{driver_event_id}
[edit]

Path Variables

Key Format Notes Mandatory
driver_event_id string driver Event ID. Required
