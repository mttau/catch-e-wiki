Overview

This interface enables a finance application from Catch-e to Maclease using the Maclease Partner Interface.

This page summarises the various flags and tables within Catch-e that need to be set-up and populated with data to enable the Maclease Partner Interface. Contact your Account Manager to help you implement this interface.

[edit]

Maclease system "Plan" setting

When setting up an 'account' with Maclease, please ensure that your selected Rate Plan is the 'Default' plan. This will ensure that you applications can be submitted to Maclease via Catch-e.
If you have been set up under a different Rate Plan you must make applications to Maclease directly as they will fail if you apply via Catch-e.

[edit]

Supplier Set Up

You need to create at least one supplier record for Macquarie Leasing. Set the "Interface Type" stored on the Suppliers / Account tab to 'maclease_partner'.

You need to be set up as an 'Introducer' with Macquarie Leasing and be provided with a specific login id and token to be able to access the Maclease system via Catch-e.

For existing Macquarie Leasing suppliers, the [fm_suppliers]interface_type will be automatically updated to 'maclease_partner' when this interface is implemented. Contact your Account Manager if you have any questions.

Financier Default Values

Go to Setup / Reference Data and select the "Table" qt_financier_posting_map_defaults. You can edit the "Fee" value for selected Contract / Financier combinations including Maclease.

FieldDescriptionAllowed Value
qt_financier_posting_map_defaults
feeAdds Monthly Administration Fee to the total monthly finance payment GST exclusive Value


Other values can be set as defaults, however, if a monthly administration fee applies, both Catch-e and Maclease Monthly Finance Payment values must be calculated equally for the application to be successful. A zero value can be passed.


[edit]

Finance Types

The table [qt_finance_types] stores the available finance types you can choose in the Quotes / Finance "Finance Type" field list.
The lists are set by financier, so you can create a different list for each financier that you use.

To create finance type list for other financiers, export the table [qt_finance_types] and add the new records you need.
Remember to test this in your Staging environment first!
See also a Sample import file.
