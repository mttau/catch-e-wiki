Overview

Check to append the file creation timestamp to the file name to make it unique.
Add the #timestamp# placeholder to your file name to control the location of the timestamp.
Eg. The following file names would yield the following results;

Note: The #timestamp# placeholder will only generate if "File Name Timestamp" is checked

[edit]

File Name Examples

File Name File Name Timestamp Filename
File Name Timestamp is un-checked (best for over-writing files)
inventoryinventory
inventory.xlsx inventory.xlsx
File Name Timestamp is un-checked and #timestamp# placeholder is used (not recommended)
#timestamp#_inventory.xlsx _inventory.xlsx
#date#_inventory.xlsx _inventory.xlsx
#time#_inventory.xlsx _inventory.xlsx
File Name Timestamp is checked (not recommended)
inventory inventory.20201016112526
inventory.xlsx inventory.xlsx.20201016112845
File Name Timestamp is checked and #timestamp# placeholder is used (best for unique files)
#timestamp#_inventory.xlsx 20201016112943_inventory.xlsx
inventory_#timestamp#.xlsx inventory_20201016112917.xlsx
inventory_#date#.xlsx inventory_20201016.xlsx
inventory_#time#.xlsx inventory_112526.xlsx
