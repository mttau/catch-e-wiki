[edit]

Overview

Allows you to retrieve Supplier details. This method requires the Suppliers:Get permission to be associated with your role.
By default only 'admin' role is linked in [gb_role_api_permissions].

[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

Input Fields (Header)

  • Header should contain the 'Content-Type' Key with the Value = 'application/json' or 'application/hjson'.
[edit]

Input Fields (Path)

JSON FieldFormatNotesMandatory
supplierIdIntSupplier IDYes
[edit]

Input URL Examples

For supplier details 

https://api.catch-e.com/fm/suppliers/:supplierId
