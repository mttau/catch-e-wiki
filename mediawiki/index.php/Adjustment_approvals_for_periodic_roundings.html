Adjustment approvals for periodic roundings

Main Page → Workflow → Salary Packaging → Adjustments → Package adjustments | Adjustment claims | Adjustment approvals | Adjustment approvals for periodic roundings

Qualifiers

  1. You need to create an adjustment to zero out a small rounding
    1. E.g. a Novated Lease is disbursed monthly and does not match the fortnightly or weekly payroll deductions
  2. The employee has an Employees / Item record where
    1. Disbursement Type is 'Periodic'
    2. Contract ID is populated
  3. A Vmrs code exists for WO - End of term roundings written off
    1. Go to Fleet / Controls / Maintenance / VMRS Codes and create a record for this if you don't have one.


Process

  1. Navigate to Fleet / Administration / Approvals → Approval
  2. Contract ID - enter this
  3. Click on
  4. Supplier- enter or select a suitable supplier (the Employer's supplier account or your Internal supplier account)
  5. Reference - enter a reference. It's not important.
  6. Reference Date - select the date that the transaction belongs to
  7. Row # 1
    1. Class - select the appropriate option from the list
    2. VMRS - select or enter 'WO'
    3. Description - End of term roundings written off (the description of the VMRS 'WO' populates in)
  8. Price - enter the total excluding GST (if used). This can be a positive or negative number.
  9. Cost Price - enter the same total excluding GST (if used). This can be a positive or negative number.
  10. GST - Set to 0.00
  11. Click on
  12. Click on
  13. Click on


Next Steps

  1. The transaction will be added or, subtracted from the benefit item's balance
  2. The Funds Check balance should now be 0.
