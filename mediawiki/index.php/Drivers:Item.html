Overview

User can navigate to this tab by clicking though from the Employees / Current Packages “Package Item” hyperlink.
The record you clicked on will be displayed. Any item you can see here can be hyperlinked to: Active, Inactive, Suspended. You cannot see or hyperlink on deleted items..

Visit Employees / Item for the employees version of this page.

Process pages are listed below:

Add a bank account to an item
Add a BPAY Payment Ref to an item
Change the bank account on an item
Accept an item
Activate an item
Double the first billing for Associate or Novated Leases
Populating Vary Button for an Item screen
Suspend an item that has not started
Suspend an item

Status Drop List When creating a new record, this will default to ‘Quoting’. It can be updated to one of the statuses listed below.

Item

  • Quoting - this is the default status and indicates the item is being quoted.
  • Pending - use this when you expect to proceed with an item, but paperwork remains to be finished.
  • Expired - the item start date has passed and the item cannot progress with it's current data. Edit the item to refresh the dates and calculations.
  • Deleted - the item will not be proceeding. Use the button to update the item. Deleted items are no longer accessible on screen.
  • Suspended - the item will not be proceeding. The item is visible with a pink background on the Employees / Current Packages tab.

Item with a linked contract

  • Accepted - some paperwork remains to be finished. Create the package contract without dates. The Delete button is now disabled.
  • Active - make the item active and begin processing transactions. The package contract will be created or updated to match.
  • Suspended - the item has finished, or the Employee is leaving. Suspending an item will also suspend the package contract.

Item with a linked contract (that has previously been Suspended)

  • Inactive - can be used to indicate an item where the employee is on leave for some time, but is expected to take up this item again on their return. Catch-e recommends leaving items in 'Suspended' status. Suspended items can be updated to be active again if and when required.

Action Buttons

Button Description
View
Vary is shown when an item is linked to a contract that has begun billing.

Use this to change various attributes of the item including Cycles, Deductions Per Pay and the End Date.

  1. ‘Cycles’ will be re-named to ‘Remaining Cycles’ displaying the remaining ‘pending’ pay runs.
  2. ‘Start Date’ will be renamed to ‘Next Date’ and displaying the number of the remaining ‘pending’ pay runs.
  3. ‘End Date’ will be read-only.
  4. ‘Total Deductions will become a read-only field but will still re-calculate any updates mase to ‘Deduction Per Pay’.

Refer to 'Cycles' above to read how it is affected on Vary also.
Use this to change various attributes of the item including Cycles, Deductions Per Pay and the End Date.
These changes will update the contract's Lease Term, Description, Contract End, Total Billings and Remaining Periods.
If the item's Cycles or Deductions Per Pay are changed, Contracts / Lease details will also be updated to match.
If the item's Periodic Disbursement details are changed, Contracts / Payments details will also be updated to match.
Only pending billing and payment records will be updated from these changes.
If adjustments need to be made to correct past transactions, add adjustment records to the Contracts / Payments tab and the Contracts / Lease tab.

Car Details - unable to add quote or contract

A quote and/or contract record cannot be added to an item if it is already linked to another item.
