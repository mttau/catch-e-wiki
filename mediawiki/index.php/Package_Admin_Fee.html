[edit]

Fields

  1. Clients / Quote Defaults "Package Admin Fee' - the amount entered here will default into Quotes / Contract "Package Admin Fee".
  2. Quotes / Contract "Package Admin Fee" - shows the default from the client field for any new quotes created for the client. If no default exists, this field is blank but can be edited directly in the quote.
[edit]

Placeholders

Quotes / Contract "Package Admin Fee" - if there is a value here, quote placeholders will also be calculated which enable you to incorporate the fee into quote pdf details.

Open Package Admin Fee placeholders to see the calculations.

<package_admin_fee>
<package_admin_fee_gross>
<package_admin_fee_gross_per_pay_cycle>
<package_admin_fee_gst>
<package_admin_fee_gst_per_pay_cycle>
<package_admin_fee_net>
<package_admin_fee_net_per_pay_cycle>
[edit]

Other uses

This field can be repurposed for other functions.
The placeholders above are not renamed to reflect the new function, but can still be used.
For example, it can be used as a default to populate a stored value in a quote to show an order quantity for the vehicles.
This can be used to multiply the quote values by the quantity recorded in this field to arrive at a total cost for multiple vehicles.
E.g. Lease Cost p/w $299.00 * 10 vehicles = $2,990.00 p/w.

[edit]

Rename

  1. Clients / Quote Defaults "Package Admin Fee' - can be renamed.
  2. Quotes / Contract "Package Admin Fee" - can be renamed.
