Overview

The search screen displays contract records. You can also create a new contract from this screen.
When you navigate to this screen, the selection returns all contracts that do not have a populated "Suspend Date". The column "Contract ID" is sorted in descending order.
A range of fields in the header section allow you to filter down to selected contracts. Use the "Contract ID" hyperlink to navigate directly to the contract you want.

Output Options

Output Description
Text(CSV) or Office(xls)Select the desired output from the listbox and hit the button.
The output contains only the records returned from the search you have run. All the columns displayed on screen are included in the output. An extra column is added showing the Financier Code if you are an internal user.

If the report times out, then the report will be sent to the Report Queue. You will be alerted if this has happened.
For process instructions, visit Download a report from the report queue.

Set-up Notes

The default landing page for the Contracts menu is the Contracts / Search tab.
If you want users to land on the Contracts / Contract tab when navigating into the Contracts Module, change [gb_controls]contract_internal_user_default_landing_page from 'search' to 'contract' .

The default filters for this screen will run and return the data when you navigate here. This can become slow to load if you have a lot of data.
If this becomes too slow and you want to prevent this search from running until you have added your filters, change [gb_controls]contract_auto_search_flag from 'yes' to 'no',
When set to 'no', the screen remains blank until a search is run.

If you would like to hide Salary Packaged contracts from roles that only work with vehicle contracts, go to Roles / Details "Salary Packaged Contracts" and un-check this check box.
