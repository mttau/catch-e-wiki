Suspend an item - claim

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Items → Suspend an item → claim | periodic | recurring

Qualifiers

  1. An item will no longer be deducted from the Employee's pay because
    1. It has come to it's natural end
    2. The Employee has changed their packaging arrangements
    3. The Employee is leaving or has left
  2. The item's Disbursement Type is 'Claim'

Final Pay Run Date

  1. Make a note of when the final pay run is for this contract. E.g. 15/12/2021
  2. The date may be in the past
  3. It's OK if this is a date in the near future.

Update the Item

  1. Navigate to Employees / Item
  2. Copy the amount in Deduction Per Pay
  3. Click on
  4. End Date - select the date of the final pay run (noted in step above)
  5. Deduction Per Pay - paste in correct amount (it may have recalculated)
  6. Status - change to 'Suspended'
  7. Click on

If the Disbursement Type is Claim

  1. Contract ID - click on this hyperlink to get to Contracts / Maintenance
  2. Check to see if there are any pending claims
  3. App # - click on pending record to navigate to Contracts / Claim
  4. Review the claim ( use the Funds Check pop-up to review a summary). If the last pay run has already happened and this claim will not be paid
  5. Click on to remove the claim record
  6. Pending claims will be shown on the Claim Checks exception report when the contract's suspend date has passed.

If the Disbursement Type is Periodic

  1. Contract ID - click on this hyperlink
  2. Navigate to Contracts / Billing
  3. Life Budget Totals - make a note of the Net, GST and Total amounts
  4. Navigate to Contracts / Payments
  5. Click on
  6. Total Payments - edit to reduce the number of payments so that the Payments Total matches the Life Budget Total
  7. Edit and update a payment row if a part-payment is required
  8. Suspended Contracts that don't balance will be shown on the Payment Checks exception report.
