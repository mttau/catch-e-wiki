Overview

Reporting actual finance transactions can be useful if there is a difference between the billing and finance payment cycles, E.g. finance payments are deferred, or the billing cycle is fortnightly.

Use the "Report Finance Payments" checkbox to select how the vehicle's finance is to be shown in Novated Lease reports, the Budget Km Variance report and the Maintenance screen and report. The system default for this field is unchecked.
If unchecked, the reports display Finance Budget values and Maintenance transactions as LTD Actual.
If checked, Finance payments (from the Payments tab) will be used as follows;

The Maintenance screen and report will display Finance payments.
The Budget Km Variance report and some Novated Lease reports will select Finance transactions from Payments and Maintenance instead of using Budget values to calculate LTD Actual.
[edit]

Selection Details

Finance payments are used in the reports and maintenance screen if they have a "Posted" or "History Only" status and their "Due Date" is on or before the reporting date or month.
If the Budget Km Variance report checkbox "Inc. History Only?" is unchecked, it will exclude Payment and Maintenance transactions with a "History Only" status.

[edit]

Novated Lease Reports that display Payments

  • Novated Driver Budgeted
  • Novated Driver Budgeted Operating
  • Novated Driver Budgeted with Funds
[edit]

Novated Lease Reports that do not display Payments

  • Driver Fbt
  • Driver Fbt Operating
  • Novated Driver
  • Novated Driver Operating
[edit]

Termination Notes

If "Report Finance Payments?" is used, any variation in the Finance vs. Actual values will need to be accounted for in your termination entries. For this to present correctly on the Driver Report

  1. the posting class FIN will need to be present on the Contract Types / Expenses-Approvals tab
  2. the termination entry needs to be dated within the last active month of the contract.

[edit]

Set-up Notes

The system default for this field is unchecked. A different default can be set for this field for a channel. Use Channels / Defaults "Report Finance Payments" to do this.
