Each variant has two links to the table that identifies the "Vehicle Type".

1. Quotes / Vehicle Specifications "Variant ID" - hyperlinks to Setup / Reference Data → Detail to see the first "Vehicle Type".
This is the "Vehicle Type" that your data provider (RedBook or Glasses) allocated to the variant.
This is stored as [qt_variants]vehicle_type_id.

2. Quotes / Vehicle Specifications "Vehicle Type"
The vehicle type shown here determines if Luxury Car Tax, Luxury Car Charge and a GST Claimable cap should be applied to quotes created for this variant.
It is also used is Purc. Stamp Duty calculations for NSW.
This is stored as [qt_variant_profiles]vehicle_type_id.

If you don't agree with the allocation from your data provider (RedBook or Glasses), you can update the variant profile to record the one you want to use for system calculations.
