Overview

The Warrant of Fitness profile is similar to the Registration profile and is useful in New Zealand only.
Like Registration, it supports an annual uplift rate, which you can see working in the Warrant of Fitness calculation spreadsheet.

If the Warrant of Fitness is a stored profile for the Contract Type, it will default into new quotes and calculate automatically, this can be checked OFF or edited as required.
If the profile has been added to an existing Contract Type, the WOF profile will be checked OFF on existing quotes, this can be checked ON or edited as required.

Set Up

[edit]
Global Controls

The following Global Control needs to be populated if you want to use an uplift in the calculation.

ModuleNameComment
gbwarrant_of_fitness_uplift_rateA gb_control will be created to store the required uplift rate to be applied to subsequent years based on term of quote.


[edit]
Posting Classes

To use this profile 'Warrant of Fitness' must be created as a Posting Class
Go to menu Fleet / Controls / Accounts / Posting Classes / Details Tab.

To create a new posting class

1. Select 'New'
2. Enter class code 'WOF' - this must be 'WOF' to ensure that there is consistency through out quotes and reporting.
3. Enter Name 'Warrant of Fitness'*
4. Check ON the box next to 'Event View Flag?' if you wish to create contract events
5. Select Quote Profile Table 'qt_quote_warrant_of_fitness' from the list to enable the Profile to be created when quoting.

If a posting class already exists, make sure that the Quote Profile Table is set to 'qt_quote_warrant_of_fitness'.

If you want to create an event for Warrant of Fitness, but not have it link up to Quotes, simply leave the posting class screen reference 'Quote Profile Table' blank.

  • You can nominate another name if you prefer. Refer to Posting Classes for more information and what impact this has on screen field names.


[edit]
Contract Types

Now that the profile has been set up, it needs to be included into the Contract Type you will offer it under. Go to menu Fleet / Controls / Accounts / Contract Types.

1. Enter or search for Contract Type Code
2. Go to Accruals Tab
3. Select Edit
4. From the drop list select 'WOF'
5. Enter required set up details in the right of screen and save
  • Any new quotes will have the profile available. Existing quotes, will need to be re-calculated to introduce the new profile.
