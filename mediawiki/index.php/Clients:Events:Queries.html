Overview

Client events can be created by scheduler jobs and these can be configured to send emails, report or to store sent sms texts.

Use a Mail Query to send emails or sms. You can attach template attachments (e.g. PDF) and query attachments (e.g. excel) to these emails.
Use an Event Query to record an event where an email or sms is not required. You can use query attachments on these events too.

Contact your Catch-e Account Manager if you would like us to set-up a scheduler job to generate client events.

Query parameters
