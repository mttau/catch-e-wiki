Overview

Dealer Order forms are generated from the Contracts / Order screen.
Most of the form fields are self-explanatory and often have the same names as field name.
For a complete list of available fields, print the "Field List"

The Dealer Order can also be generated from the getDealerOrder API.

[edit]

Process pages

To update your Dealer Order

  1. Go to Setup / Import or Export Files and Export the current form
  2. Each form has a different set of placeholders, so print the field list to see what's available for this one.
  3. Use these details to create the layout of the document you want, save this file
  4. Go to Setup / Import or Export Files and upload your new form.
