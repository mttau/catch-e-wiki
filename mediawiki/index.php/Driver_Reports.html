Overview

This screen returns the current Life to Date Totals for the selected vehicle.

  1. When View my balance is clicked, the users record is checked to see if they have one vehicle or more
    1. If the user has more than once vehicle, they are navigated to the Choose Item screen where they can select the vehicle they want to view
    2. If the user has one vehicle, it is selected automatically
  2. The chosen or default vehicle is checked to see which report template is being used.
    1. If the vehicle contract not configured for reporting here, the user is alerted and returned to the Home screen. "Sorry, a report for this vehicle is not available."
    2. If the vehicle is configured for reporting, the user is navigated to the Item Balance screen

Validations and Alerts

Alert Comments
No report
Sorry, a report for this vehicle is not available. If the selected vehicle contract is not configured for reporting here, the user is alerted and returned to the Home screen.
URL changed
You do not have permission to view this page using the credentials you have supplied. The user will see this message if they have changed the referenced contract_id in the URL.
