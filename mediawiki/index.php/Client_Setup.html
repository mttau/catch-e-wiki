Clients / Quote Defaults

Setup / Reference Data → gb_controls → fm
ModuleNameCommentDefault Value
fm fm_clients_allow_quoting_flag_default Update the default setting for the Clients / Quote Defaults "Allow Quoting" field here to suit your business. no
fm fm_clients_quoting_denied_message Users will see an alert when Quotes / Contract "Client Code" is entered and Clients / Quote Defaults "Allow Quoting" is unchecked.
The alert text is stored here. Update the text to suit your business. 		Client not profiled for quoting. Refer to management.

Clients / Contract Defaults

Setup / Reference Data → gb_controls → fm
ModuleNameCommentDefault Value
fm fm_clients_contract_pay_cycle_from_quote_flag_default Update the default setting for Clients / Contract Defaults "Contract Pay Cycle From Quote" here to suit your business.
Also visit the Pay Cycle Setup page for more details of this feature.
If the client is linked to a channel, the setting stored in the Channels / Quote Defaults "Contract Pay Cycle From Quote" check box is used as the default setting instead of this control.
fm fm_clients_history_only_flag_default Update the default setting for Clients / Quote Defaults "History Only" here to suit your business.
The "History Only" field setting is used as the default when creating new cards for the client in the Card Services / Cards screen. 		no
fm fm_clients_payment_method_default If you have a preferred setting for Clients / Quote Defaults "Payment Method", set it here. blank
