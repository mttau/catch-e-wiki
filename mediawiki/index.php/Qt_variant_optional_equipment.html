Overview

This table links the optional equipment (using descriptions and codes from qt_optional_equipment) to individual variants.

[edit]

Filters

Filter NameTypeDescription
MakeListDefaults to --Please Select-- or you can select a Make.
ModelListDefaults to --Please Select-- or you can select a Model.
Model YearListDefaults to --All-- or you can select a Model Year.
VariantListDefaults to --All-- or you can select a Variant.
Variant ID Text Enter a Variant ID and use "Tab" on your keyboard to load the Variant's Optional Equipment
These filters are used when a minimum of a Variant ID selected or Make & Model selected.
Option Code Text Enter an Option Code and use "Tab" on your keyboard to load the Optional Equipment for the Variant/s
Supplier Code Text Enter Supplier Code and use "Tab" on your keyboard to load the Optional Equipment for the Variant/s
Status List Defaults to --All-- or you can select a Status (active or inactive).
