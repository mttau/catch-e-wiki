Overview

Use the Booking tab to enter details of a new short-term contract booking.

Booking StatusListA new booking record is created with a 'Requested' status. The status can be changed manually, but changes are also triggered by certain changes in the booking record.

  2. Entering details in the Bookings / Condition Start Details section will change the status to 'In Progress'.

Contract IDTextIf an internal vehicle is required, search and select the desired vehicle.

A vehicle can only be selected via the pop-up availability screen if it is available for the entire duration of the booking. However, sometimes you need to partially allocate a vehicle. You can over-ride the allocation process by finding the vehicle you need in the Availability tab and then entering the Contract ID directly into this field. You will be warned if entering a contract id for a vehicle under these circumstances but the booking can still be saved.

Client code must be entered first. You cannot open the Booking Driver window until a client has been chosen.

Invalid Driver ID. The selected driver record is linked to client that is not the one stored in this booking record. Make a note of the "Driver ID" and either cancel your changes or remove the driver from the booking record. Investigate the driver's record to determine if it is correct. Depending on what you find, you can either
  1. Go to the driver record and edit the linked "Client" record to match the booking's client and re-link it to the booking.
  2. Copy the driver's record to make one to be used with the booking's client and link it to the booking.
  3. Create a completely new driver record and link it to the booking.
