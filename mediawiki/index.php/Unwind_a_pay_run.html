[edit]

Qualifiers

  1. You have receipted a pay run and
  2. You have created Banking Batches (BECS and BPAY)
  3. The Employer has advised that late corrections need to be made and the pay run is not correct


[edit]

Delete the Banking Batches (BECS and BPAY)

If you have created Banking Batches, follow the steps in the processes below.
If not, go to the next step

Delete a Disbursement ABA file
Delete a Disbursement BPAY file (if created)


[edit]

Delete the unwanted Disbursement records

[edit]
Find the records

If there is only 1-2 records to remove, you may be able to locate them directly and remove them now.

[edit]
Recurring Claim records
  1. Go to Contracts / Claim - locate the claim
  2. Un-Post
  3. Delete
[edit]
Claim Claim records
  1. Go to Contracts / Claim - locate the claim
  2. Un-Post
  3. Edit if amount is incorrect.
  4. Make sure corrected claims are left as 'Active' / 'Pending Funds'
[edit]
Periodic Payment records
  1. Go to Contracts / Payments and Un-Post the payment record.
  2. Edit or modify the payment record amount if required, or leave as is if it is correct.


[edit]

Correct the Pay Run

[edit]
Unpost the Receipt
  1. Go to to Receivables / Receipts screen and find the receipt record for this pay run
  2. Un-Post this receipt
  3. Identify and un-allocate each record in the receipt that needs to be removed or modified in the pay run
[edit]
Remove or modify the target records
  1. For each target record
  2. Go to the Contracts / Lease tab
  3. Locate the record that needs to be removed or modified
  4. Un-Post the record
  5. If the record should be modified, edit the budget amounts as required
[edit]
Create a new invoice for the modified records
  1. If any of the target records need to be re-issued
  2. Navigate to the Pay Runs / Pay Run tab
  3. Select the target records and invoice
[edit]
Post the Receipt
  1. Go to to Receivables / Receipts screen and find the receipt record for this pay run
  2. Identify and allocate any modified records that should be added to the receipt
  3. Update the Target Amounts and post


[edit]

Delete the unwanted Disbursement records (2nd check)

[edit]
BECS Employee Summary
  1. Run the BECS Employee Preview or the BECS Employee Summary report to identify which records need to be updated to correct the cost side of the Pay Run
  2. Sort the report to find records where the Received Less Actuals is not 0.
  3. Sort again by Item Description to make processing easier
  4. Update claims and payments as described earlier


[edit]

Create new BECS and BPAY Batches

  1. Re-check the BECS Employee Summary
  2. If correct, create a new BECS Batch
  3. Re-issue the ABA file
