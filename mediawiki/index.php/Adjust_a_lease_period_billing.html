When should you create an adjustment

The lease period billing has been invoiced and you later find it was not the correct amount.

Qualifiers

  1. The contract has started and it's billing records have been created.
  2. The lease period you are adjusting has been posted.

Creating an adjustment

  1. Select the contract you want to adjust
  2. Go to Contracts / Lease
  3. Select the period you want to adjust
  4. Click on
  5. Enter the amounts you want in each budget row. The values can be 0, positive or negative.
  6. Update any gst amounts that are not as you would like.
  7. Click on
