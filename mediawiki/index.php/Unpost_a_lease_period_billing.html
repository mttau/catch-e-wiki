Unpost a lease period billing

Main Page → Workflow → Accounting → Invoicing → Unpost a lease period billing

Qualifiers

You want to unpost a lease period billing and the following conditions are met:

1. The billing has not been receipted
You can check by going to Contracts / FBT and printing the Receipts report).
If the billing has been receipted, you will need to update the Receipts batch and de-select the record
2. The billing has not been exported
3. There are no adjustment records attached to the period you want to unpost
If you want to unpost records in a period with adjustments, these must be done in reverse sequence and also deleted to gain access to unpost the previous record.
E.g. Unpost record 4.3, then delete it. Unpost record 4.2, then delete it. Record 4 can then be unposted.


Un-Post

  1. Note the Contract ID and Period that you want to unpost
  2. Select the Contract ID and go to the Contracts / Lease tab
  3. Period - select the record you want to un-post. (Records must be un-posted in reverse order if there is more than one, E.g. unpost 2.2, then 2)
  4. Click on
  5. Check the alert message
  6. Click on OK
  7. Repeat for each record you need to unpost


Invoice

An un-posted record will be removed from its invoice. The invoice still exists, but will update to only show remaining records.
The invoice can be re-printed from the Billing → Reprint screen and will show the revised records and totals.
If all the records of an invoice have been un-posted, the invoice is deleted and is no longer available on the Billing → Reprint screen.

Employee Transactions

If you have linked employee transaction records, these are deleted and removed from Employees / Transactions when a billing record is unposted.

Email

  1. The client's invoice email is still pending, this will send a corrected invoice
    1. Go to the Mail Queue
    2. Locate the email and launch the invoice
    3. Check the invoice shows the correct amounts
    4. Send the email
  2. If the client's invoice email has already been sent and you want to send a corrected invoice
    1. Go to Fleet / Billing / Re-print
    2. Client Code - enter the client code or
    3. Reg No - enter the registration number or
    4. Invoice No - enter the invoice number
    5. Click on Go
    6. Click on the PDF icon to launch the invoice
    7. Check the invoice shows the correct amounts
    8. Save and email to your client contact


Next Steps

  1. Once you have un-posted a record, it is returned to ‘pending’ status.
  2. It can be adjusted at this point and will re-invoice next time the billing is run.
