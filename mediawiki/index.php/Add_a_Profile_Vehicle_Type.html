Main Page → Workflow → Fleet Management → Set Up → Add a Profile Vehicle Type

Qualifiers

  1. You are doing quotes and the "Profile Vehicle Type" is missing
  2. We recommend first testing your changes here in your Staging environment


Find the "Variant ID"

  1. Find the variant id for the addition
  2. Quotes / Vehicle Specifications "Variant ID"
  3. Make a note of the "Variant ID" or copy it


Go to the Variant Profiles Screen

  1. Go to Setup / Reference Data
  2. Table - select 'qt_variant_profiles'
  3. Variant ID - enter or paste in the number and tab off the field
  4. Edit the profile, or if no record is shown create a new profile

Edit

  1. Click on
  2. Vehicle Type - use the ?Search function to find and select the right Vehicle Type.
    1. EV Exempt Vehicle Types are 'EVE', 'PHEVE' and 'LCEVE'
    2. If choosing 'PHEVE', select the record with the most recent date.
  3. Insurance Type - select from the drop list
  4. Click on

New

  1. Click on
  2. Vehicle Type - use the ?Search function to find and select the right Vehicle Type.
    1. EV Exempt Vehicle Types are 'EVE', 'PHEVE' and 'LCEVE'
    2. If choosing 'PHEVE', select the record with the most recent date.
  3. Insurance Type - select from the drop list
  4. Click on
