Overview

Use this tab to enter finance application details for a Novated Lease applicant when you are catering for package employees.

This data can be used with the Angle Interface and the Maclease Interface.
Applications are created in the Quotes / Finance tab.

When using the Angle Interface or the Maclease Interface, the Assets and Liabilities need to be entered in the Assets & Liabilities tab, not here.


Visit Drivers / Finances for the driver version of this page.

Bank Details
Bank Details are used to make reimbursement payments to drivers from the BECS / Reimbursements (ABA) screen.

A default bank account can be created by entering the "BSB Number" and "Bank Account Number" here. The saved bank account will default into new Contracts / Claim records and claims created in the Driver Portal.
If a Driver or an Employee enters a new bank account number in the Driver Portal when making a claim, that bank account will become the new default bank account.
The bank account that is used when creating a claim is stored in the claim record and are not updated when new bank accounts are created.

BSB Number Num If this field is blank, you can enter the BSB field for a new bank account record. This field must be 6 digits. The "Bank Account Number" must be entered at the same time.

If the banking details have been populated, you will not be able to make changes in this screen.
Go to the View Bank Accounts modal window to manage existing bank account records.
You can add new bank account records when you are creating new Claims.
This field is encrypted.

Stop BECS flag Check Box Check this on to temporarily prevent a driver's claims from being included inthe BECS / Reimbursements (ABA) file for payment.

There is no report or process to warn you that claims are being stopped like this, so they can be posted and appear to be complete. If you use this check box, we recommend asking your Account Manager to arrange an exception report for you so the transactions do not get overlooked.

If the "Stop BECS" flag is checked ON, Reimbursement Notification emails will not generate automatically from the Contracts / Approvals tab for reimbursements.

Only exclude reimbursement transactions when [fm_drivers]stop_becs_flag is not “yes”.

Validations and Alerts

Below are alerts that you may receive if a record cannot be saved.

Alert Comments
Save
“BSB Number” cannot be empty if “Bank Account Number” has been entered. Enter a BSB Number
“Bank Account Number” cannot be empty if “BSB Number” has been entered. Enter a Bank Account Number
“Bank Account Number” cannot contain more than 9 digits. Enter a Bank Account Number of 9 or less digits.
The BSB could not be matched to a valid financial institution The first 2 or 3 numbers of the BSB need to be the same as a stored [sp_financial_institutions]financial_institution_number.
Visit BSB Number Checker to confirm the code you are trying to use.
If the code is correct, add the required record to your [sp_financial_institutions] table.
