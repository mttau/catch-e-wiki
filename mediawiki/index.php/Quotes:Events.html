Overview

Quote Events are intended to record transactions relevant at the quote level. Event screens exist for Clients, Quotes, Contracts, Drivers, Employees and Suppliers.

You can;

  • Create, Edit, Email and Delete events.
  • Attach documents to an event.

Events can be viewed across all quotes at Events / Quotes.
Incomplete events are also shown on the Home screen in the Events KPI chart.

Sample Events

Below are some sample quote events.

Code Type Comments
PRICINGPricing ReviewYou can store and attach pricing review information.
PHONEPhone CallMake a note of a phone conversation had on a certain date.
ANGLE Angle Application If you have the Angle Interface enabled, an events will be created when an application has been made.

A pdf records the application details that were passed.
If the application failed or was only partially successful, response information of the failure points are also recorded here. Use the pdf to review and troubleshoot the application.

Successful events have the "Completed" field checked. Failed or partially successful events are left unchecked and will be shown in the Events KPI box for follow-up.

This event type cannot be created by users directly.

SETTLESettlement DocumentsRecord when documents have been received or sent regarding the settlement of the quote.
LCE Luxury Charge Event If the quote contains a "Luxury Car Charge" value, you may be required to create an event to record the client's confirmation of this before you can accept the quote.
PROCURE Procurement Fee Check If configured, a Quotes / Event will be created automatically when a quote is accepted to record the "Procurement Fee" that has been agreed to by the Dealer.
