•Where Global_Controls 2factor_auth_enabled_flag = ‘no’, the 2FA Mandatory section will remain hidden.
•Where an ‘SMS Provider’ has not been configured, the only ‘Method’ option available is ‘User Chooses’, else display ‘SMS, Auth App & User Chooses’ in drop list when screen is in Edit mode.
By default, ‘Mandatory’ check box will be ‘OFF’ and the ‘Method’ will be ‘User Chooses’.
• [gb_roles]2factor_auth_mandatory_flag = ‘no’
• [gb_roles]2factor_method = ‘NULL’
Where [gb_roles]2factor_auth_mandatory_flag = ‘yes’ this will require every ‘franchise’ user to access the Franchise Portal via the 2FA login process. They will not be provided any option via the Portal Home Page to enrol or unenrol – these buttons will be hidden.
