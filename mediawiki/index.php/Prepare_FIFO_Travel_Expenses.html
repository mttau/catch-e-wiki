Prepare FIFO Travel Expenses

Main Page → Workflow → Salary Packaging → Claims → Enter claims | Update claims | Prepare FIFO Travel Expenses

Qualifiers

  1. You have the Scheduler Jobs set up for processing claims.
  2. You have received a travel expense spreadsheet from a supplier
  3. You have a large volume of travel expenses that need to be added in bulk

Automated Process

  1. Open the template expense spreadsheet
  2. Create a copy for this upload and add in the details
    1. id - leave blank
    2. contract_id - populate with the employee's travel expenses Contract ID
    3. supplier_code - populate with the supplier code that is to be paid
    4. 'posting_class_code - populate with the required code. This will be the same code for all records.
    5. description - populate with the description that is appropriate for this expense
    6. quantity - enter the billing period you want to assign this cost to. I.e. Next Billing Period = 1, then 2. Expenses can be assigned to one of the next 5 periods.
    7. cost - populate with the gst exclusive cost for the expense
    8. gst_cost - populate with the gst cost for the expense
    9. invoice_no - populate with the reference you want to store. This will also be used in the ABA file as the reference when paid.
    10. invoice_date - populate as the date of invoice
    11. odometer_date - tba
    12. status_flag - 'pending' for all records
    13. user_login - populate with your system login
  3. When complete, save the spreadsheet with a suitable name in *.csv format.
  4. Use the Import data to a table process to upload the file
    1. Module - select _sch|Scheduler
    2. Table - select _sch_maintenance_transactions_import


Next Steps

Scheduler jobs will create the Claims
Scheduler jobs will add the value of the claims to the payroll periods for deduction from pay
