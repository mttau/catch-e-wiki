Overview

If configured, you can generate an Insurance Declaration form with the known details of the vehicle populated in it from the system. This can be provided to the driver to finish entering the required details before sending it on to the Insurer. An Insurance Declaration usually contains vehicle and driver details, details fines incurred, previous accidents and a general overview of the driver's driving history.
Use this document to make the insurance application process easier for drivers taking up an insurance product that you have provided directly in conjunction with an insurer.

[edit]

Process pages

To update Insurance Declaration

  1. Go to Setup / Import or Export Files and Export the current form
  2. Each form has a different set of placeholders, so print the field list to see what's available for this one.
  3. Use these details to create the layout of the document you want, save this file
  4. Go to Setup / Import or Export Files and upload your new form.

Set-up Notes

  1. Check to see if you can select an 'Office(xlsx)' output in the Contracts / Insurance tab.
  2. If this output is not available, ask Catch-e Support to create the form record for you.
  3. Once this is done, you will be able to see and select an 'Office(xlsx)' output in the Contracts / Insurance tab.
  4. Create the xlsx version of the document you want to use. To get started, download this sample: insurance_declaration.xlsx
  5. Work with your insurance company to create or modify a form that meets their requirements.
  6. Upload the form
