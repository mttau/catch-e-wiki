Overview

This screen is available for use in the Driver Portal and the Employee Portal and is a default option.

When selected, a user can change their password.
If you set-up single signon access, this button can be hidden.

If the user has been issued with a Temporary Password, they'll be directed here to change their password when they next login to the portal.
