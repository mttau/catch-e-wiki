[edit]

Overview

Allows you to retrieve a Drivers' Assets for use in the updateDriverAssets() web services.

[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

Input Fields (Header)

  • Header should contain the 'Content-Type' Key with the Value = 'application/json' or 'application/hjson'.
[edit]

Input Fields (Body)

JSON FieldFormatNotesMandatory
driver_idIntDriver IDYes
pagestring Optional page number. Defaults to 1No
page_sizestringOptional Page size. Defaults to 25 records per page.No
[edit]

Input URL Examples

For a list of a Drivers' Assets: 

https://api.test.catch-e.com/fm/driver/assets?driver_id={{driver_id}}&page=1&page_size=25

For a single Drivers' Assets: 

https://api.test.catch-e.com/fm/driver/assets?{{driverAssetId}}
