Bank Accounts

  1. Each company must be linked to it's own bank account record
  2. A GL Code record is required for each Bank Account. This is used in the Receipts Module.
[edit]

BECS file Details

  1. Create a becs record for each company. This is used on the BECS screens. ' [gl_bank_account_aba_mappings]process_type = 'becs'
  2. Create a payables record for the Vehicle company. This is used on the Payments / Periodic screen. [gl_bank_account_aba_mappings]process_type = 'payables'
[edit]

BPAY file Details

  1. Create a bpay record for the Salary Packaging company. This will be used on the BPAY screen. See the [gl_bank_account_bpay_mappings] table.
[edit]

Disbursements

Check and add the records required in [gb_report_queries] table to run the following reports.
Insert and update scripts available in the tasks noted in System Notes.

  1. BECS / Disbursements (ABA) → Client Balance Check
  2. BECS / Disbursements (ABA) → Company Balance Check


[edit]

Useful Queries


# Bank Accounts
SELECT * FROM gl_bank_accounts;
SELECT * FROM gl_codes WHERE account_type = 'bank';

# BECS file Details
SELECT * FROM gl_bank_account_aba_mappings;
SELECT * FROM gl_bank_account_aba_mappings WHERE company_id IS NULL;

# BPAY file Details
SELECT * FROM gl_bank_account_bpay_mappings;
SELECT * FROM gl_bank_account_bpay_mappings WHERE company_id IS NULL;

# Screen Reports - Disbursements
SELECT * FROM gb_report_queries
WHERE sub_page IN ('disbursements') 
ORDER BY sub_page, name;
