Overview

Create a channel to enable selected clients to be managed independently.
Users with a Channel Manager role can access Clients, Drivers, Quotes, Events and selected reports for their nominated channel.

Use Channel Groups to create a group of channels. Users with a Group Channel Manager role can access Clients, Drivers, Quotes, Events and selected reports for all of the channels in the nominated group.

Channels can be configured to have different branding on their invoices, reporting and email templates.
These templates would be used whenever a document is generated for a channel client or their driver's.

Clients are linked to the channel that is stored in the Clients / MLA "Channel" field.
Any drivers, quotes or contracts that are linked to these clients are part of the same channel.

Each channel is also linked to a Company.
If your channels are for access and branding purposes, you would link them to your one company record.
If your channels have been set-up to run independently, link it to matching Company record to enable separate bank accounts and accounting exports.
When quote's is accepted, the Contracts / Contact "Company" field is updated with the channel's company, which can then be used to separate invoicing, receipt batches and multi-company MYOB exports.

[edit]

Recommended Steps

  1. Go to Channels / Contact and create the new channel.
  2. Add the desired Channels / Quote Defaults for the channel.
  3. Create the channel's clients and add the Clients / MLA "Channel"
  4. Create the required Channel Manager and Group Channel Manager logins.
  5. Channel Managers can then create their own new clients. In this case, the Clients / MLA "Channel" field is read-only and is pre-populated.
[edit]

Pre-populate Clients / Quote Defaults fields

When a new client is being created and the Clients / MLA "Channel" field has been selected,
the values stored in Channels / Defaults will populate into the following Clients / Quote Defaults fields

  • Financier
  • Novated Commission Rate
  • Novated Management Fee
  • Management Fee

When a new client is being created and the Clients / MLA "Channel" field has been selected,
the values stored in Channels / Defaults will populate into the following Clients / Contract Defaults fields

  • Contract Pay Cycle From Quote
  • Finance Budget Term Deferred
  • Periods Deferred

When a new Contract is being created and the linked Clients / MLA "Channel" field has been selected,
the values stored in Channels / Defaults will populate into the following Contracts / Finance fields

  • Report Finance Payments


Note: If you add a "Channel" to an existing client, the Clients / Quote Defaults fields above are not pre-populated. You will need to check and update these fields.

[edit]

Channel Filters

Channels can be filtered on the following screens;

  • Fleet / Administration → Order Management
  • Fleet / Administration → Due Off Management
  • Fleet / Administration → Sale Management
  • Fleet / Billing → Recharge Billing
  • Fleet / Billing → Periodic Billing
  • Fleet / Billing → Combined Billing
  • Fleet / Billing → Billing Re-print
  • Fleet / Payments → Periodic
