Receipt a pay run

Main Page → Workflow → Salary Packaging → Receipts → Receipt a pay run | Post a receipted pay run

Qualifiers

  1. You have created a Pay Run invoice
  2. You have received the payroll funds in your bank account
  3. If required, you have checked and updated the Pay Run invoice to match the received funds


Process

  1. Navigate to Fleet / Receipts / Receipts → Allocate
  2. Click on New Batch
  3. Banking Date - update to match the payroll date
  4. Bank Account - select your Packaging Bank Account
  5. Target - enter the received total
  6. Navigate to Allocate / Item
  7. Target - enter the received total
  8. Client - enter the client's code
  9. The Target amount should be the same as the Due amount
    1. If the totals do not match, investigate and reconcile as required
  10. When the totals match, check the check box in the Summary header


Next Steps

You can now Create a Disbursements ABA file
