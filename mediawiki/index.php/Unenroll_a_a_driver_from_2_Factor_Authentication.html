Main Page → Workflow → Fleet Management → Drivers → Unenroll a a driver from 2 Factor Authentication

Qualifiers

  1. You are a user with an 'admin' role.
  2. An driver is enrolled for 2 Factor Authentication and
    1. Needs to re-set their authentication or
    2. Wants to opt out of using 2 Factor Authentication

Process

  1. Go to Drivers / Contact
  2. Driver ID - enter or lookup the record for the driver you want to update
  4. 2FA Enrolled - uncheck
