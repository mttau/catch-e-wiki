Overview

The Swann Interface is a legacy product that is no longer offered by Swan Insurance.
Go to Tyre and Rim Insurance for the current product information.

[edit]

Global Controls

The following Global Controls should be populated to enable scheduled services. Each swann product also requires Global Controls to be set-up, and those are detailed on each product page:

NamePurposeExample
swann_interface_enabled_flagThis turns the Swann Interface on or off yes
swann_interface_sourceThis is the file source type used by Swann catche
swann_interface_agent_numberThis is your Agent Number given to you by Swann 25511
swann_interface_commissionThis is your commission negotiated with Swann 200.00
swann_sftp_connectionEncrypted SFTP connection details
swann_sftp_folderName of SFTP folder catch-e


[edit]

Posting Classes

Each Swann product type that is being interfaced needs three posting classes populated. They are detailed on the product pages, but all follow the same pattern: SWAN***.

  • Swan Proposal Number
  • Swan Proposal Sent
  • Swan Policy Received


[edit]

Scheduled Jobs

The following Scheduler Job Types need to be created to run as required.

exportProposals()
importPolicies()
importVehicles()
