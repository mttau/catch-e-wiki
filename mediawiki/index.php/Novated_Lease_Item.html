[edit]

Process

The process steps for this type of item is as follows;

  1. Pay Runs / Pay Run - post the pay run record
  2. Receipts / Allocate - receipt and post the pay run
  3. Scheduler / Job - sp / payments / autoPostPayments runs automatically
  4. Contracts / Payments - the job will post each payment row that it can while maintaining a Current Funds Balance of 0 (or a surplus if a full payment cannot be made).
  5. BECS / Disbursements (ABA) - create a BECS batch and download the ABA file
    1. Periodic payments are shown as one record per payment record.
    2. The ABA reference shows the stored Contracts / Payments "Supplier Ref".
    3. This is usually the cars 'Registration Number'.
  6. Banking Software - import the ABA file and confirm it for payment


An item like this may have a running balance if the item's Pay Cycle and Disbursement Cycle are note the same. E.g. Fortnightly and Monthly.
When the pay runs and payments for this item are complete, no further deductions or disbursements will be made.
It can now be suspended.
