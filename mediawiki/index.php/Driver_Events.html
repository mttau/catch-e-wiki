Overview

This screen allows you to see and report on events created on the Drivers / Events or Employees / Events tab for all drivers and employees.

You can navigate directly to Events / Driver from the Event Chart if it is displayed on your Home screen.

You can filter the events in various ways for review. On entry to this screen, the display shows only incomplete events, I.e. the "Completed" check box is not selected.

[edit]

Event Descriptions

  • Documents Received
  • Documents Sent
  • Email
  • Enquiry
  • Note
  • Quote Accepted
  • Reminder
  • Telephone

These events are user definable and are contained in the table fm_driver_events.

[edit]

Field Descriptions

  • Group: Displays the "Group Code" if applicable
  • Client: Displays the "Client Code"
  • Driver: Displays the driver or employee's name - this is also a hyperlink back the driver's own Drivers / Events or Employees / Events tab.
  • Type: Full name of the event type
  • Date: Effective date of the event
  • Value: Some events require a value. E.g. FBT KM Target Warning
  • Description: Free text (unlimited size) to record a description
  • Action: Closed, Follow Up, None, Urgent
  • Due Date: effective for events with a Follow Up Action
  • Completed: You can mark the event as "Completed" by selecting the check box
  • Attachment: A paper-clip is displayed if an attachment exists for this event. View attached documents by clicking on the paper clip
  • Last Edit: Displays the timestamp and user login
  • User: Displays the user's login


  • If a filter is changed, click on the ‘GO’ button to refresh the results. Events may be printed by using the print buttons at the bottom right of the screen.
  • Attached documents can be viewed and downloaded by clicking on the paper clip icon.
  • Multiple Event Types can be selected in one search function. Select one event type by clicking on it.
[edit]

Output Options

You can print a PDF or CSV report of the events.
