Overview

Use this event to record the date the renewal label was forwarded to the driver or client contact.
You can also generate an advice letter to send to the Client Contact or Driver within this event.

[edit]

Workflow

Click and select the Class REGL - an event will be created with today's date as the event date, but this can be overwritten.
If a "Registration Label Sent Note:" has been entered in the Clients/Contract Defaults screen, the message will appear in a pop-up window.
You will be asked "Create Letter?"

If "OK", a PDF letter will be generated and stored against this event.
A pop-up window containing the letter will appear for you to print and send with the registration renewal label.
You can close this window and view (and print) the letter later by clicking on the PDF icon.
If "Cancel", the REGL event will be created without a PDF letter.

[edit]

Letter Configuration

The REGL letter can be sent to either a Client Contact or a Driver, depending on the preferences you set in in the Clients/Contract Defaults screen.
If the recipeient is a Client, their name and company address will be used in the letter. If a Driver is prefered, their name and address is used instead.
If no Driver is specified on the contract, the advice sentence stops before reference to the Driver is made.

Default Registration Letter
[edit]

Set-up Notes

The Posting Class REGL needs to be set-up as an event, I.e. the "Event?" Checkbox should be checked ON.
The default address for the letter needs to be set at Clients/Contract Defaults. If it is not set, the "Create Letter?" question will not be asked and a PDF letter will not be generated.
There are separate defaults for novated and non-novated contracts. This allows the letter to be addressed to either the driver (for a novated lease) or the fleet contact (for an operating lease).
A letterhead incorporating your logo can be set-up for this letter - Contact Catch-e: mailto:support@catch-e.com.au or phone us on +61 03 9017 3520 if you wish to do this.
