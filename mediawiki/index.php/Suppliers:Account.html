StatusList BoxActive or Inactive
Supplier Type Drop List Set the "Supplier Type" to limit supplier selections in various system screens.
Go to the Lookup with Supplier Type page for a summary of these supplier filters.
  1. 'Broker' - choose for Brokers. They can be added to the Quotes / Finance "Broker" field.
  2. 'Card Services' - choose for Fuel or Toll suppliers. They are linked to Card records and are also used in the Card Services Interface. E.g., Shell, BP, Linkt.
  3. 'Dealership' - choose for vehicle suppliers. They can be picked in the Quotes / Vehicle "Preferred Dealer" field.
  4. 'Financier' - choose for Financiers. They can be added to the Quotes / Finance "Financier" field. Visit the Financier page to find out how to link the suppliers to interest rates and finance calculation settings.
  5. 'Insurer' - choose for Comprehensive Insurers. They can be added to the Quotes / Contract "Insurer" field. Some may have linked insurance budget calculations.
  6. 'Rental' - these suppliers are used in the Bookings module when rental vehicles are being supplied.
  7. 'Roadside' - choose for Roadside Assistance suppliers. They can be selected in the Quotes / Lease Roadside Assistance profile.
  8. 'Salary Packaging'- choose for packaging suppliers. They can be selected in the Employees / Item “Supplier Type” field.

ABA Format Type Drop List 'Supplier' is the default setting.

For vehicle contract payments in the Payments (ABA) file.

  1. Leave as 'Supplier' to populate the Contracts / Payments "Supplier Ref" into the 'Lodgement Reference' field of the Payments (ABA) file.
  2. Choose 'Payment Reference' to make separate payments to subsidiary bank accounts. Only use this for finance payments being made to St George. A batch will not be able to be paid if any of the records has a 'Supplier Ref' that is not a correctly entered bank account. I.e., 9 or less digits long.

For packaged contract payments in the Disbursements (ABA) file.

  1. Leave as 'Supplier' to group payments for one supplier into one record in the Disbursements (ABA) file. This is only applicable if the Employees / Item "Payee" is also 'Supplier'. Use this for items such as Packaging Fees.
  2. For all other field combinations of Suppliers / Account "ABA Format Type" and Employees / Item "Payee", a separate record is created in the Disbursements (ABA) file for each transaction.

Useful Queries

# Screen Aliases
select * 
from gb_screen_aliases
where module = 'fm' and page = 'suppliers' and sub_page = 'account'
;
# Role Restrictions
select r.name as role, rr.* 
from gb_roles as r
left join gb_role_restrictions as rr on rr.role_id = r.role_id
where module = 'fm' and page = 'suppliers' and sub_page = 'account'
order by r.role_id
;
