Contracts Activity

The Contracts Activity chart is broken down into the two groups; Started and Ended.

This chart can be made available to Internal users only.

  • Started - Identifies contracts that have for the given period (Today, Current month to date, Last month, two months ago and all quotes created prior to two months ago). Contracts with an Order Status on_order_flag = 'no' are included in these results.
  • Ended - Identifies contracts that have for the given period (Today, Current month to date, Last month, two months ago and all quotes created prior to two months ago).

The Chart identifies contracts that are linked using the user id associated with the Account Manager allocated against the client unless the 'User' filter is activated and an alternative user is selected.

Hyperlinks will divert Internal users to a blank Contracts screen.
