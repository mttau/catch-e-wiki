Action Buttons

Button Description
View
Select a screen report to print. The button is hidden if there are no active screen reports set-up in your system.
Click to generate an export file for the selected batch.
Click to edit the "Status" of the selected record, or to add "Notes".
New
Review the proposed export counts and total and then click to create the export batch.
Cancel if you don't want to create an export batch at this time.

Validations and Alerts

Alert Comments
Export
Interface requires setup!, support has been notified!Either [gb_controls]dmz_path is not set correctly for your business or the dmz_subfolder_path is not set in the server.
Check the control setting and call or email Catch-e Support for assistance.
Nothing to export This alert is shown if the sales export has a "Control Total" of 0.00.
This can arise if you have exported an invoice batch and then unposted it in the Billing Re-print screen. I.e., the matching sales export batch is updated and becomes 0.
Error in the controls!, support has been notified! This alert is shown if there is a mis-match between the export's "Control Total" amount and the value of the underlying transactions.
This can arise if you have exported sales invoices and then unposted them from the Billing Re-print screen.
