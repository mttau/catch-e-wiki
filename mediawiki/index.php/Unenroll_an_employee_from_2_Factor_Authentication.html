Main Page → Workflow → Salary Packaging → Employee Portal → Unenroll an employee from 2 Factor Authentication

Qualifiers

  1. You are a user with an 'admin' role.
  2. An employee is enrolled for 2 Factor Authentication and
    1. Needs to re-set their authentication or
    2. Wants to opt out of using 2 Factor Authentication

Process

  1. Go to Employees / Contact
  2. Employee ID - enter or lookup the record for the employee you want to update
  4. 2FA Enrolled - uncheck
