Overview

The following table identifies validation error(s) and to how resolve them to pass a successful application.
If you receive an error not specified in the list below, contract Catch-e Support for assistance.

[edit]

Troubleshooting

  • If an error/warning pop up message is received that contains "Liability Description is required for SopLiability 'RENT'" and the 'Residential Status' = "Rent", St George are expecting the 'Landlord Name' to be completed.
  • If the Drivers "Position Start" date is within 5 years from today, the Previous Employment fields on the Drivers:Finances tab are mandatory.

Sovereign Validation errors

Validation ErrorCatch-e ScreenCatch-e Field NameResolution
General Error From Provider System If you see an error alert that contains "General Error From Provider System", this means the error or validation warning has been received from Sovereign. Please contact your St George Account Manager to find out what to do.
Provider Error Code "GEN001" The alert may indicate that the Driver's gender does not match their salutation. Eg Gender = Female whilst Salutation = Mr. Update the Gender or Salutation accordingly and re-submit.
Provider Error Code: StreetNameExceedsDrivers / Finances"Employer Address:" Address 1 and Address 2 are joined with a space separating the fields and submitted as <street-or-postal-name> in the <lor:employment-details> section of the application. The total length of this joined field cannot exceed 30 characters. Review and shorten the values here.
