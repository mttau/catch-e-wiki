Overview

The editable fields are in the left column of the screen, all other fields are for display purposes only. As the system stores each period (e.g. Month) separately, Life totals, invoiced to date and amounts to be invoiced are displayed. These totals should be used to check that contracts are set-up correctly at commencement and whenever lease variations are performed (e.g. change to a Fuel Budget).

[edit]

Checklist to ensure the correct Billing Setup of a contract

  1. Lease Term in months matches Quote/Internal Sheet
  2. Contract Start and Contract End dates are correct
  3. Contract Type is correct as per Quote/Internal Sheet
  4. Periodic Interval is set to months (defaults from client screen)
  5. Total Charges equals Lease Term in Months (this defaults)
  6. Initials is 1 (unless an initial charge is not needed)
  7. Periodics is 1 number less than the total, e.g. 35 in a 36 month lease
  8. Start date is the 1st of the following month - i.e. the date regular billing starts assuming initial is done
  9. Life Date Totals inc. GST should equal the Quote rental multiplied by the Lease Term

Lease Variations

Leases can be varied by editing the billing screen and lease screen. Periods can be added or removed and budgets can be edited in any period which has not already been billed. Adjustment entries can be inserted at the next period date. For example to insert an adjustment after the initial rental has been done the following steps can be followed (assuming a 60 month contract with initial rental of 6/4/2004).

  1. On the Billing Screen, Vary the contract. Add one to the number of Periodic Billings so that it is 60 instead of 59. Also change the 'Next Periodic' from 01/05/2004 to 06/04/2004. Save this. This will create an extra billing record for 6/4/2004.
  2. Go to the lease screen and edit the 2- 06/04/2004 period. Change the budgets so that it is the adjustment only, e.g. -10.66 for finance, -0.08 for CSD and 10.74 for Admin Fee. Save this. Note the total for this period is zero as we are only entering the adjustment amounts however an adjustment plus or minus could be done if desired.
  3. Go to the Fleet / Billing / Periodic Screen, change the date to 06/04/2004, and enter the reg no as required. It should bring up a record for zero dollars. Invoice and Post this, which will generate the journal entry required. It will generate an invoice for zero value which you do not need to print or keep.
  4. Go back to the Billing Screen, hit use Vary button again and change the 'Next Periodic' from 06/05/2004 to 01/05/2004. Save. This will change the dates for the remaining billing periods back to the 1st of every month.
  5. Go to the Lease Screen. The default period should be 3 - 01/05/2004. Edit the budgets to match the 'new' amounts as desired and save it.

Note:If you change the contract start date after billing has commenced make sure the next Periodic Date is correct as the system will re-calculate it based on the new entered date.
