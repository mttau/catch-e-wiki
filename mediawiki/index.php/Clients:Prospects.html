Prospects are a type of client as defined on the Clients Contact Screen. The prospect screen contains information related specifically to the prosepecting process. Available fields are:


Clients Prospects Screen Fields:

Field Type Description
Prospect StartDateDate prospecting commenced for this prospect
Last ContactDateLast contact date
Next ContactDateNext contact date
ProposalDateDate proposal presented
FinancialsDateDate financials obtained
Lead TimeListAnticpiated lead time in months
PriorityListHigh, Low of Medium category
ProbabilityListPercentage range of expected success
Prospect NotesTextUnlimited notes field
Potential Fleet SizeNumProspect's potential fleet size by major category
CompetitorsTextField to record competitors (60 characters)
end



Prospect Report:


This report produces a csv report of the prospect fields and the relevant contact fields. The report is linked to the user login, so that only Prospects for the salesperson logged in appear in the report. An overall report can be run using the SQL report writer.

end of screen.
