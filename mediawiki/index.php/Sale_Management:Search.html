Overview

The Sale Management search screen defaults to contracts with an Active sales status in the current and previous months (based on system date).

Sales statuses are stored in the fm_sale_statuses table and can be defined as active or inactive (e.g. 'For Sale' may be active while 'Sold' may be set as inactive as the process is complete). Contracts default to a status of 'blank' and need to be set to a relevant status for searching on this screen.
