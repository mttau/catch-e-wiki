Overview

This table contains codes and names of optional equipment items. These names are stored here with a unique ID to allow the same names to be used on various variant optional equipment within the system. User defined options can be defined here indicated by the data-source flag. The 'FBT Exempt' flag, when ticked, omits the optional equipment item from the FBT Value calculation for quotes with 'FBT Type' set to ‘statutory’
