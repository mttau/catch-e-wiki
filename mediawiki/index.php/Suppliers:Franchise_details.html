Overview

The Franchise details screen contains the detailed information particular to the Franchise. Data sections are:

  • Franchise - address, contact and rates/discounts etc. Note the 'Name' field links to the table qt_makes (i.e. vehicle make types).
  • Consumable Cost (Per Litre)
  • Other Fees & Charges
  • Maintenance & Service Options
  • Trading Hours
  • Comments

Action Buttons

Button Description
When the 'New' button is clicked, the screen will be cleared of any existing quote data, ready to receive information for a new quote.
If selected, you will be asked if you want to copy the record. If you confirm, all the information from the current franchise record will be copied into a new franchise record for the same supplier. The new new ‘copied’ record is opened in edit mode.
If selected, the record will be deleted if you confirm on an alert. Deleting sets the status_flag to 'deleted'. These records are not viewable, but details are retained in the table fm_supplier_franchises_vehicle.
Click on the 'Edit' button to edit an existing record.
Click on the 'New' button to create a new record.
When you have finished editing or creating a record, click 'Save' to save.

Validations and Alerts

Alert Comments
Field Entry
Fleet Labour Rate cannot be greater than Retail Labour RateWhen you enter and tab out of "Fleet Labour Rate" or "Retail Labour Rate", the two field values are checked. "Retail Labour Rate" must be equal or higher than "Fleet Labour Rate". Modify the amounts in these fields to conform to this rule.
