Overview

Your company may have custom reports. These are available in the following locations;

Contracts → Order | Finance | Lease | Payout | FBT
Quotes → Contract | Vehicle
Employees → Current Packages
BECS → Disbursements (ABA)
Xero → Xero PL Exports

[edit]

Process pages

To update a Screen Report

  1. Go to Setup / Import or Export Files and Export the current form
  2. Each form has a different set of placeholders, so print the field list to see what's available for this one.
  3. Use these details to create the layout of the document you want, save this file
  4. Go to Setup / Import or Export Files and upload your new form.
[edit]

Print Format

The report formats that are available when Screen Reports are used are shown below. At a minimum, 'Office(xls)' will be available, and depending on your set-up, the other formats shown below may be available.
The list is shown in their presentation order. The first choice in the list that is available to you is the default format that is populated into the drop list on screen.

[edit]

Minimum Available Print Formats

  1. Office(xls) (the standard xls settings are used when this report is run)
  2. Text(CSV)
  3. Field List (can be chosen if you are an 'admin' user)
[edit]

All Available Print Formats

If all possible outputs and formats are in use, the report format list would be as follows;

  1. Office(xlsx) as PDF
  2. Office(docx) as PDF
  3. Office(xls) as PDF
  4. Office(xlsx)
  5. Office(docx)
  6. Office(xls)
  7. Text(CSV)
  8. Field List

Validations and Alerts

Below are alerts that you may receive if a file cannot be generated.

Alert Comments
Print - Screen Report
Nothing to report!Below are some possible reasons for this alert
  1. There are no transactions or records that match the screen report query, so a report cannot be generated.
  2. The [gb_report_queries]name contains a space
  3. The [gb_report_queries]query is mal-formed
