Client Login Setup

The following steps show how to add a Client / Manager Login link to your web page. The link can be attached to a button or image as required; when clicked it opens a new window to the Catch-e login screen.


To add a link to the Catch-e login page for external users, add the following link to the appropriate page on your website: 

<a href="javascript:void(0)" onclick="javascript:login('https://clientname.catch-e.net.au/index.phpo?_screen_width=' + screen.width
 + '&_screen_height=' + screen.height, 'catche');">
[edit]

Separate Channel Login

If the channel users require a different logo and color scheme on the system screens then a different link markup is required. This can be setup on the appropriate page on their own website.
Catch-e will need to be provided with channel logo images to develop a separate colour scheme and logo for the channel.
The channel branding is governed by the login link only. This means any user that logs in using the channel login will have the channel branding applied to their screens, regardless of their role. This branding may also cache in the users browser, so a login via the main system may also remain channel branded.

 
<a href="javascript:void(0)" onclick="javascript:login('https://companyname.catch-e.net.au/bo.phpo?channel_code=channelname&_screen_width=' + screen.width
 + '&_screen_height=' + screen.height, 'catche');">

Important: Replace companyname with the name of your company and replace channelname with the name of the channel which requires separate logos and color scheme. Change the screen_width and screen_height as per the requirement.


The Client must also add the following code into the <head></head> section of the same page: 

<script type="text/javascript" language="JavaScript" src="https://clientname.catch-e.net.au/pop_window.js"></script>

Important: Replace clientname with the clients name e.g. demo
