Download a report from the report queue

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Users → Reporting → Download a report from the report queue

[edit]

Qualifiers

  1. You are running a system report and checked Report Queue or
  2. You are running a system report and it has timed out or
  3. You have received an email advising that a report is ready to download.
The email will note the Queue ID and the Report Title
[edit]

Process

  1. Go to System / Report Queue
  2. A list of your active reports will be shown
  3. Source - select the source to filter for one source
  4. Report Title - select the report title to filter for one report
  5. Queue ID - if you received an email, you can enter the queue id to filter to the exact report
  6. User - change to '--All--' or to the user who normally receives the report you want to see
    1. You can only do this if you have access to this field
  7. Click on
  8. - click on the paperclip to download the report
