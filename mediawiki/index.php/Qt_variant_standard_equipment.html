Overview

This table shows the descriptions of the Standard Equipment available for the selected Variant. These descriptions are stored here with a unique ID to allow the same names to be used on various variant standard equipment within the system.

[edit]

Filters

Field Type Description
MakeDrop List Select a make.
ModelDrop ListSelect a model.
Model YearDrop List'--All--' is the default filter. Choose a model year to narrow your search.
VariantDrop List'--All--' is the default filter. Choose a variant to see the Variant ID.
Variant ID Number You can enter a Variant ID directly to see that variant. The Variant ID will be displayed if a variant has been selected.
View Edit HistoryThe Variant ID must be entered to display the Audit History window. This will display any and what changes have been made to Variant Dealer Discount profile.
