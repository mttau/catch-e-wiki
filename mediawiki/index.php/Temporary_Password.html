Overview

This feature makes the password reset process quick and easy for administrators.
If you're not using it, we recommend that you set it up.

[edit]

Users

If enabled, a temporary password is issued to users when their login is first created and when a password reset is required.
When the user next logs in, they will be directed to the Users / Password tab to change their password.

[edit]

Drivers and Employees

The Temporary Password feature is also used in the Driver Portal and the Employee Portal.
When the user first logs in, or logs in after a password reset, they will be directed to the Change Password screen to change their password.

[edit]

Is the temporary password feature in use?

You can check to see if the feature is enabled by doing the following:

  1. Select a user and navigate to users / Details
  2. If the "Temporary Password" check box is displayed above "Password Expiry Days", then the feature is enabled.

or

  1. Select a user and navigate to Users / Password
  2. Click on Reset
  3. If you can
    1. the
    2. the Send Password button
  4. Then the temporary password function is enabled
