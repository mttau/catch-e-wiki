Field Access

By default, all users have full access to bespoke fields. You can add entries to the gb_role_restrictions table to change access to various roles to either display only or hidden.
The entries for bespoke fields is done slightly differently to system fields (they are described in the Role Restrictions page).
Below are the key entries required in the gb_role_restrictions to create a bespoke field restriction.

role_idmodulepagesub_pagetypenamerestriction_typebit_arrayconditions
[gb_roles]role_id fm clientsother field [gb_bespoke_fields]field_name 'display_only' or 'hidden' 0 blank
[gb_roles]role_id fm quotesother field [gb_bespoke_fields]field_name 'display_only' or 'hidden' 0 blank
[gb_roles]role_id fm contractsother field [gb_bespoke_fields]field_name 'display_only' or 'hidden' 0 blank
