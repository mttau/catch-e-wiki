Main Page → Workflow → Fleet Management → Quoting → FBT Exemption for Electric Cars

This page is no longer current as system changes on 27/02/2023 have removed the need for a workaround.

Qualifiers

  1. You want to quote on a car that is electric
  2. Treasury Laws Amendment (Electric Car Discount) Bill 2022 has passed
  3. The car you are quoting on qualifies for an FBT exemption
  4. Catch-e has not released the development change required for quoting exempt electric cars


Why is a workaround required?

Although you can already set a quote to be FBT exempt, the FBT Value needs to be captured for electric cars that are exempt under the new legislation.
Electric cars will have an exemption for FBT Payable, but a Reportable fringe benefit amount is still required.
The quote and it's subsequent contract must still store an FBT Value for FBT Reports to return the required Reportable fringe benefit amount .
If a quote is marked as FBT exempt when it is created (I.e. without this workaround), the FBT Value will not be stored and reports will not be correct.


Creating a quote

Create a quote with an FBT Value

  1. Enter the required quote details
  2. Quotes / Contract FBT Type - set as either 'Statutory' or 'Operating'
  3. Quotes / Vehicle FBT Value - this will calculate
  4. Quotes / Vehicle FBT Value Lock - check on to lock the amount
  5. your quote

Update the quote to make it FBT exempt

  1. Quotes / Contract FBT Type - update to 'Exempt'
  2. your quote


Editing a quote

Edit the FBT exempt quote and add back the FBT Value

  1. Quotes / Vehicle FBT Value Lock - un-check to un-lock the amount
  2. Quotes / Contract FBT Type - set as either 'Statutory' or 'Operating'
  3. Enter the required quote changes
  4. Quotes / Vehicle FBT Value - this will calculate
  5. Quotes / Vehicle FBT Value Lock - check on to lock the amount
  6. your quote

Update the quote to make it FBT exempt

  1. Quotes / Contract FBT Type - update to 'Exempt'
  2. your quote
