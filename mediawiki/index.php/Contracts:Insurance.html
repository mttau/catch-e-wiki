Overview

The Insurance screen has two main sections.
The section on the left of screen, Client Details displays client insurance details stored in the Clients / MLA tab. These details are usually applicable to operating leases where insurance is taken as blanket cover by the client.

The right section of the screen, Individual Details is used to store insurance details particular to this vehicle only. This is typically insurance is vehicle specific, such as novated leases.

Output Options

Form NameDescription
Insurance DeclarationIf configured, you can generate an Insurance Declaration form with the known details of the vehicle populated in it from the system. This can be provided to the driver to finish entering the required details before sending it on to the Insurer. An Insurance Declaration usually contains vehicle and driver details, details fines incurred, previous accidents and a general overview of the driver's driving history.

Use this document to make the insurance application process easier for drivers taking up an insurance product that you have provided directly in conjunction with an insurer.

Select the 'Office(xlsx)' output to print an Insurance Declaration for this vehicle.

If it is not configured, you will only see the 'Text(CSV)' option and perhaps the 'Field List'.

Troubleshooting

Visit the main Troubleshooting page for a list of all the available problem-solving tips.


[edit]
Fire Service Levy does not match expected value

Contracts / Insurance "Fire Service Levy" is populated from quotes when comprehensive insurance is calculated.
Check that Quotes / Contract "Delivery State" and the "Insurance Loc." state are the same.
Fire Service Levy is calculated using [qt_quotes]state.

[edit]
Stamp Duty does not match expected value

Contracts / Insurance "Stamp Duty" is populated from quotes when comprehensive insurance is calculated.
Check that the values in [qt_insurance_rates]base_rate and [qt_insurance_rates]cost are the same.
[qt_insurance_rates]base_rate is used to calculate stamp duty and [qt_insurance_rates]cost is used in other parts of the calculation.
