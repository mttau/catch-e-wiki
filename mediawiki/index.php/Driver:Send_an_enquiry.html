Overview

This screen is available for use in the Driver Portal and the Employee Portal.

If configured, a driver has a menu choice to send an email to you via the Portal. The generated email is sent to the recipient address linked to the email template through the table gb_template_recipients.
The driver selects an enquiry 'Reason' and then types the email text in a box.
If only one 'Reason' record is configured, this selection will default into the screen.
The email content can be customised, but will usually contain the message typed by the driver and their particulars.

A Driver Event can also be configured to record the email and will be created automatically when an email is sent.
Code = 'ENQUIRY', Type = 'Driver Enquiry', Date and Due Date = creation date, Description = Reason Description, Action = 'Follow up', Completed = unchecked
User = the client's account manager when the record was created.

  • (PORTAL-16925) Portal Enquiry Type default when only one active. Live 22/01/2024
  • (PORTAL-16936) 'catch-e.com.au ' is not a valid hostname for the email address. Live 10/06/2024
