Create a Reimbursements ABA file

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Disbursements → Create a Reimbursements ABA file

Qualifiers

  1. Your system has been set-up to allow direct link Novated Lease contracts
  2. Clients / Packaging Defaults Client Bank Account is un-checked
  3. Employee vehicle claims have been processed
  4. the scheduler job autoPostClaims has run

Process

  1. Go to the BECS / Reimbursement (ABA) screen
  2. Click on
    1. Bank Account - if required, select the bank account to use
    2. Click on
    3. Open reimbursement_preview.xls and review to check the details are correct
    4. Save down this file if required
  3. Click on
  4. Click on
  5. Save down the reimbursement_aba_{becs_id}.aba file
    1. Click on
    2. Save down reimbursement_{becs_id}.xls if this file is required
    3. Click on
    4. Status - update to 'exported_ok'
  6. Upload and confirm reimbursement_aba_{becs_id}.aba in your banking software
