Overview

In the Driver Portal, click on View submitted vehicle claims to get to the Vehicle Claims History screen.
In the Employee Portal, click on View submitted claims to get to the Claims History screen.

[edit]

Claims shown

Displays claims submitted by the driver if the following conditions are met

  1. Contract Types / Details "Group" is 'Novated' [1]
  2. Contracts / Contract "Reg Number" is not 'ORDER'
  3. Contracts / Contract "Contract Start" is populated
  4. Contracts / Contract "Contract End" is populated
  5. Contracts / Claim "Reimbursement" is checked
  6. Contracts / Claim contains rows at 'Pending' or 'Posted' status


[1] Claims for novated contracts remain on this screen indefinitely.

Set-up

The Claims History screen displays the claim status to users.
If the claim contains a Sub Status, you can add in an external alias to show this description instead of the Status.
This is done in the field [fm_approvals_sub_statuses]external_alias 

select * 
from fm_approvals_sub_statuses
where sub_page = 'claim'
;
