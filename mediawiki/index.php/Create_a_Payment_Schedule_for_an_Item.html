Owl → Add → Create a Payment Schedule for an Item

Replaced by Activate an item


Qualifiers

  1. The item's Disbursement Type is 'Periodic'
  2. The item's Status is 'Active'


Create a Payment Schedule

  1. In the selected Employees /Item tab, check the following:
    1. Disbursement Type is 'Periodic'
    2. All other Disbursement Details are populated
    3. If information is missing, populate before continuing
  2. Click on the Contract ID hyperlink to nagivate to Contracts / Maintenance
  3. Click on the Contracts / Payments tab
