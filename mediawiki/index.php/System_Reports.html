Overview

Send selected Fleet / Administration / Reports or Reporting / Standard Reports as an email and attachment to clients using a scheduled email job.
The standard email report is configured to be sent to "Contact 1" for all clients.

The reports can be modified to be sent to other client contact's and also the linked Account Manager and Sales Manager. You can send to multiple recipients (cc'd or bcc'd). One or multiple or all client(s) can be selected to receive these reports.

The email content can be either plain text or html.

Contact your Catch-e Account Manager if you wish to set such jobs. Please also provide below details:

  • Nominated Client Code, or business rule to identify clients who should receive the report (it can be all)
  • Nominated recipients and if To, CC or BCC
  • Email subject
  • Email body
  • Sender name
  • Sender email
  • Report frequency (Quarterly, Monthly, Weekly)

Catch-e Set-up Notes

1. Create a record in gb_templates to hold email and sender details.
2. Copy in the required query from Demo Live gb_queries (each report has a corresponding query held there.)
3. Modify the query to deliver the selected records and link it to the appropriate email template.

Note: The default link is to [gb_templates]name which should be modified as required, you can use [gb_templates]template_id or [gb_templates]sub_name if you prefer.

4. Set up the scheduled mailQuery job.

Note: Clients must UAT each job before deployment.
