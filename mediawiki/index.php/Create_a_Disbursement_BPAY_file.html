Create a Disbursement BPAY file

From Help wiki

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Disbursements → Create a Disbursement BPAY file

Qualifiers

  1. The client has a Clients / Packaging Defaults record
  2. You have receipted the pay run
  3. Since the receipt was posted,
    1. the scheduler job autoPostClaims has run
    2. the scheduler job autoBuildClaims has run
    3. the scheduler job autoPostPayments has run
  4. You have created an ABA file today


Select

  1. Navigate to Accounts / Export / BPAY → Disbursements (BPAY)
  3. ABA Type
    1. choose 'Company' if you maintain one bank account for your clients.
    2. choose 'Client' if you maintain a separate bank account for the client.
  4. Client Code
    1. leave this field blank of you process all client disbursements together in a company bank account
    2. lookup or enter the client's code, tab off this field
  5. If there are no transactions, you will be alerted, click OK to exit the screen
  6. The calculated total for the file is displayed

Preview Reports

  1. Output Name - select either the 'Preview Company' or 'Preview Client' to review the transactions that will be included in the batch.

Create the batch

  1. Transaction Date - pick or enter the bank processing date you need for this file

Transaction Report

  1. Run the transactions report for a breakdown of the bpay records for recordkeeping, or to create a banking upload file.
  2. Output Name - 'Transactions'
  4. You may also have a custom report configured for your banking upload that can be generated.
