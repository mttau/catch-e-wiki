Packaged Contracts

[edit]

Business Rules

If a contract is linked to the group 'Salary Packaged':

  1. Contracts / Contract "Contract Status", "Sub Status", "Suspend Date" and "Terminated" are all read-only. "Contract Status" and "Suspend Date" are updated from changes to the item record.
  2. Contracts / Billing "Periodic Interval" is read only. Changes can be made in the item record in certain situations.
  3. Contracts / Claim "Budget" list box displays all the classes that are stored in the Contract Types / Expenses-Approvals tab.
  4. Approvals / Search "Approval" hyperlink will always navigate to the Contracts / Claim tab.
[edit]

Set-up Options

  1. Roles / Details "Salary Packaged Contracts" - use this to control the visibility of Salary Packaged contracts in search screens.
