Overview

The Notifications Bell icon is used to alert you to scheduled system outages or deployments. Below are notes about the different states of the icon and the recommended actions you can take.

The Notifications Bell icon state is only updated when you manually refresh the page or interact with the system in a way that triggers a page refresh.
To check the icon state for the latest updates on scheduled outages and deployments, navigate to another page to trigger a page refresh.


[edit]

Notifications Bell Icon States

Icon State Description Action
  Bell Icon OnlyWhen the Notifications Bell icon is displayed without any additional indicators, it means there are no scheduled outages or deployments at this time. Clicking on the icon will display a message saying, "There are no scheduled outages at this time"
  Bell Icon with Blinking Red DotA blinking red dot on the Notifications Bell icon indicates that there are unacknowledged notifications. The blinking red dot will appear:
  • 12 hours before a scheduled Release Deployment
  • For a Hotfix Deployment

Once acknowledged, the blinking red dot will turn into a solid red dot. The icon will start blinking again 15 minutes prior to the scheduled deployment. Click on the blinking icon and note the deployment time. Save your changes before the deployment time. The icon will continue to blink until the deployment starts. Once the deployment starts the system will not be accessible.

Clicking on the icon will display a popup dialogue with details about the upcoming deployment(s), including the time and date when the system will be unavailable. The popup will have two buttons:

Acknowledge: Clicking this button will record your acknowledgment. The blinking red dot will then change to a solid red dot.
Cancel: Clicking this button will close the popup dialogue without acknowledging the notifications.

  Bell Icon with Solid Red DotA solid red dot on the Notifications Bell icon indicates that there are acknowledged notifications. Clicking on the icon will display a popup dialogue with details about the upcoming deployment, including the time and date when the system will be unavailable. The popup will contain a ‘Close’ button to dismiss it.
