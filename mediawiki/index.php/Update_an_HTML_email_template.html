Update an HTML email template

From Help wiki

Jump to: navigation, search

Main Page → Workflow → System Administration → Data Updates → Update an HTML email template

Qualifiers

  1. You are updating an existing HTML email template
  2. You know the template_id of the email you want to update
  3. We recommend first testing your import file in the Staging or Support environment


Create an Import file

  1. Create your updated HTML markup
  2. Use a generated report from the scheduler to get the template details you need.
  3. The file must be saved as csv file.
  4. The file must contain the fields below.
  5. Only the data in the 'mail_message' field should be updated

A template_id
B channel_id
C name
D sub_name
E mail_message

Add HTML data to the Import file

  1. Place your cursor in the cell E2 (the 'mail_message' field)
  2. Copy your HTML text
  3. Place you cursor in the Formula cell above the spreadsheet data
  4. Ctrl P or Paste
  5. The data will have been placed entirely tin cell E2

There are many instruction videos about how to do this online.
Search for 'how to copy and paste large text into one excel cell'

Import data to a table

  1. Go to Setup / Import Table Data
  2. Module - select 'gb | Global'
  3. Table - select 'gb_templates'
  4. Drop or browse to import the data file
  5. Click on Upload
  6. The uploaded file content will be validated.
    1. If there are problems, the Error Report button will be enabled.
    2. Download the Error Report, review, fix your file
    3. Re-Upload
  7. A file validation message is displayed
  8. Click on
  9. Navigate to the system and generate the updated email to check the data you have imported
