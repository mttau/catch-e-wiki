Card Services
Card NumberNumLists the cards linked to this contract. The card number hyperlinks to Cards / Card Details and if you hover over the number, further details are displayed. The cards are listed in the following order;

Cards with an 'active' status and no deactivated date are shown first in "Card ID" order (lowest number first).
Cards with an 'active' status that have a deactivated date are then shown in "Deactivated" order (most recent date first).
Cards with an 'inactive' status and no deactivated date are shown first in "Card ID" order (lowest number first).
Cards with an 'inactive' status that have a deactivated date are then shown in "Deactivated" order (most recent date first).

[edit]

Action Buttons

Button Description
Print the chosen report.
Click to edit the record.
When you have finished editing the record, click to save.
Saving triggers a GST check which validates the GST of the "Residual Price Net" and "Residual Value". The validation checks if the GST is within the tolerance set in gst_tolerance. If the validation fails, it gives a warning message for the particular field that needs to be corrected.
Cancel if you don't want to make your editing changes.
