Overview

In Edit mode click the colour swatch icon to launch a Colour Picker pop up window that will allow you to enter up to three colour preferences along with an additional note.
"Colour Preference 1" will be shown in Display mode (labelled "Colour").
In Display mode, to view the other colour preference fields, hover your mouse over the "Colour" field.
The data stored in these fields will be added to similar contract fields when accepting a quote.
Note 'Settle' does not update colour fields in the contract.

[edit]

RedBook Paint data

RedBook can supply an additional file containing manufacturer paint (colour) details. The column names can be seen in this workbook here.
Only the columns named VehicleKey, PaintDescription, PrimaryGenericColour & PrimaryPaintType are used in Catch-e.
Paint data imported from RedBook is stored in the Catch-e table [qt_variant_paints].

[edit]

Editing Colour fields

The table [qt_variant_paints] can be populated either manually (via Import Table Data) or from RedBook Scheduler Job.
When a matching [qt_variant_paints] record is found, the Colour & Colour Picker feature will operate as follows:

  1. Colour field is read-only.
  2. All Colour preference fields must be selected using the Colour Picker pop up window (launched by clicking the colour swatch icon).
  3. Colour Preference 1, 2 & 3 are all list boxes with their data sourced from matching [qt_variant_paints] records.

The list box will display values as shown in table below:
