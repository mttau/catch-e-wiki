Overview

This library is used to calculate emission values based on existing data.
These readings are shown on screens and used in various reports, such as the FBT Statutory Report.

This library can also be used with the following scheduler job methods: executeQuery, mailQuery and eventQuery.
This library can also be used with the following APIs: printReportQuery.

[edit]

Derived fields from the /lib/PDF/fbt_statutory library

This library is used to calculate fbt statutory placeholders.

[edit]

Input Fields

The following Input Fields need to be added to your query:

Field Source Comments Mandatory
contract_id fm_contracts → contract_id Yes
ecm_flag fm_contracts → ecm_flag Yes
date_filter Any valid date dd/mm/yyyy If not passed, the report will use the current date.
Add the field into your query or report query to use a specific date or date logic.

If you are running the Print a report query API, you can make this field a JSON variable as 'dd/mm/yyyy' or 'yyyy-mm-dd'.

No
fbt_statutory_rate_date Any valid date dd/mm/yyyy or yyyy-mm-dd This date is used to look up the transitional statutory rate.
If blank today's date will be used.
Add the field into your query or report query to use a specific date or date logic.

If you are running the Print a report query API, you can make this field a JSON variable as 'dd/mm/yyyy' or 'yyyy-mm-dd'.

No
