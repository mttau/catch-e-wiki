Overview

The Periodic invoice format can be used for billing pending lease rentals from each contract. It gets the records from the Contract Lease screen and also includes Credit or Inertia charges.

Other charges such as recharges and booking invoices will not appear in this screen or in periodic invoices generated from it.

Lease charges are typically monthly leases invoiced to clients at the beginning of each month for operating type leases and based on pay cycle for Novated Leases. Upon entering the screen, the system displays any charges which are due on or before the system date. Due dates are set-up in the billing page in each contract.

Use the filters below to create invoices for all clients at once or to do selective groups of clients.

Periodic invoices can be set-up to run automatically from the Scheduler. This is the billing process that Catch-e recommends.

If you cannot see a client's transactions on this screen, check which Clients / Contract Defaults / Billing Type has been used, only 'Periodic' or 'Any' selections will be visible here.
Packaged Contracts (contracts that are linked to an item) cannot be viewed or invoiced from this screen.

Automation

We strongly recommend scheduling your bulk billings.Visit the scheduler billings page to find out more about how to do this. Your Account Manager can also assist you in setting this up.
This will be more efficient because

  1. Scheduled billings are 'set and forget'
  2. Emails are sent out in the scheduler process and do not require a second step
  3. Billings can be set to run out of hours, which reduces the likelihood of users being disrupted by the system locks that bulk billing creates. See Note.

Note - the billing process generates system locks that prevent users from editing records in many areas. Visit the System Locks page for full details about system locks.

Channel Billing

If you have arranged for separate Invoice Templates for one or more of your Channels, before selecting any filters in the header section of this screen, you must select the invoice template from the list box at the bottom right corner of the screen. The list will contain (for example on the Periodic Billing screen) "Standard Periodic Invoice" (default value) and any Channel templates will be listed below it (eg. "CHANNEL Periodic Invoice").

Once the Channel template has been selected, the "Channel Code" filter in the header section will be read only and will display the relevant Channel Code accordingly. All other Client selection filters (Group Code, Client Code, Reg No) will now only return Billing records linked to the selected Channel once you click the Go button.
