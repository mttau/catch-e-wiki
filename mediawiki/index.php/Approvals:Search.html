Overview

The Approvals / Search screen is used to display approval records.
From this screen a new approval can be created or existing approvals can be selected for editing and further processing.
A number of filters in the header section of the screen allow the user to search for specific approvals.
The default criteria for this screen is to display requested approvals.

Filter buttons

Button Description

Column Descriptions

The column headers of this screen are dynamically sortable. By selecting the grey box header, the data will be sorted in an ascending order. If selected again, the data will be sorted in a descending order.

Column Type Description [table]field
Approval Num Displays the Approval ID. The hyperlink navigates you to either the Approvals / Approval tab, or to the Contracts / Claim tab.
  1. Approvals / Approval - for vehicle supplier payments
  2. Contracts / Claim - for driver reimbursements or for 'Salary Packaged' contracts.

Hint: Use the "Back" navigation button to return to the Approvals / Search tab from a claim.

[fm_approvals]approval_id
