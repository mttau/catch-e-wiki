Field Descriptions

Screen NameTypeDescription[table]field
Contract IDTextDisplays the contract ID. The Contract ID can be entered directly, or the search funtion can be used to find the required contract.[fm_approvals]contract_id
Reg NumberTextDisplays the Registration Number. The Registration Number can be entered directly, or the search funtion can be used to find the required approval.[fm_contracts]reg_no
VehicleTextDisplays the Vehicle Description. This is a fixed field and cannot be used as search filter.[fm_contracts]description
ClientTextDisplays the Client's name. This field hyperlinks to the Clients/Contact tab.

When in Edit mode, the icon is available and you can View Client Details in a pop-up screen.

[fm_clients]name
DriverTextDisplays the Driver's name (Last, First). This field hyperlinks to the Drivers/Contact tab. Also, when you hover your mouse over the hyperlink, in the bottom left corner of the screen, the driver's "Work Phone", "Mobile" and "External Code" will appear, if they have been entered against the Driver record.[fm_drivers]given_name,[fm_drivers]surname
TypeText-Num-NumDisplays the Contract Type, Term and Contract Kilometers of the selected contract.[gl_posting_maps]posting_map_code, [fm_contracts]term, [fm_contracts]contract_kms
StartDateDisplays the Start Date of the selected contract and the Start Kilometres.[fm_contracts]contract_start, [fm_contracts]start_odometer
EndDateDisplays the scheduled End Date of the selected contract and the contract kilometres.[fm_contracts]contract_end, [fm_contracts]contract_kms
LastDate, NumDisplays the date and odometer reading of the last valid recorded transaction.[fm_maintenance]odometer_date, [fm_maintenance]odometer
TyresNum, NumDisplays the number of tyres allowed for in the selected contract's budgets and the number used so far including approval rows from all 'pending' approvals, unless they have been recharged. Recharged tyres are not counted.

Where the number used exceeds the allocated amount, the field is highlighted green.

[fm_contracts]tyre_allowance
ProjectedNumDisplays the projected ending kilometres of the vehicle based on existing odometer readings.
DivisionTextDisplays the Division the selected contract is linked to. This is created or changed using Contract / Events.[fm_contracts]division
IconView Edit History is a pop-up screen that lists all the field changes in the selected record.
IconPerformance Indicators is a pop-up screen that displays a summary of budgets and actual costs that are useful for maintenance control.
IconFunds Check is a pop-up screen that displays a summary of budgets, actuals, receipts and available funds. This is useful for determining if reimbursements can be paid or not.
Message RowTextAt the bottom of the Header area a red message row may appear. This is populated with information based on these rules:
  1. If a "Terminated" date has been entered in the Approvals / Contract or the Contracts / Contract tab, display the terminated date. E.g. “Terminated 28/06/2018”.
  2. If "Terminated" is blank and a "Suspend Date" has been entered in the Approvals / Contract or the Contracts / Contract tab,, display the suspend date. E.g. “Suspended 16/06/2018”.
  3. If “Terminated” and “Suspend Date” fields are blank and a "Replacement Order ID" exists in the Contract / Terminate tab, display the replacement contract. E.g. “Replacement Order ######”.
  4. If “Terminated”, “Suspend Date” and "Replacement Order ID" fields are blank suspended, display the first 40 characters of the "Notes:" field. If this is blank, no message will display.
