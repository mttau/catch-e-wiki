Overview

This screen is used to display, filter and report on all contract events that have been created in the Contracts / Events tab.
You can navigate directly to Events / Contracts from the Event Chart if it is displayed on your Home screen.

  • If a filter is changed, click on the ‘GO’ button to refresh the results.
  • Attached documents can be viewed and downloaded by clicking on the paper clip icon.
  • Allow users to attach more than one file using a Zip folder.
  • Contracts that have been accepted from quotes have events created for the driver and cost centre (if recorded). The registration number is normally created when the contract is settled and started.


See also the Events Report.

 The "Value" and "Description" fields can use an asterix, or wild card character '*' to allow for partial text searches. E.g. you enter 'John*' into the field, and events for John1, John2 etc will be returned.


[edit]

Search Filter Definitions

Field Type Description
GroupTextSelect by client group.
ClientTextSelect by individual client.
Contract IDNumber Enter or select the Contract ID.
Reg NoNumber Enter or select the Registration.
From and ToDateSelect events within a date range (event date).
Value Use this to search for values such as old registration numbers.
Description Use this to search for values.
TypeListSelect the event type you want, or choose a multiple by holding down the “Ctrl” key and use your mouse to select.
ActionListSelect the action you want to see.
Due ByDateSelect the due by date. Note: records where the due by is blank will always be shown.
Incomplete onlyCheck boxCheck this off if you only want unfinished events.
UserListSelect the user who created the event.



[edit]

Output Options

Field Description
Print a PDF or Text(CSV) output of the filtered search results. All the fields displayed on the Search screen are included in the output.
