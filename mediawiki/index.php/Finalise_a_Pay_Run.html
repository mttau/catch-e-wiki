Finalise a Pay Run

From Help wiki

Jump to: navigation, search

Main Page → Owl → Pay Run Processing → Finalise a Pay Run

Qualifiers

  1. Posting Classes / Details Class code 'PRSUM' has been created
  2. Clients / Contact Payroll Name and Payroll Email are populated
  3. Clients / Quote Defaults Payroll Cut Off Days is set appropriately
  4. Clients / Other Payroll Deduction Report is set to 'Summary Only'
  5. Clients / Other Cut Off Days is set appropriately
  6. The required scheduler jobs for this process has been created and is active


Client receives the report

  1. When the scheduled jobs for this process runs and the report is due
    1. the Payroll Summary Report is generated and emailed to the client's payroll contact
    2. an email event is created with a Value of 'Pending', Completed is un-checked

Monitor unconfirmed pay run reports

  1. The Home screen KPI Box called Events contains a section for client events
  2. The pending email event created above will display in the Clients # count
  3. Click on that number bar to navigate to the Events / Clients tab.
  4. Type - select 'Payroll Summary Report'
  5. Review this list and follow-up any clients that have not confirmed their pay run deductions. The date details in the events will help guide you.
  6. To update an event, click on the Client hyperlink to navigate to the Clients / Events tab

Client needs changes to the payroll deductions

  1. When the client has advised that changes are needed
  2. Update the records by editing the contract or running the Skip process (more later)
  3. When the changes have been made, edit the client's event and update Value to 'Re-issue'
  4. The scheduler job will run and
    1. The existing email event is updated to set Completed as checked
    2. A new email event is created with a Value of 'Pending', Completed is un-checked

Client confirms the payroll deductions

  1. Navigate to the Clients / Events tab
  2. Select and edit the pending event, update Value from 'Pending' to 'Authorised'

Next Steps

You can now Bill the Pay Run
