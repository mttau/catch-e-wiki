Overview

External report writers such as Crystal Report, Microsoft Access and MySQL can be used to generate user defined reports.
To use a report writer with Catch-e system data, you will need to install a secure VPN Connection and a suitable report writer.
We will provide you with "read-only" access to the relevant databases. Steps for setting up access are as follows:

[edit]

Installing MySQL Query Browser

MySQL is an open source query browser that is used by Catch-e staff.

To install MySQL Tools for 5.0, contact your Account Manager for assistance. They will provide you with a link to suitable software.

  1. Click on the link provided
  2. Double-click to execute the file - press "OK" to bypass the warning message.
  3. Click "Run" and follow the prompts.
[edit]
Connecting to MySQL
  1. Locate and Open MySQL on your program menu
  2. Click on the “…” button to open Connections
  3. Click on "General Options" and check that the tick-box 'Store Passwords' is checked on.
  4. Click on 'Connections", then the “New Connection” button and enter the information into "Connection Parameters"
  5. Apply & Close (to return to the SQL login box)
  6. Your connection information should now be stored and will display when you connect to MySQL

Contact Catch-e support if you need connection details.

[edit]
ODBC Set-up
  1. Run the executable file MyODBC-standard-3.51.07-xp.exe which will install the driver to run on MySQL databases (file obtainable from Catch-e)
  2. Go to Control Panel / Administrative tools / Data Source (ODBC) and add a new data source using the new driver and database attributes (obtainable from Catch-e). Tick 'Use Compressed Protocol' in ‘Options’ to reduce download times (this significantly reduces data size). Use 'Test Data Source' button to test the connection.
  3. In the report writer application, select your database as the data source and enter the user login and password provided by Catch-e support.
