Overview

You can configure your data to enable particular options to be added to new quotes by default.

If you have a Quotes / Contract "Deal Type" list set-up, this feature will only include default options when the Deal Type has a matching [gb_controls]include_optional_equipment_by_deal_type' value.
You can store more than one deal type in [gb_controls]include_optional_equipment_by_deal_type', so you can set this default to work for multiple deal types.

Additionally, this feature can be limited to where "New or Used" = 'New' vehicles only.

Note: An empty Quote Deal Type and a blank [gb_controls]include_optional_equipment_by_deal_type' deal_types value is considered a match.

[edit]

Global Control Setup

[gb_controls]include_optional_equipment_by_deal_type' has two parts.

Part 1: "New Only". Default is 'no'. To restrict this feature to where "New or Used" = 'New' vehicles only, update this part to 'yes'.
Part 2: "Deal Types". To limit this feature to selected "Deal Types", add as many deal types as required.
Note: If you are setting this up for the first time, please contact Catch-e Support for assistance.

[edit]

Configuration Alternatives

If Global Control "include_optional_equipment_by_deal_type" deal_types value contains deal_types:
When a target Quotes / Contract "Deal Type" is chosen, if there are any Options with the 'include_by_default_flag' = yes, these options will be automatically added to the Quote.

If Global Control "include_optional_equipment_by_deal_type" deal_types value contains deal_types:
If a Quotes / Contract "Deal Type" is left blank, if there are any Options with the 'include_by_default_flag' = yes, these options will not be automatically added to the Quote.

If Global Control "include_optional_equipment_by_deal_type" deal_types value does not contain any deal_types:
If there are any Options with the 'include_by_default_flag' = yes (regardless of Quotes / Contract "Deal Type"), these options will be automatically added to the Quote.
