Overview

If configured, this icon appears next to the "On Road Cost" field. Clicking on this icon launches a pop-up window which contains the vehicle pricing in detail.
Amounts displayed here are initially populated from the quote when it is accepted. These amounts will also be updated when a quote is settled.
These fields can be edited and changes here will also update the amounts shown in the Contracts / Vehicle "List Price", "On Road Cost" or "Total Options" fields.

If you make changes in this window, the amounts stored in the linked quote will no longer match. Update the quote and use the Settle button if you want to maintain the same amounts in both records.


All fields on this window can be relabeled.

Set-up Notes

Below are a couple of recommended ways of configuring access to this window. Access can be set at a role level, so you can provide different settings to different roles. Contact Catch-e Support if you would like to change your settings.

[edit]
View and Edit Vehicle Price

This is the recommended setting for roles that need to update vehicle pricing in a contract.

In Contracts / Vehicle

  1. The View Vehicle Price icon is available.
  2. The fields "On Road Cost", "List Price" and "Total Options" are display-only.

In the View Vehicle Price pop-up, the Edit button is available.

[edit]
View Vehicle Price

This is the recommended setting for roles that do not need to edit vehicle pricing.

In Contracts / Vehicle

  1. The View Vehicle Price icon is available.
  2. The fields "On Road Cost", "List Price" and "Total Options" are display-only.

In the View Vehicle Price pop-up, the Edit button is hidden.

[edit]
Traditional

This configuration was applied to existing systems to preserve screen behaviours. It can be changed.

In Contracts / Vehicle

  1. The View Vehicle Price icon is hidden.
  2. The fields "On Road Cost", "List Price" and "Total Options" are editable.
