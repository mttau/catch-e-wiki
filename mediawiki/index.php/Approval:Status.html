Status Sub Status Description
Requested Requested approvals have been created by franchise users in the Merchant Portal.
The maintenance rows for approvals in this status are set as pending.
Active Created but not yet posted. If you review and save a requested claim, it will save into 'Active' status.
The maintenance rows for approvals in this status are set as pending.
Posted The claim has been finalised. It can now be exported to your accounting system.
If the maintenance row has the "R" column set to 'Y' for recharge, those rows will display as blue until the recharge invoice has been created.
Cancelled Created but cancelled without saving. Cancelled approvals are not visible on the Approvals / Maintenance tab.
Deleted Approvals can be deleted by clicking on Delete. This is only possible if the claim has not been posted.
Deleted approvals are not visible on the Approvals / Maintenance tab.

To see these approvals, go to the Approvals / Search tab and filtering on this status.
Click on the hyperlink to get to the claim. The maintenance row is shown in grey.

History Only Active approvals can be set to history only by clicking on History.
These approvals will be shown in Driver Reports but will no be added to accounting exports.
This status is available for adding legacy records, but is not recommended for general use.
Maintenance rows for this status are shown with a grey background.
