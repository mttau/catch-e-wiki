Overview

The following table identifies the Angle Finance Interface validation error(s) and to how resolve them to pass a successful application.
For Angle Finance, a successful application is defined when an 'Application Id' is returned and populated in the field Finance Application.
Use this number in the Angle Portal to search for the submitted application to complete any outstanding items.
Once an 'Application Id' has been issued, the 'Apply' button cannot be re-used to 'push' any further changes from Catch-e to Angle Finance. This will trigger an error as the Angle Interface will not re-issue another 'Application Id' against a Catch-e 'Quote ID' already been submitted.

Acknowledged

If your application returns an "Application Status" of 'Acknowledged', it has failed. A PDF copy of the data submitted is stored as Quote Event.
Go to Quotes / Events and find the latest 'Angle Application' event.
Open the stored PDF file to review and fix the recorded errors.
Go to Application Outcomes and Troubleshooting for a summary of the application statuses.
If the response provided is the one shown below, please contact your Angle Auto Finance representative directly for assistance.
