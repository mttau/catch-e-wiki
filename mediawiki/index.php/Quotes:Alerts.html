Amount Financed is out of balance

This message is rare but can occur in the following situations
Issue 1
Your administrator has changed the method of calculating Option GST recently. When switching the from one way of calculating Option GST to the other and an older quote is copied, an “Amount financed is out of balance” error message may present. Typically this is due to the GST rounding difference of the accessories incorporated in the "Total On Road GST" compared to the "GST Claimable" amount.

To fix this, alter the "GST Claimable" amount manually to fix the imbalance and trigger a recalculation of the lease budget totals.

Issue 2
Your internet connection has been disrupted whilst the quote was being saved. This can lead to the "Amount Financed" field not adding up correctly. The validation protects users from this error as a quote cannot be saved or edited again until the field is correct.

To fix this, go to the Finance tab and enter a value into a field that affects the "Amount Financed" total (say $100 in Broker Fee)and tab out of the field . Take the amount out again and save (to then have a correct quote). This forces the system to re-calculate the "Amount Financed" total and in the process - remedies the error! 

Validation failed because of the following fatal error:
Amount financed is out of balance by $x.xx 
This is most likely due to a change in the way rounding is being applied to 'On Road Price GST' since this quote was last edited. 
You can fix it by adjusting the value of 'Claimable GST' on the Finance screen by this amount, 
or triggering a recalculation by manually re-typing one of the Vehicle Price components

FBT Payable amount out of sync with FBT Value

This message is rare but can occur if some disruption to the users Internet connection has occurred whilst the quote is being saved. This can lead to the "FBT value yearly" field not adding up correctly. The validation protects users from this error as a quote cannot be saved or edited again until the field is correct.

To fix this, go to the Vehicle tab and modify the FBT value back and forth. This forces the system to re-calculate the "FBT value yearly" field and in the process - remedies the error!

Selected Driver ID linked to different Client ID

This alert is shown when the Driver ID selected is not linked to the allocated Client in the quote. This will occur where the Driver ID has been altered in the text box, rather than using the search function icon.To avoid such an error, it is recommended that any driver allocations are completed using the search function icon, as this restricts the list of drivers linked to the allocated client in the quote.

To fix this issue, select a driver from the lookup search box or create a new driver for this client if required.

The driver you have selected belongs to [client name].  The quote client code will change to [client code].
