Overview

Get a history of Query changes for a Query stored in the [gb_queries] table.

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on Queries:Get.

[edit]

URL Examples

https://api.catch-e.com/gb/queries/archive
[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

Parameters
