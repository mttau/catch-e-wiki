Update claims

Jump to: navigation, search

Main Page → Workflow → Salary Packaging → Claims → Enter claims | Update claims | Prepare FIFO Travel Expenses

Qualifiers

  1. You have the Scheduler Jobs set up for processing claims.
  2. You have a stored claim that needs to be allocated to a different Contracts / Lease Period
  3. The employee has an active Employees / Item record and
    1. Benefit Type is 'Exempt'
    2. Disbursement Type is 'Claim'
    3. Status is 'Active'
    4. Contract ID is populated
  4. Contracts / Lease contains pending periods


Process

  1. Contracts / Lease
    1. Note which period is the next pending one.
    2. Check you have enough pending periods (up to 5) to use
  2. Navigate to Contracts / Maintenance
    1. Note how many pending claims there are and which periods they are allocated to.
    2. Work out how many periods from the next pending period you need this claim to be moved to.
  3. Find and click through to the claim you need to update
  4. Click on
  5. Date - do not edit this - it is updated by the scheduler job
  6. Odo - do not edit this - it is updated by the scheduler job
  7. Qty - this will be shown as 0, update this to the future period you want to change it to. This must be a value between 1 and 5. If you enter anything else, the claim will be added to the next period (I.e. the next pending)
    1. 0 or 1 will update the next pay run period deductions. E.g. 02/08/2021
    2. 2 will update the next pay run period + 1. E.g. 09/08/2021
    3. 3 will update the next pay run period + 2. E.g. 16/08/2021
    4. 4 will update the next pay run period + 3. E.g. 23/08/2021
    5. 5 will update the next pay run period + 4. E.g. 30/08/2021
  8. Total - enter the total including GST.
  9. GST - GST will calculate if required and can be edited to match if the rounding is not correct. It will set to 0 if the claim excludes GST.
  10. Click on
  11. Click on
  12. Attach the supporting file you have received to the claim
  13. Click on
  14. Expect to see a message that says 'Current funds balance will be -$###.## after this payment and is now Pending Funds


Next Steps

A scheduler job will run and

  1. Add the amount of the claim to the specified Contracts / Lease Period PRETAX budget record.
  2. Date
    1. If the the Contracts / Lease Period date is before or within the current month, the claim row Date is updated to match
    2. If the the Contracts / Lease Period date is beyond the current month, the claim row Date is updated to show as the last day of the current month
  3. Odo is updated to show the matching Contracts / Lease Period
  4. Qty is updated to show as 0
