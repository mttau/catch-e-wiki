Overview

Calculating the "Establishment / Admin Fee" is affected by a number of settings in the system. See below for a description of each setting that is used and some examples of the value that would be calculated in various scenarios.

Use the calculation spreadsheet to model the results you want to see: Establishment Fee.

[edit]

Fields affecting "Establishment / Admin Fee"

ScreenFieldNotes
Setup / Reference Data → gb_controls novated_establishment_feeThis value is only invoked when;

1. Contract Types / Details "Group" = 'Novated'.
2. Contract Types / Details "Include Client Establishment Fee" is checked off.
3. Setup / Reference Data → qt financier posting map defaults Financier / Contract Type combination does not exist.

Contract Types / DetailsGroupThis can be 'Lease','Fleet Managed','Novated','Insurance' or 'Non Billing'.
Contract Types / DetailsInclude Client Establishment Fee If this checkbox is selected, any value stored in the Clients / Quote Defaults "Establishment Fee" will be added to the Quote's "Establishment / Admin Fee".
Clients / Quote DefaultsEstablishment FeeIf "Include Client Establishment Fee" is selected for the quote, this value will be added to the quote's "Establishment / Admin Fee", otherwise it is ignored.
Setup / Reference Data → qt financier posting map defaultsEstablishment FeeIf a Financier/Contract Type combination matches the quote this value will always be added to the quote's "Establishment / Admin Fee".
