Overview

The Channels / Contact tab stores information about the selected channel.


[edit]

Filters

Field Type Description
Channel CodeLookupEnter or look up the Channel Code you want to view or edit.
NameDisplayDisplays the name of the selected channel.
Active ContractsDisplay Displays the number of active contracts that are linked to the selected channel
Active UsersDisplayDisplays the number of active users that have a Channel Manager role that is linked to the selected channel.
