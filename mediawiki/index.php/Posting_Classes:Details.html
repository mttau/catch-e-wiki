Overview

Posting class records are used in many places in the system.

  • Contract Types - defines the allowable Budgets and Expenses for the contract type.
  • Contracts / Events - classify each event record type
  • Contracts / Maintenance - classify each payment or expense shown and to apply business rules such are recharge flags (as defined in Contract Types).


New classes can be created here and can optionally be marked as being available to appear as contract events.

To review a complete list of your posting classes, go to Setup / Export Table Data and choose the table 'gl_posting_classes'
