[edit]

Overview

If you configure a web quote, system quotes will be generated on a page in your own website instead of creating a pdf. Using the email option will send a URL link to the driver, instead of an attached pdf.

[edit]

Process

1. Look at a Quote

When the feature is in place, you can select 'URL' from the output options and "Print". A new browser window will launch and direct you to the completed URL statement. I.e. an online quote.
e.g.https://www.clientname.com.au/LeasingQuote?Key=101234

2. Email a Quote

Select 'Email' from the output options and "Print". The email will contain the URL instead of an attached PDF.

[edit]

Notes

  1. The URL information will be returned regardless of quote status or expiry date.
  2. Catch-e does not store any information regarding the content of quotes sent to recipients using this method. You will need to use audit history information to reconstruct the quote details.


[edit]

Client Set-up

1. A web page needs to be developed within your website to accept the Catch-e URL. Your page will use the Get Quote web service to access the required quote placeholders.

2. The "URL" field in Contract Types / Details needs to be populated with the opening string of the address;

e.g.https://www.clientname.com.au/LeasingQuote?Key=
The 'URL' output option is only available on Quotes/Contract once this field is populated.

3. The quote email needs to be updated to refer to the URL, so that the email contains the embedded URL for that quote.
