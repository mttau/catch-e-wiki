Issue a temporary password

From Help wiki

Jump to: navigation, search

Main Page → Workflow → System Administration → User Management → Issue a temporary password

Qualifiers

  1. Your role is 'admin'
  2. The Temporary Password feature is turned on for your system
  3. Users / Details Status is 'active'

Create a temporary password

  1. Go to Users / Password
  2. Click Reset
  3. Generate the temporary password. You can
    1. Click on the to generate a random temporary password or
    2. Manually enter and re-enter a password. This might be easier if you're on the phone to the user and want to relay a password that's easy to remember.
  4. Click Save
  5. Click Send Password if you want to email the user to advise them of the temporary password.

While a temporary password exists

  1. Until the user has updated their password, the temporary password is visible to you on the Users / Details tab
  2. You can re-send the advice email from either the Users / Details tab or the Users / Password tab at any time.

What does the user do next?

  1. On Login, the user is navigated to the Users / Password tab and must change their password before they can continue to use the system.
  2. Once the user has updated their password, Users / Details is updated so that:
    1. the "Password" field is hidden
    2. the "Temporary Password" field is un-checked
    3. the Send Password button is hidden
