Overview

Details here form part of the data used to populate new quotes. Visit the Financier page for a summary of the set-up requirements for financiers and quote default rules.

Interest Margin Rates:
The Financiers screen allows multiple financiers to be linked to the client. Each financier can have an interest rate margin applied to the client per term (months). The base rate is looked up by the system from the fm_financier_rates table and added to the client margin rate (where the term matches) to display a final effective interest rate.

When a quote is being created or edited and there is a matching "Margin Rate" stored here for the selected financier and term combination, it will be populated into the Quotes / Finance "Client Margin" field and form part of the quote's "Financier Rate".

Account Details:
A total credit limit and review date can be entered per financier. A status flag of 'active' or 'inactive' is available. If set to 'inactive' the financier is not looked up during the quote process.

The default financier is initially looked up in the quote process from the Clients / Quote Defaults screen.
