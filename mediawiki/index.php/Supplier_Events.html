This screen allows you to see and report on all supplier events within the system.
These are first recorded in in the screen Suppliers / Events tab.
You can navigate directly to Events / Supplier from the Event Chart if it is displayed on your Home screen.
The system defaults to “Incomplete Only” events to allow the user to see outstanding events at a glance. These are also summarised on the Home screen in the Events KPI box.
Individual users can filter their own events and update the Completed box as applicable.

Users can update events as "Completed" by clicking on the completed box per event.
This screen also contains CSV and PDF reporting.
Attached documents can be viewed and downloaded by clicking on the paper clip icon.
Multiple Event Types can be selected in one search function. Select one event type by clicking on it.
User can assign the event to any other user using the User_id field,Access to this field can be role restricted.
