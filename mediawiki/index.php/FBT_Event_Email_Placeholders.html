Placeholders

The placeholders listed below are available for use in Contract Event FBT Emails.

When [gb_templates]name = 'pdf_fbt_declaration_advice_email_message' FBT Email Code = FBTA.

When [gb_templates]name = 'pdf_fbt_declaration_email_message' FBT Email Code = FBTD.

When [gb_templates]name = 'pdf_fbt_confirmation_email_message' FBT Email Code = FBTC.

The markers ## or {} can be used to make placeholders. E.g. #contract_id# or {contract_id} are both fine.
