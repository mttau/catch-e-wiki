Fm::swann::importVehicles()

Main Page → SYSTEM → Scheduler → Job_Types → importVehicles()

This job type is an interface used to import Swann Vehicle Reference data into the fm_swann_vehicles table. This data is used by the exportProposals() job when creating Extended Warranty proposal documents.

See Swann Interface for further details on this interface and Swann Interface Setup for a checklist of set-up items.

  This Job Type requires additional set-up. You will need to contact Catch-e at mailto:support@catch-e.com.au, and Swann Insurance help set this up.
