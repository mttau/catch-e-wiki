Qualifiers

  1. You are updating existing records or adding new records to an existing system table
  2. We recommend first testing your import file in the Staging or Support environment


Import data to a table

  1. Go to Setup / Import Table Data
  2. Module - select the module that the data belongs to
  3. Table - select the table to import the data to
  4. Drop or browse to import the data file
  5. Click on Upload
  6. The uploaded file content will be validated.
    1. If there are problems, the Error Report button will be enabled.
    2. Download the Error Report, review, fix your file
    3. Re-Upload
  7. A file validation message is displayed
  8. Click on
  9. Navigate to the system and check the data you have imported
