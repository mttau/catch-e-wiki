Overview

Allows you to autocomplete (lookup) address detail records from the Geoscape Geocoded National Address File (G-NAF).

This feature is used in the following system locations:
Drivers / Contact Home
Drivers / Finances Previous Address

Employees / Contact Home Address
Employees / Finances Previous Address

Supplementary SWAGGER documentation is available here: https://api.catch-e.com/docs/#/G-NAF/autocompleteGeocodedNationalAddress

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on GNAF:AutocompleteAddressRecord

This API is not configured for external use but can be made available on request.
Contact Catch-e Support to request this update.


[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.

[edit]

URL Examples

https://api.catch-e.com/gnaf/autocomplete
[edit]

Parameters - Path

Key Format Notes Sample Value Mandatory
search_query string Pass the start of the address you are looking for. 172 Little yes
states string Available values : NSW, VIC, QLD, SA, WA, TAS, NT, ACT, OT. WA,VIC no
result_limit string Defaults to 10 if not passed. WA,VIC no
