Qualifiers

  1. You need to create an adjustment to correct the transaction balance of a benefit item
    1. E.g. you have received a refund or money was returned
  2. The employee has an Employees / Item record where
    1. Status is 'Active'
    2. Disbursement Type is 'Claim' or Recurring'
    3. Contract ID is populated


Process

  1. Navigate to Contracts / Claim
  2. Click on
  3. Date - select the date that the transaction belongs to
  4. Row # 1
    1. Date - select the date that the transaction belongs to
    2. Budget - select the appropriate option from the list
    3. Description - enter a description of the adjustment
    4. Total - enter the total including GST (if used). This can be a positive or negative number.
    5. GST - GST will calculate if required and can be edited to match if the rounding is not correct. It will set to 0 if the claim excludes GST.
  5. Click on
  6. If the claim has a negative value and the Payee is 'Employee', the following updates will be made by the system
    1. BSB - removed
    2. Account Number - removed
    3. Reimbursement - unchecked
  7. Click on
    1. Attach the supporting file you have for the adjustment being made
  8. Click on
  9. Click on


Next Steps

  1. The transaction will be added or, subtracted from the benefit item's balance
