Summary

Page Tabs Tab Summary
Set-upSet-up information
SearchSearch for, or create a new approval
HeaderHeader details displayed on the following tabs: Approval, Contract, Maintenance, Vehicle, Events.
ApprovalCreate and edit active approvals.
ContractA replication of the existing Contracts/Contract tab. Displays contract information. This is a non-editable display of contract information only.
MaintenanceA replication of the existing Contract/Maintenance tab. Displays all associated maintenance/approval entries for the selected contract. Fuel Performance and Maintenance reports can be produced from here.
VehicleA replication of the existing Contracts/Vehicle tab. This is a non-editable display of vehicle information only.
EventsA replication of the existing Contracts/Events tab. This is a non-editable display of contract events information and a report can be produced from this page.
