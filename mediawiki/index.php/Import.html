Overview

A file import process is used in some system screens. When required, the screen will provide a drag and drop or browse window for you to identify the file you want to use.
The window is also used in the Attach pop-up.

[edit]

Zip files

If the attach location accepts zip files, they can be in the following formats: .zip, .gz, or .tgz
