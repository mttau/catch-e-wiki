Overview

If enabled, budget amounts for ECM and ECMC can be calculated and added to your novated lease quotes.
Open the Employee Contribution (ECM) calculations file to see the calculation details.

This feature is not available in related APIs just yet.

[edit]

Contracts and FBT Reports

Like other budgets, the ECM and ECMC budgets are populated into accepted contracts.
The ECM budget covers the amount required for the lease term, but will not align for each FBT year.
If you want to align FBT Reports to zero out the ECM amount, you will need to create ECM events, or budget adjustments to shuffle the ECM between FBT years.

See also the Employee Contributions page for more details on this topic.

[edit]

Business Rules

In order to populate these quote budgets, following conditions must be met;

  1. Setup / Reference Data → gb_controls for ‘lease_ecm_create_budgets’ is‘yes’
  2. Contract Types / Details “Group” = ‘Novated’
  3. Contract Types / Accruals "Class" - includes records for ‘ECM’ and ‘ECMC’
  4. Contract Types / Accruals "G" - records for ‘ECM’ and ‘ECMC’ are both set as 'Y' or 'N'.
  5. Quotes / Contract “FBT Type” is either ‘Statutory’ or ‘Operating’
  6. Quotes / Contract “Use ECM?” is checked
  7. Quotes / Contract “FBT Treatment Type” = ‘Standard’


[edit]

Calculation Notes

  1. ECM is calculated as a positive amount
  2. ECMC is calculated as a matching negative amount
[edit]
GST
  1. Contract Types / Accruals "G" - if set to 'Y' the ECM and ECMC budget GST is calculated
  2. Contract Types / Accruals "G" - if set to 'N' the ECM and ECMC budget GST is set to ‘0.00’.
  3. Contract Types / Accruals "G" - if the the ECM and ECMC classes have difference settings, you will see a Validation Alert and the quote cannot be saved until this is changed.


[edit]
Periods Deferred

The calculated monthly ECM will be higher when the contract's billings will be shortened by a deferral.
This will happen in the following conditions:

  1. Clients / Contract Defaults "Budget Term Deferred" flag = yes
  2. Quotes / Finance "Months Deferred" is not zero
