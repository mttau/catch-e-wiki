Overview

This API replicates the action of the Quotes / Finance button.
This refreshes the Quotes / Finance "Base Rate" of a quote and its corresponding contract.
Visit Refresh Base Rate to see the full details.

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on Quotes:RefreshBaseRate.

[edit]

Authentication

Authenticate with the API before running this API.

[edit]

HTTP Method

Use the HTTP Method 'POST' for consuming this web service.

[edit]

URL Examples

https://api.catch-e.com/qt/quotes/{quote_id}/refresh-base-rate
[edit]

Path Variables

Key Format Notes Mandatory
quote_id string Quote to accept. Required
