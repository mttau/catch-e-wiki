[edit]

Overview

Allows you to extract details for a driver

[edit]

Permissions

If you want to use this API in your system, enable Drivers:Get.

  1. Go to System Roles and select the 'web_services' role
  2. Navigate to the Roles / APIs tab
  3. Find the Drivers:Get permission and check its check box

Authenticate to api using this method Authenticate with the API, Please refer this link for login information.

[edit]

HTTP Method

Use the HTTP Method 'GET' for consuming this web service.


[edit]

Input Fields (Body)

JSON Field Format Notes Mandatory
driver_idStringClient ID Yes
