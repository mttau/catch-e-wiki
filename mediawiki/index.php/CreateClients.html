Overview

Enables you to create new Client/s including associated Client Bespoke fields, Quote Defaults and Contract Defaults.
Visit the Clients page for full details.
Supplementary SWAGGER documentation is available here: https://api.catch-e.com/docs/#/Clients

Multiple Client/s can be create with a single createClients request. See Creating Multiple Clients section for further details.

[edit]

Permissions

To run this API, the nominated 'web-services' role needs to be given permission.
If you are not actively using the API, leave the permission off for better security.
Go to Roles / Apis and check on Clients:Create

This API is not configured for external use but can be made available on request.
Contact Catch-e Support to request this update.


[edit]

HTTP Method

Use the HTTP Method 'POST' for consuming this web service.

[edit]

URL Examples

https://api.catch-e.com/fm/clients
