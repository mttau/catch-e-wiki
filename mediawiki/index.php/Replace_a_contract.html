[edit]

Qualifiers

  1. The vehicle has been replaced
  2. Perhaps an insurance write off


[edit]

Note old contract details

  1. Navigate to the old contract
  2. Contracts / Billing - Remaining - note
  3. Contracts / Lease - print Lease Schedule
  4. Contracts / Contract
    1. End Date - note
    2. Suspend Date - enter
    3. Notes: - add notes
  5. Contracts / Payments
    1. Stop Payments check and adjust if required
    2. Due Date and Period - note date of first stopped payment and number of remaining stopped payments for each payment type.
  6. Contract ID - copy
  7. Click on


[edit]

Create new contract

  1. Contracts / Order
    1. Contract ID Replaced - enter old Contract ID
  2. Contracts / Billing
    1. Contract Start - enter day after Suspend Date of old contract
    2. Contract End - check this remains the same as the old contract (can refer to Lease Schedule)
    3. Periodic Billings - refer to Lease Schedule printed above and noted Remaining of old contract
    4. Start Periodic - refer to Lease Schedule printed above for next billing date
  3. Contracts / Lease
    1. check amounts match Lease schedule printed above
    2. adjust if required
  4. Contracts / Events
    1. create a 'Registration Change' event
    2. create a 'Driver' event (if required).
  5. Contracts / Vehicle
    1. Description - use picker to find and add the correct variant
    2. Build Date
    3. VIN
    4. Engine No
  6. Contracts / Payments
    1. Create a new schedule for each remaining payment
    2. Check these match the stopped payments on the old contract
  7. Click on
  8. Contracts / Order
    1. Contract ID - copy
    2. Contract ID Replaced - navigate to old contract


[edit]

Update old contract details

  1. Contracts / Terminate
    1. Replacement Order ID
    2. Termination Notes - add notes
