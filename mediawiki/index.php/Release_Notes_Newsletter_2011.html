November 2011 emailed on 08/12/2011

  • Pricing lock checkboxes have been added to Quotes/Vehicle to lock down dealer quoted pricing
  • A "Cost Price" field has been added to Quotes/Options
  • A "Due Off Quote Manager" field has been added to Contracts/Terminate to facilitate termination management
  • A vehicle's "Fuel Capacity" can now be stored in Contracts/Vehicle
  • Finance Application fields in Quotes/Finance can now be edited outside of web-services
  • The field “Location” has been added to the Fuel Performance xls and CSV output files.
  • An "e-Tag" status field has been added to Contracts/Order
  • Contract ID's can now be looked-up in Contracts/Order and Contracts/Terminate


Create Driver - November 2011 emailed on 18/11/2011

  • Drivers can now be created on the fly whilst creating a quote using a pop-up window.


Pay Cycle Billing - September 2011 emailed on 09/09/2011

  • Pay Cycle Billing created an ability to structure a contract's billing by pay cycle (e.g. Weekly, Fortnightly) on the Billing screen.


July 2011 distributed on 01/09/2011

  • Define Contracts / Maintenance exclusions using a Global Control
  • A new insurance product called Walkaway has been created
  • Client Managers to be selected instead of defaulting in the Clients / Contact screen
  • Quotes can be locked when finished with the "Lock Quotes" flag on Quotes / Contract screen
  • Toll charges can now be consolidated on the Combined Invoice
  • Receipts Import Validation has been extended to verify the Contract ID and Reg No match


June 2011 PAYG update emailed on 30/06/2011

  • System controls have been updated to include updated PAYG as of 1st July 2011 – these incorporate the Flood Levy that is being applied in the 2012 financial year. Refer to http://www.treasury.gov.au/floodrebuild/content/keyfacts.asp for more details on this.
  • Quotes created with an “Effective Date” of 1st July 2011 and beyond will use the new rates.


May 2011 distributed on 21/06/2011

  • Refine your quote registration calculations by using business or private values
  • Send financial transactions to multiple instances of MYOB for separate companies
  • Store banking details by channel so they can be used on channel driven invoices
  • See your vehicles Division in the Contract/Header area
  • Send invoices to more than one person with multiple email addresses

May 2011 FBT Reforms distributed on 20/05/2011

  • New field "Statutory Rate Date" changes how the rate is calculated following Federal Budget changes.

April 2011 distributed on 17/05/2011

  • Use a Driver Report template "Novated Budgeted Report with Funds" that shows the Drivers cash position
  • Use "To" and "From" filters in Fuel Performance and Mainenance Reports
  • Report Actual Finance payments in your Driver Reports
  • Specify your own screen resolution
  • Send emails from a default emaill address when it is not stored

March 2011 distributed on 12/04/2011

  • Driver Reports - configure to show Reg No in subject area of email
  • Allocate Brands by Channel ID
  • Create second shortcut using multiple domains
  • Create email when a Fuelcard is sent
  • Create an ABA file for a Receipts batch
  • Use UPSERT to update/insert values into database fields
  • Create event records when scheduled emails are sent
  • Add timestamp to Odo records in the Maintenance tab
  • Pre-populate the "From" field in the Events/Contract screen
  • Show the vehicles target reading in the Declaration page if flagged ON in gb_controls
