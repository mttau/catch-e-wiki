Overview

Many fields in the system have a icon next to them when you have clicked on , or .
Click on this if you do not know the required code and want to search for a particular record to use.
When you have clicked on a particular record in the lookup pop-up list, the field will populate and the lookup pop-up will close automatically.

Also visit the Lookup page.

[edit]

Filters

Filter Type Description
By:Drop ListSelect the field you want to search on
Text: Text Enter the value you want to search on. This is not case-sensitive. E.g. entering 'a' will return all records that contain 'a' or 'A'
Button Click to close the lookup pop-up.
Supplier Type: Drop List If a supplier type has been populated, this is the recommended type of supplier to use in the subject field, otherwise the selection will be '--All--' for all suppliers.
Contains Text: Check Box Un-checked by default. Check this if the entered text should be searched for as a value within a string. E.g. if un-checked, entering 'a' for "Supplier Code" will return 'A 1FIN' and no other records. If checked, any record that contains 'a' or 'A' will be returned.
Button Click to execute your chosen filtering.
