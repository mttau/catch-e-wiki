Accessibility

Internal Users - Full access to Catch-e is currently only possible using supported browsers. Please find the list of supported browsers below.

External Users - All screens available to the following external roles, driver, fleet_manager, group_manager, package_employee, package_advisor, and fbt_declaration, are accessible via the Firefox, Safari, and Chrome browsers as well as Internet Explorer. These external roles typically access Catch-e via a link / button on the Organisation's Internet site.

[edit]

Supported browsers

Browser Version
Microsoft Edge Version 18 or greater. Edge is the browser we recommend to run the Catch-e System.
Google Chrome Version 30 or greater
Mozilla Firefox Version 27 or greater
Safari Version 7 or greater
Safari(mobile) Version 6 or greater


[edit]

iPad Support

Catch-e can run on Safari in an iPad for externally accessed screens and for raising approvals.

  • Go to Settings/Safari and set "Block Pop-ups" = OFF
  • Enter the following address: https://[yourname].catch-e.net.au/bo.phpo?screen_width=1024&screen_height=720

Open the iPad configuration guide for detailed steps and visuals.

[edit]

Multiple sessions

You can run two sessions of Catch-e in a browser. Visit the Getting Started page for set up details.

[edit]

Third Party Toolbars

Third party toolbars e.g. yahoo, and Google often include additional security options such as popup blocking that can interfere with the operation of Catch-e. If Catch-e is still not working correctly after applying all of the suggested settings below, please try disabling / removing these toolbars.

[edit]

Browser Add-ons

Occasionally a faulty add-on can interfere with the operation of Catch-e.
If you're experiencing strange browser behaviour when navigating around the system, the best way to diagnose if an add-on is the cause is to temporarily disable them. Once disabled, you can re-enable them one by one as you test for a resolution.

[edit]

Edge

When finished, your downloads are visible in the download bubble shown in the Browser Header.

  1. Go to Settings
  2. Select Downloads (edge://settings/downloads)
  3. Show downloads menu when a download starts - enable this setting


  1. Go to Settings
  2. Search Downloads ((edge://settings/?search=Downloads)
  3. Downloads button - enable this setting


[edit]

Chrome

When finished, your downloads are not visible in the download bubble shown in the Browser Header.

  1. Go to Settings
  2. Search Downloads
  3. Show downloads when they're complete - enable this setting
[edit]

Firefox

Some users have had problems launching the Help Wiki. If this happens, do the following:

  1. Close all Firefox browser windows
  2. Launch a generic Firefox session, I.e. not a Catch-e System URL
  3. Navigate to find if any browser updates are pending
  4. If there are pending updates, update the browser
  5. Relaunch Catch-e. The Help wiki should now launch successfully (the irony of asking you to read this Help Wiki content whilst being unable to launch it is not lost to us).

Supported Screen Resolutions

If your screen display is missing items or looks incomplete, you may be using an unsupported screen resolution. Our supported resolutions are listed below.
To remedy the problem, consider using address parameters to define a supported resolution when you run Catch-e.
Visit Configure your screen resolution for details on how to set this up.

Adjusting the Internet Explorer Zoom level may affect the screen display area as the screen height is affected. We recommend 100% zoom level if you experience issues at other zoom levels.

Screen size can be set using the command line parameters screen_width and screen_height. eg. http://demo.catch-e.net.au/bo.phpo?screen_width=1024&screen_height=768

Supported Resolutions

Supported Widths:

1024, 1152, 1200, 1280, 1400, 1440, 1600, 1792, 1920, 2048

Supported Heights:

768, 800, 864, 900, 960, 1024, 1050, 1080, 1200, 1344, 1440, 1536

Note: If your screen size is not supported, the system will automatically set your monitor to use the closest supported screen height (this will be the first available that is lower than the monitor height).
