Overview

This has been superceded by Maclease Partner Interface.
An interface has been developed with MacLease (Macquarie Leasing). This is in the form of an XML file which can be generated from the Fleet/Administration/Quotes/Finance screen.

The option to generate the MacLease interface is made available when the Financier code is set to Macquarie Leasing (recognised by the appropriate supplier_id advised to Catch-e).

The process can be done via 2 methods:

[edit]

A. Create XML file and upload manually

  1. Create XML file in Catch-e by selecting the "Maclease" output option. Click the "Print" button and save the file.
  2. Login to MacLease.
  3. Select the option Application/Import Application.
  4. Import the saved XML file.
  5. Review the information uploaded and follow normal MacLease processes
[edit]

B. Web service using "Apply" button

An "Apply" button appears on the Quote/Finance screen when certain criteria are met:

a) The Financier must be Macquarie
b) The Status in the Finance Application section must not be "successful"
c) Login credentials for the MacLease service must have been set up for the relevant user_id (further details may be given in your Wiki)
d) Supplier "Interface Type" of 'maclease' must be selected on Fleet / Administration / Suppliers / Account

When clicked the system sends the application details and retrieves a reply from Maclease with a message (i.e. successful or unsuccessful) and an EAN number which is then stored on the Quote/Finance screen. This process takes only a few seconds. An application may be done only once per quote. A copied quote will retain an application. You can login to Maclease to view and edit the application if required.

If an application fails the user will be given an error message and the Finance Reference may be set to "Not Assigned".

The existing method of creating an xml file manually is still available.

To use the interface, Maclease logins and passwords need to be supplied to Catch-e for storing in the system table gb_user_logins and the web service activated.
The apply button will not appear unless this is done. The Maclease user login may be associated with the A/C Manager on the quote OR the user logged in to Catch-e. Catch-e needs to be notified of the association method to be used.

Note: a Drivers's MacLease application will not work unless their salutation is Ms, Miss, Mrs, Mr or Dr.

A number of fields are compulsory for the interface to proceed - users are prompted if these fields are not populated:

  1. Title (driver salutation).
  2. Surname.
  3. First name (note middle name is appended to this field in the interface.
  4. Date of Birth.
  5. Address 1.
  6. Address 3.
  7. State.
  8. Postcode.
  9. Previous Address 1 IF years at address < 2.
  10. Previous Surburb/City IF years at address < 2.
  11. Previous State IF years at address < 2.
  12. Previous Postcode IF years at address < 2.
  13. Residential Status.
  14. Landlord name IF residential status is Rented.
  15. Landlord phone IF residential status is Rented.
  16. Years employed.
  17. Position (i.e. occupation).
  18. Previous Employer Name if years employed < 2.

The Maclease user account must be setup as an "Introducer" within Maclease. The webservice will give a Maclease error if the user account is setup as an "Assistant".

[edit]

Additional Information

Two new fields were added as a requirement to Consumer Loan products in Dec 2014, but XML applications are unaffected by the change.
If that information is required by Macquarie, their Lending and Settlement team will get in touch with the relevant introducer to obtain the missing information.

Catch-e Set-up

Each finance application made to Maclease is performed using common Maclease user signons. This mapping maps a Catch-e user to a Maclease record. Contact Catch-e support to activate or cancel a users Maclease signon.

  The interface will not be available if this mapping has not been set up for the logged in user. This means the ‘Apply’ button will not be available.

FieldDescriptionAllowed Value
gb_user_logins
user_idLinking to Catch-e user id in gb_users Example 10006
serviceService for which this login to be usedmaclease
login8 character Maclease User CodeProvided by Macquarie.
passwordencryptedProvided by Macquarie.
status_flagActive or Inactiveactive or inactive
