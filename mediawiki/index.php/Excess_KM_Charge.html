[edit]

Overview

Use the excess km charge amount when quoting operating leases.
This records an amount per km that can be charged at lease end if the vehicle is returned and it's km's are higher than agreed.

[edit]

Excess KM Charge

FieldTypeDescription[table]field
Select Check if you want to add an Excess KM Charge to this quote.
It is checked by default if the Contract Type / Details "Quote Excess Km" field is checked.
$ Per KM Number (4dp) If unchecked, this field will display as '0.0000'.

If checked, a default amount will populate into this field, but can be updated.
Depending on your system's configuration, the default amount will either be

  1. Calculated using a factor of the quoted maintenance cost per km
  2. A flat rate from the Clients / Quote Defaults "Excess KM Rate" field.


[edit]

Set-up

For each contract type, review and check on the Contract Type / Details "Quote Excess Km" field if you want to apply the charge by default in new quotes.

Decide if you want to charge a factor of the quoted maintenance cost per km for this vehicle or a flat cents per km rate.

Note that the Factor method uses a 2 decimal calculation, e.g. 0.07 result is 7 cents. The flat rate method allows 4 decimal places, e.g. 0.1255 is 12.5 cents.

[edit]
Factor

To use a factor, update [gb_controls]excess_km_factor to record the factor you want to use.
"$ Per KM" is calculated as the Maintenance "Cents per KM" x [gb_controls]excess_km_factor / 100.
E.g. ROUND((3.75*2.00)/100,2) = 0.0700.

[edit]
Flat Rate

To use a flat rate, advise Catch-e Support, as a code change is required to switch to this method.
Once switched, update [gb_controls]qt_client_defaults_excess_km_rate_default to record the default amount you want to use when creating new client records.
Review the amount stored in the Clients / Quote Defaults "Excess KM Rate" field for each client.

The checkbox is available here and also the Quotes / Contract tab. It will default on or off as per the linked Contract Types / Details "Quote Excess Km" checkbox.
