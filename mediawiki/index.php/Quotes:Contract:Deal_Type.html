Overview

Quotes / Contract "Deal Type" is available for you to set a description for the quote that is useful for your business.

[edit]

Set-up Notes

[edit]
List

This field has no pre-set list when your system is first created. Create the choices you want for this field by adding them into the table [qt_deal_types].
The list is presented in [qt_deal_types]deal_type_id order.

[edit]
Auto Add Options

Use a "Deal Type" selection to trigger the use of Options Included by Default

[edit]

APIs

Get the available list from the API getDealTypes.

[edit]

System Notes

  • (5558) Optional Equipment Read Only feature. Live 01/03/2012
  • (DEV-20981) Optional Equipment is removed when Deal Type is changed. Live 17/02/2020
  • (DEV-21652) Exclude default accessories for Used vehicles. Live dd/12/2022
  • (API-20369) Create getDealTypes API. Live 24/07/2023
