Overview

Prospects are a type of client as defined on the Clients / Contact Screen. The prospect screen contains information related specifically to the prosepecting process. Available fields are:

[edit]

Header Buttons

Button Description
Click to create a new client. Three fields in the Clients / Contact tab are mandatory. These are: "Client Code", "A/C Manager" and "Sales".
Click to copy an existing client record. All existing record details except for the "Client Code" are copied to the new client record.

[edit]

Output Options

A csv report of the prospect fields and the relevant contact fields can be printed from this page.
The report is linked to the user login, so that only Prospects for the salesperson logged in appear in the report. An overall report can be run using the SQL report writer.
Contact your Account Manager if you would like a scheduled report set-up for this.
