Overview

This web service has been deprecated and is no longer supported by Catch-e.
You must use APIs → Attachments to develop attachment API functions.


This web-service allows you to delete an attachment via a web service rather than using the regular web-based attachment upload function.
The system will set the [gb_attachments]status_flag of the attachment to 'deleted'. The record is left linked to the parent record, E.g. Event.
