Overview

This table stores the details of insurance suppliers that you offer a managed insurance product for.

If the Clients / Quote Defaults "Policy Owner" is selected as 'internal, active records from this table can be selected as the "Insurer". These values will default into new quotes.

If the Quotes / Contract "Policy Owner" is selected as 'internal', active records from this table can be selected as the "Insurer". These values will default into contracts from accepted quotes.

If the Contracts / Insurance "Policy Owner" is selected as 'internal', active records from this table can be selected as the "Insurer".

You can set up multiple "Policy Owners" for different channels, each using a separate Certificate of Currency document.

Contact your Account Manager for help to set-up in internal insurers.

Records for internal insurers used to be stored and managed from [gb_controls]insurer_internal records. These records has been moved into this table and the last record added has been set to 'Active' status in here. If you used other suppliers in the past, their records are set to 'Inactive' here.
The records stored in [gb_controls]insurer_internal have been kept, but are no longer functional in the system and are for historical interest only.
