Overview

Each electric vehicle needs an 'EV Charging Method' event stored to control which kinds of Fuel / Charging claims can be made.

[edit]

Process

Contracts / Events - 'EV Charging Method'

This coded event is only applicable for full electric vehicles, I.e. "Fuel Type" is 'Electric'.
The default choice is 'Home Charging'.

1. Actual Cost: Claim actual costs from a external charging stations.
2. Home Charging: Claim 0.042c per km based on the odometer reading.
3. Home Charging Percentage: Make claims using both methods.
    Chose 'Home Charging Percentage' if the vehicle can generate an annual report that provides how much charging was done externally and how much charged at home.
    Actual costs can be claimed on an ongoing basis.
    Home charging can be claimed when the driver has determined the percentage of charging that was done at home for the year.

Data Illustrations

Below are images and details to illustration the available settings.
The Scheduler Job 'DP - Contract list with Fuel Claim Logic' classifies each of your vehicle contracts with descriptions shown below.

[edit]

0 Bad data - Fuel Type is blank

  1. Contracts / Vehicle "Fuel Type" does not have an entry.
  2. To fix data
    1. Edit your contract and update the "Fuel Type"

   

[edit]

0 Bad data - Fuel Type is not Electric

  1. Contracts / Vehicle "Fuel Type" is not 'Electric'
  2. Contracts / Events 'EV Charging Method' event is present.
  3. To fix data, either
    1. Update the "Fuel Type" to 'Electric' (if the fuel type is incorrect)
    2. Delete the Contracts / Events 'EV Charging Method' event
[edit]

1 Switches to Home Charging

  1. Contracts / Vehicle "Fuel Type" is 'Electric'.
  2. Contracts / Events 'EV Charging Method' is 'Home Charging'.
  3. The Home Charging screen is shown.
  4. The "Type" defaults to 'EV Home Charging'
  5. No other "Type" can be selected

[edit]

2 Choose from Home Charging Percentage or Electric Charging

  1. Contracts / Vehicle "Fuel Type" is 'Electric'.
  2. Contracts / Events 'EV Charging Method' is 'Home Charging Percentage'.
  3. The Home Charging screen is shown.
  4. 'EV Actual Cost' or EV Home Charging Percentage' can be chosen.
  5. No other "Type" can be selected


[edit]

3 Type is set as Electric Charging

  1. Contracts / Vehicle "Fuel Type" is 'Electric'.
  2. Contracts / Events 'EV Charging Method' is 'Actual Cost'.
  3. The Standard screen is shown.
  4. The "Type" defaults to 'EV Actual Cost'
  5. No other "Type" can be selected


[edit]

4 Type is set as Electric Charging- Contract Start is before 01/04/2022

Even if an event is set, contracts that start before '01/04/2022' will not be taken to the Home Charging or Home Charging Percentage screens.

  1. Contracts / Contract "Contract Start" is before '01/04/2022'
  2. Contracts / Vehicle "Fuel Type" is 'Electric'.
  3. Contracts / Events 'EV Charging Method' is 'Home Charging' or 'Home Charging Percentage'.
  4. The Standard screen is shown.
  5. The "Type" defaults to 'EV Actual Cost'
  6. No other "Type" can be selected


[edit]

5 Type is set as Electric Charging

  1. Contracts / Vehicle "Fuel Type" is 'Electric'.
  2. The Standard screen is shown.
  3. The "Type" defaults to 'EV Actual Cost'
  4. No other "Type" can be selected


[edit]

6 Choose from standard list incl Electric Charging

  1. Contracts / Vehicle "Fuel Type" is not 'Electric'.
  2. The Standard screen and choices are shown.
  3. Home Charging and Home Charging Percentage screens are disabled.
  4. All other "Types" can be selected.
