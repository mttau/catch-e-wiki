Synopsis

This web service has been deprecated and is no longer supported by Catch-e.
You must use Quotes / Variants APIs instead.


Get a list of model years for a given vehicle model. Suitable for use in a list box, using as both the option and value properties.

Model Years are linked to Varaiants, and Variants are excluded if they have a status of 'pending'. So if all the Variants for a particular Model Year are at status 'pending', that Model Year will not be returned in the list.

[edit]

Input Fields

  Query string key
Format
Notes
Mandatory
1 model_id
integer
Unique model identifier
yes
2 profile_required
yes/no
If yes only model years for variants that have an active qt_variant_profile record will be included. The default is no.
no
3 row_number_flag
yes/no
If yes will make each node unique i.e. <row_1>, <row_2>... if no all nodes will be the same i.e. <row>. The default is yes.
no
