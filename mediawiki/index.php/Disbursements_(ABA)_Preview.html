Overview

This report is a default screen report.

[edit]

Report Details

Field Name Comments Source
BECS IDBlank
Bank Account
  1. When "ABA Type" is 'Client'
  2. When "ABA Type" is 'Company'
  1. [gb_becs]client_id links to [sp_client_packaging_defaults]bank_account_name
  2. [gb_becs]bank_account_id links to [gl_bank_accounts]account_name
Payee
  1. When "Payee" is 'Supplier', show the supplier's name
  2. When "Payee" is 'Employee' or 'Driver', show
    1. Bank Account Name if populated
    2. Given Name Surname
  1. [fm_suppliers]name
  2. Either
    1. [fm_driver_bank_accounts]account_name if populated
    2. [fm_drivers]given_name [fm_drivers]surname
BSB
  1. When "Payee" is 'Employee'
  2. When "Payee" is 'Supplier'
  1. [fm_driver_bank_accounts]bsb
  2. [fm_suppliers]bank_bsb
Account Number
  1. When "Payee" is 'Employee'
  2. When "Payee" is 'Supplier'
  1. [fm_driver_bank_accounts]account_number
  2. [fm_suppliers]bank_account_number
Contract ID
  1. Employees / Item "Contract ID"
  2. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then 0
  1. [sp_employee_package_items]contract_id
  2. 0
Pay Cycle
  1. Contracts / Billing "Periodic Interval"
  2. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then blank
  1. [fm_contracts]pay_cycle_id linked to PROPER [gb_pay_cycles]pay_cycle_code
  2. blank
Approval ID
  1. If the selected record is a Contracts / Claim
  2. Otherwise 0
  1. [fm_approvals]approval_id
  2. 0
Reference
  1. Contracts / Claim "Reference"
  2. Contracts / Payment "Supplier Ref"
  3. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then blank
  1. [fm_approvals]invoice_no
  2. [fm_contract_payment_structures]supplier_reference
  3. blank
Date
  1. Contracts / Claim
  2. Contracts / Payment "Post Date"
  3. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then blank
  1. [fm_approvals]posted_date
  2. [pl_payables]posted_date
  3. blank
Class
  1. Contracts / Claim "Budget"
  2. Contracts / Payment
  3. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then blank
  1. [fm_maintenance]posting_class_id links to [gl_posting_classes]name
  2. [fm_contract_payment_schedules]posting_class_id links to [gl_posting_classes]name
  3. blank
Description
  1. Contracts / Claim "Description"
  2. Contracts / Payments "Payment Type"
  3. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then blank
  1. [fm_maintenance]description
  2. [fm_contract_payment_structures]payment_type
  3. blank
Net
  1. Contracts / Claim Row "Net"
  2. Contracts / Payments Row "Net"
  3. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then Sum of Amount Net for Supplier's Claims and Payments
  1. [fm_maintenance]total_cost
  2. [pl_payables]amount_net
  3. SUM( [fm_maintenance]total_cost + [pl_payables]amount_net )
GST
  1. Contracts / Claim Row "GST"
  2. Contracts / Payments Row "GST"
  3. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then Sum of Amount GST for Supplier's Claims and Payments
  1. [fm_maintenance]gst_cost
  2. [pl_payables]amount_gst
  3. SUM( [fm_maintenance]gst_cost + [pl_payables]amount_gst )
Total
  1. Contracts / Claim Row "Total"
  2. Contracts / Payments Row "Total"
  3. When "Payee" = 'Supplier' and the Supplier's "ABA Format Type" = 'Supplier' then Sum of Amount Gross for Supplier's Claims and Payments
  1. [fm_maintenance]total_cost + [fm_maintenance]gst_cost
  2. [pl_payables]amount_net + [pl_payables]amount_gst
  3. SUM( [fm_maintenance]total_cost + [pl_payables]amount_net + [fm_maintenance]gst_cost + [pl_payables]amount_gst )
