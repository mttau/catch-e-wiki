[edit]

Derived fields from the /lib/PDF/fuel_performance library

This library is used to calculate fuel usage values e.g. Litres / 100Km. To do this it calculates a km span between the current row and the previous one.

The following Input Fields are required as input from your query:

Field Source Rules
reg_no fm_contracts->reg_no Mandatory
odometer fm_maintenance->odometer Mandatory
odometer_valid_flag fm_maintenance->odometer_valid_flag Mandatory
litres fm_maintenance->litres Mandatory


The following Derived Fields can be added to your query using the Derived Fields syntax:

Field Description
_span Odometer for the current record minus the odometer from the previous record
litres_per_100k Running Litres / 100km for the current transaction
litres_per_100k_group_code Running Litres / 100km for the current Client Group
litres_per_100k_client_code Running Litres / 100km for the current Client
litres_per_100k_division Running Litres / 100km for the current Division
litres_per_100k_cost_centre Running Litres / 100km for the current Cost Centre
litres_per_100k_reg_no Running Litres / 100km for the current Reg No
