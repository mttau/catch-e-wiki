Overview

When editing an Item, click on the icon to open the Select Bank Account modal window.

A variant of the Select Bank Account modal window is also used in the Contracts / Claim tab.

[edit]

Field Descriptions

FieldTypeDescription[table]field
Bank Accounts
Account Name Text When an account is being created, enter an “Account Name”. [fm_driver_bank_accounts]account_name
BSB Num Display or add the ”BSB” for the bank account record [fm_driver_bank_accounts]bsb
Account Number Num Display or add the “Account Number” for the bank account record. [fm_driver_bank_accounts]account_number
Claim Check Box Check if the account is the employee's own account and is available for Novated Lease Reimbursements. [fm_driver_bank_accounts]claims_flag
DefaultCheck Box Check if the account is the employee's own account and is available for Novated Lease Reimbursements. Only one account can be checked. This will default into new Contracts / Claims records for novated leases. [fm_driver_bank_accounts]default_flag


[edit]

Action Buttons

Button Description
View
Click to add a new record. A new account row will become available and editable. Enter the Account name and number. Only check on "Claim" if the account can be used for Novated Lease Reimbursements. Only check on “Default” if this should be set as the default account for Novated Lease Reimbursements.
CloseClick to close this window and return to the Employees / Item tab in edit mode
Edit
Create Click to create the new record. This will be saved back to the Employee / Item record you are editing.
If you choose Cancel, you are returned to the Select Bank Accounts window in view mode.
