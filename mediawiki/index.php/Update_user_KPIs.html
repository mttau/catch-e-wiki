Qualifiers

  1. User KPIs need to be added or reviewed

Process

  1. Go to System / Users
  2. Role - enter or lookup the user you need to change
  3. Navigate to Userss / KPIs
  4. Click
  5. Check on each KPI you want users with this role to see
  6. Uncheck each KPI you want to hide from users with this role
  7. Click
