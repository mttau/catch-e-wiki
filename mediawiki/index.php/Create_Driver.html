Overview

When creating a quote, a new driver can be created via a new driver window. The screen behaviour is as follows;

When creating a new quote, a icon (Create Driver) is displayed next to the "Driver ID" field.
If you click on

  • The "Create Driver" window will open. The window presents the Drivers / Contact tab.
  • Enter the driver details and "Create". Note ~ the "Client" field will be filled in from the quote record and cannot be changed here.
  • "Create", "Cancel" or "X" will close the driver window.

If you enter the driver's name directly into the quote;

  1. And the driver's name is new, you will be invited to create a new driver record. Selecting "Yes" will take you to the "Create Driver" window where the driver's name and Client details will already be stored. Selecting "Cancel" will return you to the quote screen.
  2. And the driver's name already exists for the selected client, the quote will be refreshed with that driver's record.
  3. And the driver's name already exists more than once for the selected client, a pop-up window will display the drivers attached to the client for you to select from. If none of the listed drivers is correct, close the pop-up and either create a new driver or finish the quote without a driver record.
  4. And the driver's name exists for another client within the group the quote's client belongs to, a pop-up window will display these drivers. You can select from or close this window.
    1. If you select a driver from another client within the group, the quote will update to show client that the selected driver belongs to.
    2. If you exit from the window, you can then create a new driver record, or continue quoting without one.

When a driver record has been selected or created, the icon will change to (Edit Driver).
When you click on , the driver window opens, you can edit the driver details on the Driver / Contact tab here - the other driver tabs are not accessable at this point.

When the quote is saved and you are looking at the quote in view mode, you will see an icon (View Driver). When you click on , you are able to view all the driver record tabs within the driver window as if you had gone to the driver record itself. This screen contains most of the functionality of the driver screens (such as editing finance fields and creating driver events). You cannot change the client or enable login access from this window.

Note:
It is recommended when editing Driver information from the Fleet / Administration / Quotes | Contract/Driver Details screen to use this workflow:

  • Next to the Driver ID click on the 'View Driver' Icon
  • In the pop up 'View Driver' window click 'Edit'
  • Make the required changes and Save
  • Close the View Driver Window using the 'Close X' at top right of the window
  • Changes will be carried through to the quote

Changes to the Driver's records at either Fleet / Administration / Drivers or from Edit mode in the quote by clicking on the 'Edit Driver' icon do not carry through to the quote. The Driver ID needs to refreshed for this to take place.

Refresh the Driver ID from Edit mode by:

  • Deleting the Driver ID
  • Tab off the Driver ID field to clear the driver details
  • Reenter the Driver ID and tab off the field
  • Driver data will be refreshed
  • Save the quote.
