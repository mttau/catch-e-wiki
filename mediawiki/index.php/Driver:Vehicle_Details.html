Main Page → Driver Portal → Sidebar → Vehicle Details

[edit]

Overview

This screen is available for use in the Driver Portal and is hidden by default.
To access this screen you need to activate the [gb_controls]driver_portal_view_vehicle_details.

  1. When Vehicle Details is clicked, the users record is checked to see if they have one vehicle or more
    1. If the user has more than once vehicle, they are navigated to the Choose Car screen where they can select the vehicle they want to view.
    2. If the user has one vehicle, it is selected automatically.
  2. When a vehicle record is clicked, the user is taken to the vehicle details for that record.
    1. There is a selection list available on the vehicle details screen to select other 'Active' or 'Suspended' record.
    2. All the fields on this screen are display-only and cannot be edited.
