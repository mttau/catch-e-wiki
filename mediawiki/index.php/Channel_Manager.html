Overview

Access Type: Internal

Restriction: The Channel Manager can view quotes, drivers, events, and clients that are linked to clients that belong to the nominated channel.

Set Up:

  1. Restrict Key - enter the Channels / Contact "Channel Code" that you want to restrict records to.
  2. All clients with a matching channel code in the Clients / MLA "Channel" field are available to these users.
  3. The Restrict Range feature is not available for this role.
