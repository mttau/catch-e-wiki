Overview

Each event screen allows you to filter and report on all the events that are recorded for a particular module.
Wildcards may be used when searching for event data by using the ‘*’ symbol.
You can navigate directly to an event screen from the Event Chart if it is displayed on your Home screen.

The events in each module are entered and stored in a similar way and are listed below

  1. Contracts / Events
  2. Clients / Events
  3. Drivers / Events
  4. Suppliers / Events
  5. Quotes / Events

Search Filter Definitions

Screen NameTypeDescription[table]field
Group CodeTextDisplays the Group Level of Clients or Suppliers
Client/Supplier CodeTextDisplays the Client Code or the Supplier Code, depending on the Events screen selected
Contract IdNumDisplays the Contract Id. The Contract Id can be entered directly, or the search function can be used to find the required contract.[fm_contracts]contract_id
Reg NoText Displays the Registration number. The registration can be entered directly, or the search function can be used to find the required registration number.[fm_contracts]reg_no
ValueTextUse this to search for values such as old registration numbers.
FromDateSelect events within a date range of when event was created.
ToDateSelect events within a date range of when event was finalised.
DescriptionTextDisplays comments entered at time of event creation.
TypeTextDisplays the list of event codes. You can select one or a group of event types. Select one event type by clicking on it. To select particular event types, hold down the “Ctrl” key and use your mouse to select. To select a range of clients, hold down “Ctrl” and “Shift” keys and use either your mouse or arrow keys to select.
ActionDrop BoxDisplays the action codes allocated to each event.
Due ByDateDisplays events that are due to be completed by the selected date.
Incomplete OnlyCheck BoxWhen checked ON only the incomplete events will be displayed. This is the default setting.
UserDrop BoxDisplays the User that created or edited the event.


  • Note that all contracts will have an opening registration number and cost centre to coincide with the commencement of the contract.
  • If a filter is changed, click on the ‘GO’ button to refresh the results. Events may be printed by using the print buttons at the bottom right of the screen.
  • To find contracts using an old registration number, search by entering the old registration number in the ‘value’ field. The relevant contracts and new registration number will then be displayed.
  • Attached documents can be viewed and downloaded by clicking on the paper clip icon.
