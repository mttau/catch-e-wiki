Overview

This job automates the import of Glasses vehicle data from the Glass FTP server.

This can also be done manually from the system menu Fleet / Import Data â†’ Glass Data(FTP Only)

Go to the Glass Data(FTP Only) page for details about this process and for set-up steps.

Import Glasses data is a legacy function that uses an FTP connection. This function was deprecated in September 2024.
Use the Import Glass Data function instead. It uses an SFTP connection and is more secure.


[edit]

Parameters

These are the parameter settings that are available for this job method.

Parameter Notes
Import type Choose from 'Monthly' or 'Weekly'
Filename The file name can be left blank. The job seeks out the required file.


[edit]

Messages

The messages below are shown on Scheduler / Schedule as "Last Message" and also on Scheduler / Queue as "Message".

Message Comments
Completed
No message is displayed when the job runs successfully.
No new data to import.
Failed
Import timeout exceeded. Review timeout set in global control glasses_vehicle_data_import_timeout. The job will return this message if is takes longer to run than the stored timeout in the control glass_vehicle_data_import_timeout. Update the control timeout if this occurs.


[edit]

Troubleshooting

Message Comments
Failed
CATCH_E_ERROR_GLASS_OFFLINE Try re-running the job at a later time. If the issue persists, contact Glasses directly in the first instance.
