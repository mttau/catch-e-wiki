Header Field Descriptions

Field Type Description [table]field
Column One
Driver ID Number Displays the Driver ID. Enter the Driver ID directly or use the search function to find the driver you want.

When a search is done using the field "Mobile", the number entered in the search field must contain no spaces. The stored mobile number can contain spaces and will still be found.

[fm_drivers]driver_id
Name Text Displays the driver's name. This shows Surname, Given Name. [fm_drivers]surname, [fm_drivers]given_name
Employer Text Displays the employer's name. [fm_clients]name
Column Two
FBT Type Text Displays the employer's "Employer FBT Type" [fm_clients]employer_fbt_type
Payroll Cycle Text Displays the driver's stored "Pay Cycle"
