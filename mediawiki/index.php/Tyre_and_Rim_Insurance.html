Overview

Use this inclusion to quote on a group of products that have a fixed price per product or or a fixed price per product and state.

[edit]

Total Brokerage

The selected amount will be shown in the the Quotes / Finance "Tyre and Rim" field and is also added to the "Total Brokerage" field.

[edit]

Pricing Structure

The pricing structure of this inclusion can be

  1. a simple calculation set in the Global Control qt tyre_rim_insurance record
  2. a price per plan
  3. a price per plan/state
  4. you can have a combination of plan pricing and plan/state pricing in the table and both will be available to choose.
  5. if the pricing is by state, only the records that match the "Delivery State" of the quote will be shown.


[edit]

Set-Up

This inclusion can be renamed by updating [gb_controls]quote_tyre_rim_insurance_label.
This renames both the Quotes / Lease "Tyre and Rim Insurance" field and the Quotes / Lease Inclusions "Tyre and Rim Insurance" field

Use Setup / Import Table Data to update plans in the [qt_tyre_rim_insurance_plans] table.
Process Page: Import data to a table

  1. [qt_tyre_rim_insurance_plans]name - enter the plan name you want to show in the Quotes / Lease Inclusions row for selection.
  2. [qt_tyre_rim_insurance_plans]state - enter the code of the state the plan is for.
  3. [qt_tyre_rim_insurance_plans]status_flag - set to 'active' for new or updated plans. Set the record to 'inactive if the plan is no longer being offered.
  4. [qt_tyre_rim_insurance_plans]swann_interface_flag - is a legacy field for a products in the Swann Interface. This is no longer offered by Swan Insurance.

Troubleshooting

Visit the main Troubleshooting page for a list of all the available problem-solving tips.
[edit]
Quotes / Lease - Tyre and Rim Insurance - plans don't show

If existing Tyre and Rim plans do not show in the Quotes / Lease "Tyre and Rim" selection list, your plan records may have recently been updated.
If the field [qt_tyre_rim_insurance_plans]state is NULL, create an upload file where the [qt_tyre_rim_insurance_plans]state field empty (i.e. do not include the text 'NULL')
Use Import data to a table to upload the file to correct the data.
