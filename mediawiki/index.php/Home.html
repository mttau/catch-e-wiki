Overview

The Home page is the screen displayed when you log in.

The Menus shown at top left are always shown and and available to you from any screen. They areare the starting point for navigating to the different system areas.
If configured, you may also see chart boxes on the home screen.
To return to the Home screen at anytime, click on the symbol found at the top right of all screens.

Field Descriptions

Field Type Description [table]field
User Drop List Displays your login by default. If set, you can select other users in a drop list.
Select '--All--' to see the KPI data for all available users, or select a particular user.
Changing your selection means all the KPI data needs to recalculate. If you choose '--All--', this can take a while!
Workgroup Drop List Displays your workgroup by default. If set, you can choose to see KPI data for other users.
Select '--All--' to see the KPI data for all available workgroups, or select a particular workgroup.
Name Display Displays your name.
Contact Display Displays your phone number.
Email Display Displays your email address.
Position Display Displays your position description.
Reports to Display Displays your Manager's name.
Position Display Displays the position description of your Manager.
RefreshButton KPI data is scheduled to refresh every 15 minutes. Click to refresh and see changes immediately. Some charts are only updated nightly due to the complexity of data and calculations they use. The refresh button will not update those charts. Go to KPI Data for more details.

Set-up Notes

Create or rename Workgroups in Setup / Workgroups,
Set up role access to KPI Charts in Roles / KPIs.

Set up a user's Workgroup in Users / Details.
Set up a user's KPI Charts in Users / KPIs.
Set up a user's access to other users KPI Charts in Users / Details.
Set up a user's access to other Workgroups in Users / Details.
