November 2009 distributed on 7/12/2009

  • Small Damage profile created
  • Originator Fee field
  • Video Content enabled in Help and Client wiki
  • Use docx templates in Catch-e
  • Online Maps enabled

October 2009 distributed on 11/11/2009

  • St George Interface
  • Driver reports updated to show N/A where the Statutory Percentage is 26%
  • MYOB Field “Customer PO” can be populated
  • IMEI call on the fly in the Maintenance screen
  • AccPac Supplier Code Alias
  • Driver Reports do not include Suspended Contracts
  • Scheduled file via XML
  • Preferred Dealer can be mandatory when accepting a quote

September 2009 distributed on 12/10/2009

  • International Mobile Equipment Identity fields "IMEI"
  • Separate Quote Notes for Clients and for Staff
  • Tailor your Billing Type
  • Carbon Emissions Offset calculation (quote budget)
  • Client and Driver Events now have a “Value” field
  • Summit Leasing Interface
  • Contract Events now has unlimited text
  • Catch-e “Payments” has been released!

July 2009 distributed on 07/08/2009

  • Revision of Billing Type "Client Reimbursement"
  • Global Control updates ‘Start Period’
  • Bookings Module – Recharge Invoice
  • Division now available in CSV Export

June 2009 distributed on 03/07/2009

  • PAYG Tax Rates update for 1st July, 2009
  • SA Contract Stamp Duty now 0 as of 1st July, 2009
  • Negative Budgets in Quotes/Lease screen
  • Bookings – new “Operations” role
  • “Procurement” role can now access the Supplier/Financier tab
  • Add Interface field to Supplier record

May 2009 distributed on 01/06/2009

  • New Bookings module introduced
  • Multiple Driver email addresses
  • New Insurance Billing Type "Client Reimbursement"
  • Inventory Report – new CSV fields
  • BP Card Services Interface - Use Invoice Date as Invoice Number

April 2009 distributed on 04/05/2009

  • External Users can now access Catch-e using Firefox, Safari or Chrome!
  • New finance calculation for Quotations
  • Luxury Car Fuel Efficient Limit
  • Post Card Transactions by Contract
  • FBT Operating Report – Expense Classification
  • Contract FBT Y/E and Month End Default

Stress Free FBT distributed on 23/03/2009

  • Detailed description about the set-up and use of Catch-e's report to generate a one-time URL link email to drivers. This enables drivers to access a 'Declaration' page in Catch-e and submit an odometer reading online without the need for a login and password.

February 2009 distributed on 11/03/2009

  • New screen added to list contracts in the Driver area as Drivers/Contracts screen
  • Aftermarket Options enabled
  • ACT Stamp Duty calculations updated
  • Unlimited text for Driver events
  • Last Edit column displays on Quotes/Search

January 2009 distributed on 06/02/2009

  • Booking and Repair Flags added to Contracts/Contract
  • Accounts role can raise approvals on suspended contracts
  • Client Order Number field now displays on Inventory CSV file
